9. hitunglah panjang HD dalam gambar berikut!​


Jawaban:

cari panjang AC dulu

[tex]ac = \sqrt{ {12}^{2} + {9}^{2} } \\ ac = \sqrt{144 + 81} = \sqrt{225} = 15[/tex]

panjang AD

[tex]ad = \sqrt{ {15}^{2} + {8}^{2} } \\ ad = \sqrt{225 + 64} = \sqrt{289} = 17cm[/tex]

Penjelasan dengan langkah-langkah:

• panjang AC

AC = √(AB² + BC²)

AC = √(12² + 9²)

AC = √(144 + 81)

AC = √225

AC = 15 cm

• panjang AD

AD = √(AC² + CD²)

AD = √(15² + 8²)

AD = √(225 + 64)

AD = √289

AD = 17 cm

Jadi, panjang AD adalah 17 cm
