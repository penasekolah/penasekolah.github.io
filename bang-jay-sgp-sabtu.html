13. Jay membuat 18 loyang pizza. Setiap loyang pizza dipotong menjadi 8 potong sama besar. Jay mengundang 32 orang teman untuk datang kerumahnya. Jika setiap teman Jay memakan 3 potong pizza, berapa sisa pizza setelah dimakan?


Jawaban:

48 potong pizza

Penjelasan dengan langkah-langkah:

DIKETAHUI :

pizza = 18 loyang (1 loyang dipotong 8 jadi dikalikan)

= 18 x 8

= 144 potong pizza

Teman jay = 32 orang ( 1 orang makan 3 potong pizza)

= 32 x 3

= 96 potong pizza

MENCARI SISA PIZZA

= jumlah awal pizza - pizza yang dimakan

= 144 - 96

= 48

Jadi, sisa pizza setelah dimakan teman jay adalah 48 potong pizza atau setara dengan 5 loyang.

29. Quis 200 + 23³ = Itu siapa jay ?​


Jawaban:

200 + 23³

= 200 + (23 x 23 x 23)

= 200 + (529 x 23)

= 200 + 12.167

= 12.367

____________________________

Pembahasan :

Pangkat adalah perkalian berulang / berganda pada bilangan yang sama.

Contoh Bilangan Berpangkat :

Pangkat 2 :

1² = 1 x 1 = 1

2² = 2 x 2 = 4

3² = 3 x 3 = 9

4² = 4 x 4 = 16

5² = 5 x 5 = 25

6² = 6 x 6 = 36

7² = 7 x 7 = 49

8² = 8 x 8 = 64

9² = 9 x 9 = 81

10² = 10 x 10 = 100

Pangkat 3 :

1³ = 1 x 1 x 1 = 1

2³ = 2 x 2 x 2 = 8

3³ = 3 x 3 x 3 = 27

4³ = 4 x 4 x 4 = 64

5³ = 5 x 5 x 5 = 125

6³ = 6 x 6 x 6 = 216

7³ = 7 x 7 x 7 = 343

8³ = 8 x 8 x 8 = 512

9³ = 9 x 9 x 9 = 729

10³ = 10 x 10 x 10 = 1.000

Contoh Soal Perpangkatan :

2² + 3²

= (2 x 2) + (3 x 3)

= 4 + 9

= 13
