Harga promo

Paket B

1 kaos + 2 kemeja = 200.000

Misalkan

Kemeja = x

Kaos = y

x+3y=200.000 x (2)

2x+y=200.000 x (1 )

2x+6y=400.000

2x+ y=200.000 -

5y=200.000

y=200.000/5

y=40.000

x+3y=200.000

x+3(40.000)=200.000

x+120.000=200.000

x=200.000-120.000

x=80.000

Maka

x=kemeja=Rp 80.000/unit

y=kaos=Rp 40.000/unit

Harga normal

1 kaos dan 1 kemeja

1 kaos = (1 x 40.000 x 1,5)

= Rp 60.000

1 kemeja = (1 x 80.000 x 1,5)

=Rp 120.000

Rp 60.000 + Rp 120.000

14. di Alfamart harga sampo Rp50.000 dan Diskonnya 20%, berapa sisa harga sampo​


Jawab:

Harga setelah diskon adalah Rp. 40.000;

Penjelasan dengan langkah-langkah:

Diketahui :

HB = Rp. 50.000Disc = 20%

Ditanya :

Berapa harga bayar?

Penyelesaian :

Harga Bayar = (100% - Disc) x HB

                     = (100% - 20%) x 50.000

                     = 80% x 50.000

                     = [tex]\frac{80}{100}[/tex] x 50.000

                     = [tex]\frac{4}{5}[/tex] x 50.000

                     = Rp. 40.000;

Jadi Harga setelah diskon adalah Rp. 40.000;
