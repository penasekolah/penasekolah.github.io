Jawaban:

1. Vegetables aren't grown on the land.

2. Any phone calls aren't answered by Audrey today.

3. Is a new material explained in front of the class by the teacher?

4. When is the book returned by Jesslyn to the library?

5. Sugarcanes are planted by farmers in this village.

Penjelasan:

Untuk mengubah kalimat aktif menjadi kalimat pasif, kita harus mengenali jenis tense yang digunakan terlebih dahulu.

Kalimat di atas menggunakan simple present tense.

Rumus passive simple present tense adalah

(+) object + am/is/are + verb 3 + by subject.

(-) object + am/is/are + not + verb 3 + by subject.

(?) Am/is/are + object + verb 3 + by subject?

Wh questions: wh questions + am/is/are + object + verb 3 + by subject?
