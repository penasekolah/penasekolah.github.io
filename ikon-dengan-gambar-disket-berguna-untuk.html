4. Ikon yang digunakan untuk pemadu gambar adalah ikon​


Jawaban:Beberapa ikon pemandu gambar adalah sebagai berikut

- Eraser. Berfungsi untuk menghapus bagian dari gambar dan menggantinya dengan warna yang sama dengan background lembar kerja.

- Fill With Color. Berfungsi untuk memberi warna pada gambar dan background

- Color Picker / Pick Color. Berfungsi untuk memilih warna dari gambar lain dan menggunakannya untuk menggambar

- Magnifier. Berfungsi untuk membesakan bagian dari gambar

- Pencil. Berfungsi untuk membuat bentuk garis bebas dengan ketebalan yang ditentukan sesuai dengan kebutuhan

- Brush. Berfungsi untuk menggambar dengan berbagai bentuk kuas gambar.

- Air Brush. Berfungsi untuk menggambar dengan kuas seperti cat semprot

- Text. Berfungsi untuk memberi tulisan pada gambar

- Polygon. Berfungsi untuk membuat berbagai macam bentuk sesuai kebutuhan

Command yang digunakan untuk menyimpan file adalah "Save/Save As" yang memiliki Ikon "Disket" (A)a. Disket

itu adalah cara cepat untuk menyimpan file, fungsinya sama dengan menekan tombol "save" atau "save as".

Namun biasanya untuk mencari tombol

"save" dan "save as", kita harus menekan menu "File" terlebih dahulu.

Sedangkan ikon ( tombol ) disket, terdapat di bagian atas , dan tidak perlu menglik "File" terlebih dahulu, karena memang sengaja ditempatkan di luar, untuk mempermudah untuk menyimpan file.
