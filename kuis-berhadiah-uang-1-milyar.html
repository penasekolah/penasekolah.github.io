27. Perbandingan uang Umar dan Ustman saat ini 3 : 5. Jika selisih uang mereka berdua 28 milyar, berapa total uang mereka berdua? A. 112 milyar B. 84 milyar C. 56 milyar D. 48 milya


Jumlah uang:

(3 + 5)/(5 - 3) x 28

= 8/2 x 28

= 4 x 28

= 112 milyar

Jawaban:

112 milyar ( Opsi A )

Penjelasan dengan langkah-langkah:

Uang Umar :

[tex] = \frac{3}{5 - 3} \times 28[/tex]

[tex] = \frac{3}{2} \times 28[/tex]

[tex] = \frac{3}{ \cancel{2} \: {}^{1} } \times \cancel{28} \: {}^{14} [/tex]

[tex] = 3 \times 14[/tex]

[tex] = 42 \: milyar[/tex]

Uang Ustman :

[tex] = \frac{5}{5 - 3} \times 28[/tex]

[tex] = \frac{5}{2} \times 28[/tex]

[tex] = \frac{5}{ \cancel{2} \: {}^{1} } \times \cancel{28} \: {}^{14} [/tex]

[tex] = 5 \times 14[/tex]

[tex] = 70 \: milyar[/tex]

Total uang mereka :

Uang Umar + Uang Ustman

= 42 milyar + 70 milyar

= 112 milyar
