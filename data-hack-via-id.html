Jawaban:

Tagged VLAN (Trunking)

Penjelasan:

Jika ada lebih dari satu VLAN, maka diperlukan cara untuk membedakan paket data satu VLAN dengan paket data VLAN yang lain. Cara tsb adalah dengan menambahkan sebuah Tag (atau VLAN ID) ke dalam paket data. Oleh karena itu dinamakan “Tagged VLAN” karena didalamnya terdapat paket data yang memuat informasi VLAN ID.

Identitas yang diperlukan yakni id, name. 1. Jenis barang juga dikelompokkan menjadi Jam Tangan, HP, Dompet, Kacamata, Kunci Kendaraan, Kunci Pintu, Topi, Songkok, Kartu Identitas, dan lainnya. 2. Jawaban:

Untuk membuat ERD (Entity Relationship Diagram) dari sistem LnF, kita perlu memahami terlebih dahulu bagaimana entitas dan relasi yang ada dalam sistem tersebut. Berdasarkan penjelasan di atas, terdapat beberapa entitas yang dapat dikenali, yaitu:

Lokasi: menyimpan data lokasi tempat barang hilang dan ditemukan

Jenis Barang: menyimpan data jenis-jenis barang yang dapat hilang atau ditemukan

Barang: menyimpan data barang yang hilang atau ditemukan

Pengguna: menyimpan data pengguna (baik pengambil atau pelapor)

Kehilangan: menyimpan data barang yang hilang dan dilaporkan oleh pengguna

Pengambilan: menyimpan data barang yang diambil oleh pengguna

Berikut adalah contoh ERD yang dapat kami buat untuk sistem LnF:

Pada ERD di atas, terdapat beberapa relasi antar entitas, yaitu:

Lokasi memiliki banyak jenis barang (1-to-Many)

Jenis barang memiliki banyak barang (1-to-Many)

Barang dilaporkan oleh pengguna (Many-to-1)

Barang diambil oleh pengguna (Many-to-1)

ERD di atas hanya merupakan salah satu contoh dari banyak cara yang mungkin untuk menggambarkan sistem LnF. ERD yang lain mungkin akan menggunakan cara yang berbeda untuk menggambarkan relasi antar entitas, tergantung pada perspektif dan kebutuhan masing-masing.

21. Tuliskan syntax Php untuk menghapus data siswa berdasarkan ID siswa​


Jawaban:

berikut code nya :

<?php

// Koneksi ke database

$conn = mysqli_connect("localhost", "username", "password", "nama_database");

// Menangkap ID siswa yang akan dihapus dari parameter GET

$id_siswa = $_GET["id"];

// Membuat query untuk menghapus data siswa berdasarkan ID

$query = "DELETE FROM siswa WHERE id_siswa = $id_siswa";

// Menjalankan query untuk menghapus data

if (mysqli_query($conn, $query)) {

echo "Data siswa berhasil dihapus";

} else {

echo "Gagal menghapus data siswa: " . mysqli_error($conn);

}

// Menutup koneksi ke database

mysqli_close($conn);

?>

Penjelasannya :

1. Pertama-tama kita melakukan koneksi ke database dengan menggunakan fungsi `mysqli_connect()` . Di sini, kamu harus mengganti "localhost", "username", "password", dan "nama_database" dengan nilai yang sesuai untuk koneksi ke database kita.

2. menangkap ID siswa yang akan dihapus dari parameter GET dengan menggunakan variabel `$_GET["id"]`.

3. Setelah itu, kita membuat query SQL untuk menghapus data siswa berdasarkan ID menggunakan perintah `DELETE FROM`. Di sini, kita menggunakan variabel `$id_siswa` untuk menggantikan nilai ID siswa yang kita tangkap sebelumnya.

4. Kita mengeksekusi query menggunakan fungsi `mysqli_query()`. Jika query berhasil dijalankan, kita akan menampilkan pesan "Data siswa berhasil dihapus". Jika tidak, kita akan menampilkan pesan error menggunakan fungsi `mysqli_error()`.

5. Terakhir, kita menutup koneksi ke database menggunakan fungsi 'mysqli_close()'.

<?php

$servername = "localhost";

 die("Koneksi gagal: " . mysqli_connect_error());

?>

Jika tidak, pesan error dicetak ke layar.Koneksi ditutup dengan fungsi mysqli_close().
