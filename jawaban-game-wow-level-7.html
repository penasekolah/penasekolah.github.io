Jawaban:

U50 = 2600

Penjelasan dengan langkah-langkah:

a = 3

b = 8 - 3 = 5

c = 7 - 5 = 2

U50 = a+(n-1)b + (n-1)(n-2)c /2

= 3+(50-1)5 + (50-1)(50-2)2 /2

= 3+(49)5 + (49)(48)

= 3+245+2352

= 2600

Jadi, Amir mendapat poin sebanyak 2600 pada level 50
