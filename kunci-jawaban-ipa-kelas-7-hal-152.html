Kabel Listrik dipasang agak kendur agar kabel tidak putus saat suhu turun dan terjadi penyusutan.Koefisien muai panjang besi sebesar 12 x 10⁻⁵/°C atau  1,2 x 10⁻⁶/°CPanjang Baja pada suhu 70°C adalah 40,04 cm

Pembahasan

Pemuaian

Saat benda dikenai kalor, maka akan terjadi pemuaian pada benda tersebut yang akan mengakibatkan pertambahan ukuran pada benda. dan sebaliknya saat kalor dilepaskan dari benda, akan terjadi penyusutan, besarnya pemuaian atau penyusutan tergantung pada koefisien muai benda dan perubahan suhu.

Pada Pemuaian Panjang berlaku persamaan :ΔL = Lo x α x Δt  Lt = Lo + ΔL  

atau  

Lt = Lo + (Lo x α x Δt)  

Keterangan :  

Lo = Panjang awal = m  

Lt = Panjang akhir = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang

Pada Pemuaian Luas berlaku persamaan :ΔA = Ao x β x Δt  

dan  

At = Ao + ΔA  

Atau  

At = Ao + (Ao x β x Δt)  

Keterangan :  

ΔA = Pertambahan Luas  

Ao = Luas awal  

At = Luas Akhir  

β = Koefisien Muai Luas  

β = 2 α  

Pada Pemuaian Volume berlaku persamaan :ΔV = Vo x γ x Δt  

dan untuk Volume akhir adalah :  

Vt = Vo + ΔV  

atau  

Vt = V0 + (V0 x γ x Δt)  

Keterangan  

ΔV = Pertambahan Volume  

γ = Koefisien Muai Ruang atau Volume  

besarnya γ = 3α  

Δt= Perubahan Suhu  

Vo= Volume awal  

Vt = Volume akhir  

Pengerjaan soal :

Soal Np 1 :

Pemasangan Kabel Listrik dibuat kendur agar saatterjadi penyusutan dimalam hari kabel tidak putus, seperti sudah dijelaskan diatas jika benda mendapatkan kalor akan mengalami pemuaian, dan jika benda melepaskan kalor (mengalami penurunan suhu) akan terjadi penyuutan. Perubahan panjang tersebut dirumuskan dalam persamaan :

ΔL = Lo x α x Δt

Keterangan :  

Lo = Panjang awal = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang =  

Soal No 2 :Diketahui :

t1 = Suhu Awal = 0°C

t2 = Suhu akhir = 100°C

Lo = Panjang awal besi = 200 cm

Lt = Panjang akhir besi = 200,24 cm

ΔL = Perubahan panjang besi = 200,24 cm - 200 cm = 0,24 cm

Ditanyakan :

α = Kofisien muai panjang besi = ...

Penyelesaian :

ΔL = Lo x α x Δt

0,24 = 200 x α x 100

0,24 = 2.000α

α = 0,24 : 2.000

α = (24 x 10⁻²) : (2 x 10³)

α = 12 x 10⁻⁵   = 1,2 x 10⁻⁶

Soal No 3 :Diketahui :

t1 = Suhu awal = 20°C

t2 = Suhu akhir = 70°C

Δt = Perubahan Suhu = 70 - 20 = 50°C

Lo = Panjang Baja mula - mula = 40 cm

α   = Koefisien muai baja = 0,00002/°C = 2 x 10⁻⁵/°C

Ditanyakan :

Lt = Panjang akhir Baja = ...

Penyelesaian :

Lt = Lo + (Lo x α x t)

Lt = 40 + (40 x 2 x 10⁻⁵ x 50)

Lt = 40 + (4000 x 10⁻⁵)

Lt = 40 x (4 x 10⁻²)

Lt = 40 + 0,04

Lt = 40,04 cm

Kesimpulan :

Kabel di pasang agak kendur agar tidak putus saat penyusutanKoefisien muai panjang besi adalah 12 x 10⁻⁵/°C atau 1,2 x 10⁻⁶/°CPanjang akhir baja adalah 40,02 cm

1. Mengapa kabel listrik bertegangan tinggi dipasang agak kendur?

Jawab:

Kabel listrik bertegangan tinggi dipasang agak kendur agar mendapatkan ruang penyusutan dan ketika mengalami penyusutan pada malam hari kabel tidak putus.

2. Pada suhu 0°C, panjang sebatang besi adalah 200 cm. Jika suhu besi dinaikkan menjadi 100°C, ternyata panjang besi itu menjadi 200,24 cm. Tentukan koeisien muai panjang besi.

Diketahui:

Δl = 200,24 cm

lo = 200 cm

ΔT = 100° C

Ditanya: koefisien muai panjang besi (α)

Dijawab:

α = Δl/lo.ΔT

= (200,24-200)cm/200cm . 100°C

= 0,24cm/20.000cm.°C

= 12.10^-6° C^-1

Note: Perhatikan tanda ^ dibaca pangkat.

Kesimpulan: Maka koefisien muai panjang besi adalah 12.10^-6° C^-1.

3. Panjang sebatang baja pada 20°C adalah 40 cm. Jika koeisien muai panjang baja 0,00002/°C, berapa panjangnya pada suhu 70°C?

Diketahui:

α = 0,00002/°C = 2.10^-5

ΔT = 50° C

lo = 40 cm

Ditanya: panjang akhir baja pada suhu 70°C (l)

Dijawab:

l = lo (1 + αΔT)

= 40 cm (1 + 2.10^-5 x 50°C)

= 40 cm (1 + 0,001)

= 40,04 cm

Note: Perhatikan tanda ^ dibaca pangkat.

Kesimpulan: Panjang akhir baja pada suhu 70° C adalah 40,04 cm.

Maaf kalau salaah

2. Pak Joyo mempunyai 720 telur ayam. Telur ayam tersebut akan diletakkan ke dalam beberapa keranjang. Setiap keranjang dapat memuat 40 telur. Pak Joyo juga mempunyai telur itik 15 keranjang. Jumlah telur Pak Joyo seluruhnya … keranjang.

a. 43

b. 36

c. 33

d. 23

Kita terjemahin dulu yaa

720/40 = 18

Lalu Pak Joyo juga punya 15 keranjang telur maka kita tambah

18 + 15 = 33

Maka, jumlah telur Pak Joyo seluruhnya (C.) 33 keranjang.

5/2 = 50/20

Maka, sisa tepung terigu Ibu (B.) 1,45 kg.

kualitas bagus ⇒ permintaan bertambah

Poin nomor 1

Penjelasan:

Terdapat 2 macam bentuk verb 2 dari sebuah kata kerja:

1. Regular verb = bentuk beraturan (hanya menambahkan akhiran -ed pada akhir kata), seperti:  

cry - cried = menangis

calm - calmed = menenangkan

comfort - comforted = menghibur

carry - carried = membawa

play - played = bermain

start - started = memulai

walk - walked = berjalan

color - colored = mewarnai

2. Irregular verb = bentuk tidak beraturan (tidak ada aturan pasti dalam membentuk bentuk kedua dari kata kerja tidak beraturan ini), seperti:

is - was = to be/auxiliary verb

can - could = dapat

take - took = mengambil; mengasuh

don't sleep - didn't sleep = tidak tidur

sleep - slept = tidur

breastfeed - breastfed = menyusui

get - got = mendapatkan; beranjak

spoon-feed - spoon-fed = menyuapi

do - did = melakukan

go - went = pergi

read - read = membaca

sing - sang = menyanyi

run - ran = berlari

Untuk kata kerja bentuk tidak beraturan, kita dapat menghafalkan bentuk-bentuk tersebut, dikarenakan tidak memiliki aturan pasti bagaimana membentuk bentuk kedua kata kerja tersebut.
