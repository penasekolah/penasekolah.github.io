1. Software configuration management adalah


Jawaban & Penjelasan:

Software Configuration Management (SCM) adalah proses untuk secara sistematis mengelola, mengatur, dan mengontrol perubahan dalam dokumen, kode, dan entitas lain selama Siklus Hidup Pengembangan Perangkat Lunak.

Tujuan utamanya adalah untuk meningkatkan produktivitas dengan kesalahan minimal.

SCM adalah bagian dari bidang manajemen konfigurasi lintas disiplin dan dapat secara akurat menentukan siapa yang melakukan revisi.

3. Jelaskan fungsi software disk management !


Fungsi software disk management adalah sebagai berikut.

Membuat, menggabung, menghapus, menambah, dan mengurangi ukuran partisi.Mengubah atau menetapkan huruf drive untuk volume hard disk, drive removable disk, dan drive CD-ROM.Menghapus media penyimpanan (delete volume).Mengelola media penyimpanan.Membelah penyimpaan menjadi banyak tempat (shrink volume).Melihat angka kecepatan disk, dan sebagainya.

Pembahasan:

Disk management merupakan sebuah software atau alat interal dari Windows untuk melakukan pengaturan konfigurasi harddisk, Flashdisk, SSD, atau media penyimpanan lainnya. Beberapa cara yang dapat dilakukan untuk mengakses disk management adalah sebagai berikut.

Cara 1 : start - control panel - administrative tools - performance mintenance - klik sebanyak dua kali pada computer management - disk management.Cara 2 : start - run (nantinya akan keluar kotak dialog dan kemudian ketik diskmgmt.msc) dan terbuka disk management.

4. Sebuah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, disebut


disebut shareware...

Pengertian Shareware adalah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, jika pengguna merasa softwarenya bagus, maka diharuskan membeli. Shareware sering dibatasi lamanya waktu pakai (misalnya trial 30 hari), atau jumlah software tersebut dijalankan (misalnya 30 kali), atau feature-feature tertentu yang tidak bisa diakses. Sesudah masa ujicobanya berakhir, software bisa saja terkunci atau bisa saja tetap berfungsi sebagaimana mestinya dan tetap terbatasi.
