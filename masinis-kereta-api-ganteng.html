5. Kereta api A bergerak dengan kecepatan 20m/s dan kereta api B Bergerak dengan kecepatan 25m/s bergerak saling mendekati .. masinis kereta api A membunyikan pluit/klaksonnya dgn frekuensi 650Hz . Jika kecepatan bunyi di udara 350m/s . Maka frekuensi yang di dengar masinis kereta api B adalah .


vs = 20m/s → sumber mendekat (-)
vp = 25 m/s → pendengar mendekat (+)
fs = 650 Hz
V = 350 m/s
fp = . . . . . . . ?

Penyelesaian
- Efek Doppler -
Kereta A = Sumber bunyi (s)
Kereta B = pendengar (p)

fp = (V + vp) / (V - vs) . fs
fp = (350 + 25) / (350 - 20) . 650
fp = 375/330 . 650
fp = 738.633 Hz

Jadi, frekuensi yang di dengar masinis kereta api B adalah 738.633 Hz

v = v₀ + at
at = v - v₀
a = (v - v₀) / t
   = (0 m/s - 250/9 m/s) / (20 s)
   = -25/18 m/s²
   ≈ -1,39 m/s²

KINEMATIKA GERAK
- GLBB

Vo = 100 km/jam = ²⁵⁰/₉ m/s
Vt = 0       (berhenti)
t = 20 s
a = __ ?

Vt = Vo + a t
 0 = ²⁵⁰/₉ + a · 20
 a = -²⁵⁰/₉ / 20
 a = - ²⁵/₁₈ ≈ 1,39 m/s²  ← jwb

29. 1. Berapa jam masinis berdinas? 2. Apa itu mess masinis/griya karya? 3. Apa tugas juru langsir kereta api? 4. Bagaimana sikap kita jika kita naik kereta api tetapi kereta api yang kita naiki mengalami trouble/masalah/telat?


1. Jam kerja masinis adalah 6-10 jam kerja (2 KA (pulang pergi)).

2. Mess masinis atau griya kereta api adalah Orang yang membantu masinis dalam mengendarai kereta api.

3. tugas juru langsir kereta api adalah Menyusun rangkaian kereta api agar rangkaian kereta bisa berurutan, Menambah gerbong kereta api, Mengurangi gerbong kereta api, dan Menghapuskan rangkaian bagi kereta yang akan kembali k Dipo Kereta Api.

4. Yang kita hadapi saat kereta kita yang kita naiki kita harus sabar menunggu, mungkin ada kecelakaan atau masalah yang berhalangan dengan keretanya.
