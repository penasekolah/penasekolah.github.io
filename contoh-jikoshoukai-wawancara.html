penjelasan lebih lanjut (Partikel dan desu)

です (desu) digunakan untuk menyatakan kalimat positif dan ditempatkan diakhir kalimat

****partikel****

は (ha dibaca wa)

mengikuti topik yang ingin dibicarakan oleh pembicara. Oleh karena itu, ha (dibaca wa) sering disebut partikel penanda topik. “Topik” seringkali merupakan subjek gramatikal, tetapi dapat berupa apa saja (termasuk objek tata bahasa, dan terkadang) kata kerja), dan mungkin juga mengikuti beberapa partikel lainnya.  

pola kalimat = [ A ] wa [ B ] desu.

も (mo)

berfungsi sebagai "juga" atau "terlalu" dalam . Ini digunakan untuk menunjukkan bahwa sesuatu yang sebelumnya telah dinyatakan juga berlaku untuk item yang saat ini sedang dibahas. Ini menggantikan ga, wa atau o saat digunakan.  

pola kalimat= [Objek1] は [properti1 / tindakan1] で す  

[Objek2] も [properti1 / tindakan1] で す.

が (ga)

menandai subjek gramatikal kalimat saat pertama kali diperkenalkan ke percakapan.

Pendahuluan :

Jikoshoukai「自己紹介」

Dalam Bahasa Jepang, "Perkenalan diri" disebut dengan Jikoshoukai. Kata ini merupakan gabungan dari kata Jiko (Diri) dan Shoukai (Perkenalan). Jikoshoukai ini sangat penting dan selalu dilakukan saat bertemu dengan orang yang baru dikenal.

Jikoshoukai ini biasanya dibagi menjadi beberapa tahap, yaitu :

[Aisatsu 1] => [Perkenalan] => [Aisatsu 2]

Yang dimaksud dengan Aisatsu yaitu adalah salam atau sapaan. Pada bagian Aisatsu 1 berisi salam pembuka, seperti Hajimemashιte, Doumo, Konnichiwa, dan sebagainya. Sedangkan pada bagian Aisatsu 2 berisi salam penutup, seperti Yoroshiku onegaishimasu, Yoroshiku, Douzo yoroshiku, dan sebagainya.

Di antara kedua Aisatsu tersebut, diisi dengan bagian perkenalan. Bagian ini meliputi perkenalan nama, umur, hobi, alamat, jumlah keluarga, dan sebagainya. Pembahasan mengenai kalimat untuk memperkenalkan diri bisa dilihat di bagian pembahasan di bawah ini.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

1. Memberitahu nama

Bisa menggunakan kalimat-kalimat berikut.

Watashi wa [Nama] desuWatashi wa [Nama] to iimasuWatashi wa [Nama] to moushimasu

Arti = Saya [Nama]

Kalimatnya saya urutkan dari yang biasa ke yang paling sopan. Penggunaannya tergantung situasi. Pada bagian nama tersebut, bisa diisi dengan nama lengkap maupun nama panggilan. Jika yang disebutkan adalah nama lengkap, dan kemudian kalian ingin menyebutkan nama panggilan, kailian bisa menggunakan kalimat yang saya berikan di nomor 2.

2. Memberitahu nama panggilan

Bisa menggunakan kalimat berikut.

[Panggilan] to yonde kudasai

Arti = Panggil saja [Panggilan]

Pola kalimat yang satu ini hanya digunakan jika kalian menyebutkan nama lengkap pada pola kalimat di nomor 1. Jika kalian ingin menyebutkan nama panggilan saja, bisa menggunakan salah satu dari ketiga pola kalimat yang saya sebutkan sebelumnya di nomor 1.

3. Memberitahu umur

Bisa menggunakan kalimat berikut.

Watashi wa [Umur] sai desu

Arti = Saya berumur [Umur] tahun

Pembahasan lebih lengkap mengenai cara memberitahu umur, bisa dilihat di link berikut : https://brainly.co.id/tugas/49892916

4. Memberitahu hobi

Bisa menggunakan kalimat berikut.

Watashi no shumi wa [Hobi] desu

Arti = Hobi saya [Hobi]

Pembahasan lebih lengkap mengenai contoh-contoh hobi, bisa dilihat di link berikut : https://brainly.co.id/tugas/18347749

5. Memberitahu alamat

Bisa menggunakan kalimat berikut.

Watashi wa [Alamat] ni sundeimasu

Arti = Saya tinggal di [Alamat]

Pada bagian alamat tersebut, bisa diisi dengan nama kota atau lainnya yang lebih spesifik untuk menjelaskan alamat pembicara.

6. Memberitahu jumlah keluarga

Bisa menggunakan kalimat berikut.

Watashi wa [Jumlah] nin kazoku desu

Arti = Saya [Jumlah] orang keluarga

Watashi wa [Jumlah] nin kyoudai desu

Arti = Saya [Jumlah] orang bersaudara

Sesuai artinya, pada pola kalimat pertama memberitahu jumlah anggota keluarga, sedangkan pada pola kalimat kedua memberitahu jumlah saudara saja. Pada bagian jumlah, kalian menyebutkan jumlah orang dari keluarga atau saudara, meliputi diri sendiri juga. Pembahasan lebih lengkap mengenai cara penyebutan jumlah orang bisa dilihat di link berikut : https://brainly.co.id/tugas/50906312

- - - - - - - - -

Pada pola-pola kalimat yang saya sebutkan tersebut, kalian juga bisa tidak mengatakan 'Watashi wa/no' (Saya) jika subjeknya sudah jelas. Maksudnya yaitu jika lawan bicara sudah tau bahwa subjeknya adalah si pembicara sendiri, maka tidak perlu lagi untuk menyebutkan subjeknya. Tapi jika kalian ingin menyebutkan subjeknya, tidak usah selalu disebutkan di setiap kalimatnya, cukup beberapa saja atau di awalnya saja.

- - - - - - - - -

Berikut beberapa contoh teks perkenalan diri menggunakan pola-pola kalimat yang telah saya sebutkan di atas.

Doumo hajimemashιte, watashi wa Azuka Hirumi desu. Azuka to yonde kudasai. Watashi wa 16-sai desu. Shumi wa anime wo miru koto desu. Bukashi ni sundeimasu. Watashi wa 3-nin kazoku desu. Douzo yoroshiku onegaishimasu.

13. akhir kalimat dalam jikoshoukai ditutup dengan kata


JAWAB :

Dalam perkenalan diri (Jikoshoukai), biasa diawali dengan hajimemashlte dan di akhiri dengan salam penutup seperti yoroshiku onegaishimasu.

Selain yoroshiku onegaishimasu, bisa juga menggunakan douzo yoroshiku, douzo yoroshiku onegaishimasu, yoroshiku onegaitashimasu, dan lain-lain.

Artinya kurang lebih seperti, 'mohon bantuannya' atau 'senang berkenalan dengan anda'.

PENJELASAN :

自己紹介 (Jikoshoukai)

Perkenalan Diri dalam Bahasa Jepang

1. はじめまして (Hajimemashlte)  

= Perkenalkan / salam kenal

2. 私の名前は ... です (Watashi no namae wa ... desu)  

= Nama saya ...

3. ... と呼んでください ( ... to yonde kudasai)  

= Tolong panggil saja ...

4. 私は ... 歳です (Watashi wa ... sai desu)  

= Saya ... tahun

5. 私は ... ... 年生です。  

= Saya kelas ... ...

6. 私は ... に住んでいます (Watashi wa ... ni sunde imasu)  

= Saya tinggal di ...

7. 私の趣味は ... です (Watashi no shumi wa ... desu)  

= Hobi saya adalah ...

8. どうぞよろしくお願いします (Douzo yoroshiku onegaishimasu)  

= Senang berkenalan dengan anda

Contoh :

はじめまして。私の 名前は アズカヒルミです。アズカと 呼んでください。私は1 5歳です。高校１年生です。私は 東京に 住んでいます。趣味は アニメを 見ることです。どうぞ よろしくお願いします。  

Hajimemashlte. Watashi no namae wa Azuka Hirumi desu. Azuka to yonde kudasai. Watashi wa 15-sai desu. Koukou 1-nensei desu. Watashi wa Toukyou ni sunde imasu. Shumi wa anime o miru koto desu. Douzo yoroshiku onegaishimasu.  

Perkenalkan. Nama saya Azuka Hilmi. Tolong panggil saja Azuka. Saya 15 tahun. Kelas 1 SMA. Saya tinggal di Tokyo. Hobi saya menonton anime. Senang berkenalan dengan anda.

16. contoh wawancara tentang olahragawan

2. 1P : Apa sebabnya anda belajar berenang?
Hendri : Karena orang tua... (jawabnya cepat).. karena saya hampir pernah kelelep juga sih.. (sambil senyum-senyum.. hihihi..)
3. 1P : Siapa idola renang anda?
Hendri : Ian Thorpe dari Australia..
4. 1P : Apa sebabnya Ian Thorpe (sambil belajar nyebutin namanya.. ) dijadikan idola?
Hendri : ..bodinya bagus.. dan prestasinya bagus juga sih.. (jawabnya sedikit cuek)
5. 1P : Mulai kapan anda mulai belajar serius?
Hendri : Mulai kelas 2 SD..
6. 1P : Kapan anda mulai ikut lomba renang?
Hendri : Sejak kelas 4 SD.. di acara Lomba Renang Antar Sekolah Se-Tanjungpinang..
7. 1P : Sejak kapan anda mendapatkan prestasi?
Hendri : Sejak kelas 1 SMP.. Juara III untuk seluruh nomor.. (jawabnya mulai bersemangat)
8. 1P : Waktu itu ikut di nomor lomba apa saja?
Hendri : ..nomor lomba 50 m bebas, 100 m bebas, 50 m dada, 100 m dada..
9. 1P : Spesialis renangnya..??
Hendri : ..spesialis di nomor 50 meter dada..
10. 1P : Prestasi lain di renang?
Hendri : (sambil mengingat-ngingat..) .. waktu kelas 3 SMP.. kalau tak salah 2 Emas di nomor dada, 2 Perak di nomor Bebas..
11. 1P : ..pada acara apa..???
Hendri : (mengingat lagi) ..POPDA* Tahun 2002 di Pekanbaru.. [*Pekan Olah Raga Pelajar Daerah] ..lalu di POPDA 2004 (sambungnya) juga dapat 2 Emas, di nomor 100 m punggung dan 200 m dada.. ..juga 2 Perak, di nomor 100 m dada dan 4x50 estafet gaya ganti..
12. 1P : ..siapa saja teman-teman yang ikut 4x50 estafet?
Hendri : (mengingat sebentar) ..eeaaa.., 50 m pertama saya, 50 m kedua Hardiono, 50 m ketiga Hendrianto, 50 m keempat Julianto.. (jawabnya lancar.. malah gantian 1P yang keteter nyatetnya.. hihihi..)
13. 1P : Pernah ikut Lomba Olah Raga lain??
Hendri : Basket dan Bowling... tapi tak pernah menang sih.. paling sampai penyisihan pertama saja..
14. 1P : Lomba Basket dan Bowling masih diterusin..??
Hendri : Ngga' lagi.. tak punya dana buat main Bowling.. teman-teman main Basket juga sudah pada berangkat kemana-mana.. (agak sedih)
15. 1P : Kembali ke renang.. persiapan untuk latihan renang bagaimana?
Hendri : Latihan biasa.. 1 minggu 3x renang dan 3x fitness.. kalau untuk pertandingan training center 1 bulan sebelumnya, 1 minggu 6x renang dan 6x fitness..
16. 1P : Pelatih anda siapa?
Hendri : Sani dari PRSI* Dendang Ria Tanjungpinang.. [*Persatuan Renang Seluruh Indonesia]
17. 1P : Pernah bertemu atlit renang nasional?
Hendri : Pernah.. Elsa-Elvira Manora Nasution dan adiknya Akbar Nasution dari Jambi.. terus pernah bertemu Albert-Felix Sutanto (si kembar-red) dan adik-adiknya Hendri-Glenn Sutanto..
