3. Tentukan luas segitiga berikut

Jadi panjang sisi segitiga sama kaki adalah 8√2 cm

Jadi luas segitiga adalah 64 cm².

5. Tentukan luas persegi panjang KLMN berikut.

Penyelesaian :

Perbandingan sisi segitiga yang mempunyai sudut 30°, 60° dan 90° adalah 1 : √3 : 2. Maka panjang KN : KL : LN = 1 : √3 : 2

Menentukan panjang KL

KL : LN = √3 : 2

[tex]\displaystyle \frac{KL}{LN} = \frac{\sqrt{3} }{2}[/tex]

[tex]\displaystyle \frac{KL}{8} = \frac{\sqrt{3} }{2}[/tex]

KL = [tex]\displaystyle \frac{8 \times \sqrt{3} }{2}[/tex]

KL  = 4√3 cm

Panjang KN

KN : LN = 1 : 2

[tex]\displaystyle \frac{KN}{LN} = \frac{1}{2}[/tex]

[tex]\displaystyle \frac{KN}{8} = \frac{1}{2}[/tex]

KN = [tex]\displaystyle \frac{8}{2}[/tex]

KN = 4 cm

Luas persegi panjang KLMN

Luas = KL × KN

        = 4√3 cm × 4 cm

        = 16√3 cm²

Jadi luas persegi panjang KLMN adalah 16√3 cm²

6. Perhatikan gambar segitiga siku siku ABC di bawah. Tentukan:

a. Keliling segitiga ABC b.Tentukan luas segitiga ABC

Penyelesaian :

Perhatikan Δ ADC siku-siku di D, ∠ CAD = 60° dan ∠ ACD = 30°

AC : AD = 2 : 1

 AC : 8 = 2 : 1

      AC = 8 × 2

      AC = 16 cm

AD : CD = 1 : √3

 8 : CD = 1 : √3

 8 / CD = 1 / √3

      CD = 8 × √3

      CD = 8√3 cm

Perhatikan Δ BDC  siku-siku di D, ∠ CBD = 30° dan ∠ DCB = 60°

Panjang BD

CD : BD = 1 : √3

8√3 : BD = 1 : √3

8√3 / BD = 1 / √3

      BD = 8√3 × √3

      BD = 8 × 3

      BD = 24 cm

Panjang BC

CD : BC = 1 : 2

8√3 : BC = 1 : 2

8√3 / BC = 1 / 2

      BC = 8√3 × 2

      BC = 16√3 cm

a. Keliling segitiga ABC

Keliling Δ ABC = AD + BD + BC + AC

                       = 8 cm + 24 cm + 16√3 + 16 cm

                       = 48 cm + 16√3 cm

                       = 16 (3 + √3) cm

Jadi keliling segitiga ABC adalah 16 (3 + √3) cm

b. Menentukan luas segitiga ABC

Luas Δ ABC = 1/2 × AB × CD

                   = 1/2 × (8 + 24) cm × 8√3 cm

                   = 1/2 × 32 × 8√3 cm²

                   = 16 × 8√3 cm²

                   = 128√3 cm²

  Jadi luas segitiga ABC adalah 128√3 cm²

4. Kunci jawaban buku paket ipa halaman 102-103 kelas 8 SMP semester 1


1. Termasuk Pengungkit jenis I : titik tumpu berada diantara titik beban dan titik kuasa, Contoh alat pada Neraca, gunting dan jungkat - jungkit.

2. a. Besar Usaha yang dilakukan sebesar 240 J.

   b. Daya yang dilakukan 24 watt.

3. Pengungkit A lebih sedikit membutuhkan tenaga karena panjang lengan kuasa lebih kecil.

4. Besar beban yang ditari 32 N.

5. Keuntungan Mekanis pada bidang miring sebesar 5

Pembahasan

Soal No 1

Pengungkit dibedakan menjadi 3 macam yaitu :

Pengungkit Jenis I = Titik Tumpu terlatak diantara titik beban dan titik kuasa.

Pengungkit Jenis II = Titik Beban Berada dintara titik tumpu dan dan titik kuasa.

Pengungkit Jenis III = Titik Kuasa berada diantara titik tumpu dan titik beban.

Dimana Hubungan pada bagian Titik Tumpu, Beban dan kuasa dirumuskan sebagai berikut :  

[tex]\boxed{LK\times{F}=W\times{LB}}[/tex]

Dengan :

LK = Lengan kuasa(m)  

LB = Lengan beban(m)  

W = Beban(N)  

FK = Gaya kuasa(N)  

dan Nilai KM = Keuntungan Mekanis di rumuskan sebagai berikut :  

KM = [tex]\frac{LK}{LB}[/tex] = [tex]\frac{W}{F}[/tex]

Soal No 2

Diketahui :

m = Massa = 6 kg

S  =  Perpindahan = 2 m

F  =  Gaya  =  120 N

t  =  Waktu = 10 s

Ditanyakan :

a. W = Usaha = ...

b. P = Daya = ...

Penyelesaian :

a. W = F x S

W = 120 x 2

W = 240 Joule

b. P =  W/t

P = 240/10

P = 24 Watt

Soal No 3.

Perhatikan Rumus untuk mencari KM pada Tuas :

KM = [tex]\frac{LK}{LB}[/tex] = [tex]\frac{W}{F}[/tex]

Gambar A :

LB = Lengan Beban = 20 cm

LK = lengan Kuasa = 120 cm

KM = LK/LB

KM = 120/20

KM = 6

Gambar B :

LB = Lengan Beban = 80 cm

LK = lengan Kuasa = 60 cm

KM = LK/LB

KM = 60/80

KM = 0,75

Jadi Dilihat dari nilai KM Pengungkit Jenis A lebih menguntungkan dan lebih membutuhkan sedikit tenaga.

Soal No 4 :

Pada Katrol berlaku prinsip KM sebagai Berikut :

Pada Katrol Tetap nilai KM adalah 1.Pada Katrol bergerak nilai KM adalah 2Pada katrol Majemuk nilai KM melihat Jumlah tali yang bergerak .

Dan penghitungan KM berdasarkan Beban dan Kuasa dirumuskan sebagai berikut :

[tex]\boxed{KM=\frac{W}{F}}[/tex]

Pada Soal Disajikan Gambar Katrol Majemuk dengan jumlah tali yang bergerak sebanyak 4.

Diketahui :

F = Gaya = 8 N

KM = keuntungan mekanis = 4

Ditanyakan :

W = Beban = ...

Penyelesaian :

W = KM x F

W = 4 x 8

W = 32 N

Soal No 5 :

Pesawat Sederhana Bidang Miring

Pada Bidang MiringBerlaku Persamaan Sebagai Berikut :

[tex]\boxed{KM=\frac{W}{F}=\frac{S}{h}}[/tex]

Keterangan :

KM = Keuntungan Mekanis

W = Beban ( N )

F = Gaya ( N )

S = Panjang Lintasan

h = Tinggi Bidang miring  

Diketahui :

S  =  Panjang Lintasan = 15 m

h  =  Tinggi Bidang Miring = 3 m

Ditanyakan :

KM = Keuntungan Mekanis = ...

Penyelesaian :

KM = S/h

KM = 15/3

KM = 5

Pembahasan

Kunci jawaban buku paket BSE IPA kurikulum 2013 kelas 8 halaman 69 akan diuraikan sebagai berikut.

Poin nomor 1

Gambar a merupakan otot lurik.

Gambar b merupakan otot jantung.

Gambar c merupakan otot polos.

Otot merupakan alat gerak aktif yang memiliki kemampuan untuk berkontraksi serta berelaksasi. Otot terdiri atas 3 jenis sel otot yaitu otot polos, otot lurik dan otot jantung.

Ciri otot jantung yaitu:  

Bentuk silinder panjang dan bercabang  Inti berjumlah 1 atau 2 dan terletak di tengah  Sifat tidak dapat dikendalikan dengan sadar dan tidak mudah lelah  Lokasinya di jantung  

Ciri otot polos yaitu:  

Bentuk gelendong  Inti berjumlah satu dan terletak di tengah  Sifat tidak dapat dikendalikan dengan sadar dan tidak mudah lelah  Lokasinya di organ dalam seperti usus

Poin nomor 2

Kita dianjurkan untuk berjemur pada pagi hari adalah untuk mendapatkan provitamin D dan vitamin D yang berasal dari sinar matahari pagi.

Provitamin D dan vitamin D ini berfungsi dalam pembentukan tulang.

Tulang merupakan alat gerak pasif. Selain itu, tulang juga berfungsi sebagai penyokong atau pemberi bentuk pada tubuh kita. Tulang juga berperan membantu menegakkan tubuh kita. Secara umum, tulang manusia berfungsi untuk:  

Alat gerak pasif.  Tempat melekatnya otot.  Menyokong tubuh  Menegakkan tubuh  Meproduksi sel darah (pada sum sum tulang)  Penimbunan zat mineral berupa kalsium dan fosfat  Melindungi organ tubuh  

Tulang dapat berfungsi sebagai pelindung bagian tubuh yang vital, maka dari itu tulang harus keras. Zat yang menyebabkan tulang menjadi keras adalah Kalsium dan Fosfat. Tulang pada dasarnya dibagi menjadi 2 jenis yaitu tulang rawan dan tulang keras.

Sel tulang rawan atau kondrosit  Terdapat matriks yang homogen atau berisi serabut kolagen dan serabut elastis  Bersifat lentur dan elastis atau liat

Tulang pada hidung  Tulang pada daun telinga  Tulang pada trakea  Tulang pada embrio

Tulang hasta  Tulang tengkorak  Tulang paha  Tulang rusuk

9. jawaban buku paket ipa kelas 8 semester 1 halaman 20​


Dalam buku paket IPA kelas 8 semester 1 halaman 20, disajikan 10 peristiwa. Kita diminta untuk menentukan peristiwa tersebut merupakan penerapan Hukum I, II atau III Newton beserta alasannya. Dari kesepuluh peristiwa tersebut (dapat dilihat di lampiran), yang merupakan penerapan:

Hukum I Newton adalah peristiwa nomor 2, 9 dan 10.Hukum II Newton adalah peristiwa nomor 4, 6 dan 8.Hukum III Newton adalah peristiwa nomor 1, 3, 5 dan 7.

Untuk alasannya dapat dilihat di pembahasan.

Pembahasan

Terdapat tiga bagian dalam Hukum Newton yaitu:

Hukum I Newton: suatu benda yang resultan gayanya sama dengan nol, akan tetap diam jika awalnya diam atau tetap bergerak lurus beraturan jika awalnya juga bergerak lurus beraturan.Hukum II Newton: percepatan pada benda yang bergerak lurus berbanding terbalik dengan massa benda dan berbanding lurus dengan gaya yang diberikan.Hukum III Newton: suatu benda mengerjakan gaya (aksi) terhadap benda lain, maka benda lain tersebut memberikan gaya yang besarnya sama (reaksi) tetapi arahnya beraturan terhadap suatu benda tersebut.

Disajikan 10 peristiwa yang merupakan penerapan Hukum Newton (lihat lampiran).

Peristiwa 1 merupakan penerapan Hukum III Newton karena adanya gaya aksi reaksi pada masing-masing kedua kijang tersebut, sehingga saat saling beradu, keduanya terpental.Peristiwa 2 merupakan penerapan Hukum I Newton karena kedua badak beradu dengan besar gaya yang sama dan saling berlawanan sehingga resultan gayanya sama dengan nol.Peristiwa 3 merupakan penerapan Hukum III Newton karena saat anak badak memberikan gaya aksi terhadap induknya, anak badak tersebut terpental akibat gaya reaksi.Peristiwa 4 merupakan penerapan Hukum II Newton karena gaya yang diberikan harimau lebih kuat sehingga kijang terpental karena mengalami percepatan.Peristiwa 5 merupakan penerapan Hukum III Newton karena adanya gaya aksi saat kepakan sayap ke bawah, sehingga membuat burung terangkat ke atas karena adanya gaya reaksi.Peristiwa 6 merupakan penerapan Hukum II Newton karena gaya dorong induk gajah terhadap anaknya membuat anak gajah bergerak dengan percepatan tertentu.Peristiwa 7 merupakan penerapan Hukum III Newton karena adanya gaya aksi saat sirip ikan digerakkan ke belakang sehingga ikan dapat berenang ke depan karena adanya gaya reaksi.Peristiwa 8 merupakan penerapan Hukum II Newton karena jerapah yang memiliki kepala besar, saat menyerang jerapah jantan lainnyapun memiliki gaya yang besar, sehingga jerapah jantan yang diserang mengalami percepatan karena serangan jerapah yang berkepala besar tersebut.Peristiwa 9 merupakan penerapan Hukum I Newton karena saat kuda berhenti, paket pada kuda tersebut terlempar karena mempertahankan keadaanya.Peristiwa 10 merupakan penerapan Hukum I Newton karena joki yang terlempar saat kuda berhenti sedang mempertahankan keadaannya.

Untuk memudahkan mengingat perbandingan segitiga siku-siku yang mempunyai sudut 30° dan 60° adalah

AB : BC : AC = 1 : 1 : √2
----------------------------------------------

Pembahasan : 

No. 1.

a.  hipotenusa = √32
     sudut = 45°

    cara perbandingan
    a : h = 1 : √2
    a : √32 = 1 : √2
    a / √32 = 1 / √2
             a = √32 / √2
             a = √16
             a = 4

   cara pythagoras
   a² + a² = (√32)²
        2a² = 32
          a² = 32 / 2
          a² = 16
          a = √16
          a = 4

b.  panjang sisi = 72
     sudut = 45°

    cara perbandingan
    s : a = 1 : √2
    72 : a = 1 : √2
    72 / a = 1 / √2
           a = 72 × √2
           a = 72√2

   cara pythagoras
   72² + 72² = a²
      2 × 72² = a²
               a = [tex] \sqrt{ 72^{2} \times 2} [/tex]
               a = 72√2

c.  hipotenusa = 16 cm
     sudut = 60°

     h : b = 2 : √3
    16 : b = 2 : √3
    16 / b = 2 / √3
    16 × √3 = b × 2
              b =  [tex] \frac{16 \sqrt{3} }{2} [/tex]
              b = 8√3 cm

d.  tinggi = 17√2
     sudut = 30°

    c : t = 1 : √3
    c : 17√2 = 1 : √3
    [tex] \frac{c}{17 \sqrt{2} } = \frac{1}{ \sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{2} }{ \sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{2} }{ \sqrt{3} } \times \frac{\sqrt{3} }{\sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{6} }{3} [/tex]

  d : t = 2 : √3
  d : 17√2 = 2 : √3
  [tex] \frac{d}{17 \sqrt{2} } = \frac{2}{ \sqrt{3} } [/tex]
  d × √3 = 17√2 × 2
     √3 d = 34√2
          d = [tex] \frac{34 \sqrt{2} }{ \sqrt{3} } [/tex]
          d = [tex] \frac{34 \sqrt{2} }{ \sqrt{3} } \times \frac{\sqrt{3} }{\sqrt{3} } [/tex]
          d = [tex] \frac{34 \sqrt{6} }{3} [/tex]

e.  alas = 5
     sudut = 60°

     alas : a = 1 : 2
      5 : a = 1 : 2
     5 / a = 1 / 2
          a = 5 × 2
          a = 10

    a : b = 2 : √3
    10 : b = 2 : √3
   10 / b = 2 / √3
   2 × b  = 10 × √3
        2b = 10√3
          b = 10√3 / 2
          b = 5√3

f.  hipotenusa = 20
    sudut = 60°

   d : h = 1 : 2
   d : 20 = 1 : 2
   d / 20 = 1 / 2
         d = 20 / 2
         d = 10

   e : h = √3 : 2
   e : 20 = √3 : 2
   e / 20 = √3 / 2
   e × 2 = 20 × √3
       2e = 20√3
         e = 20√3 / 2
         e = 10√3

AB = [tex] \frac{18 \sqrt{2} }{ \sqrt{2} } [/tex]
AB = 18
