7 + 2

2a=e
b=a+100.000
c=a+150.000
d=e-180.000

(a+b+c+d+e) / 5 = 525.000
a+b+c+d+e = 2.626.000
a+(a+100.000)+(a+150.000)+(e-180.000)+e = 2.625.000
3a + 70.000 + 2e = 2.625.000
3a + 2e = 2.555.000
3a + 2(2a) = 2.555.000
3a + 4a = 2.555.000
7a = 2.555.000
a = 365.000

e = 2a
   = 2(365.000)
   = 730.000

d = e - 180.000
  = 730.000 - 180.000
   = 550.000dari pernyataan diatas, maka didapat persamaan :
2A = E
A + 100.000 = B
A + 150.000 = C
E - 180.000 = D
(A + B + C + D + E) /5 = 525.000

dari persamaan diatas, dapat dibuat persamaan baru dengan mensubtitusikan beberapa pers. diatas.
(A + B + C + D + E) = 525.000 x 5
[A + (A + 100.000) + (A + 150.000) + (E - 180.000)] = 2.625.000            (karena E = 2A, maka)
[A + A + 100.000 + A + 150.000 + ({2A} - 180.000)] = 2.625.000
5A + 70.000 = 2.625.000
A = 511.000...........................(1)

2A = E (subtitusikan pers. 1)
2(511.000) = E
E = 1.022.000.........................(2)

E - 180.000 = D ( subtitusikan pers. 2 )
(1.022.000) - 180.000 = D
D = 842.000

22. cara dapetin 1,satu perempatnya bagaimana?


5/20 kak dia itu dibagia. 1/2 + 3/4 =

Proses :

Pertama,tentukan kpk dari 2 dan 4.

2 = 2 × 1
4 = 2 × 2
KPK = 2 × 2

Jadi, KPK dari 2 dan 4 adalah 4.

1/2 = 1/2 × 2/2 = 2/4

3/4 = 3/4 × 1/1 = 3/4

1/2 + 3/4 = 2/4 + 3/4 = 5/4 = 1 1/4

Hasilnya 1 1/4 karena 5 lebih dari 4 jadi 5 - 4 adalah 1 dan 4 di jadikan 1. jadi 1 1/4.
