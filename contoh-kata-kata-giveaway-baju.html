2. apa itu kata tanya? giveaway jawaban tercerdas​


Kata tanya merupakan kata yang digunakan untuk mengajukan pertanyaan. Kata tanya selalu berada di awal kata kalimat tanya.

Kata tanya

Apa : Digunakan untuk menanyakan keadaan, benda, atau perbuatan.

Siapa : Digunakan untuk menanyakan nama orang.

Bagaimana : Digunakan untuk menanyakan proses pengerjaan sesuatu.

Mengapa : Digunakan untuk menanyakan alasan atau sebab sesuatu.

Kapan : Digunakan untuk menanyakan waktu.

Dimana : Digunakan untuk menanyakan tempat.

Contoh kalimat tanya:

Apa yang kamu lakukan hari ini?

Siapa nama narasumbernya?

Bagaimana cara membuat kolase?

Mengapa kamu tidak suka pelajaran matematika?

Kapan restoran itu dibuka?

Dimana kamu berlibur?

23. ayo giveawaymau giveaway jawab pertanyaan #jangan ngasal​


No 1L.permukaan limas segitiga siku²

luas alas + luas sisi tegak

↓↓↓ sisi tegak limas segitiga ada 3

luas alas + luas S1 + luas S2 + luas S3

= ( 1/2 × a × t ) + 24 + 32 + 40

= ( 1/2 × 6 × 8 ) + 24 + 32 + 40

= ( 1/2 × 48 ) + 24 + 32 + 40

= 24 + 24 + 32 + 40

= 48 + 72

= 120 cm²

No 2 L.permukaan limas segi empat

luas alas + luas sisi tegak

↓↓↓ sisi tegak limas segi empat ada 4

luas alas + luas S1 + luas S2 + luas S3 + S4

= ( s × s ) + 135 + 135 + 135 + 135

= ( 4 × 4 ) + 135 + 135 + 135 + 135

= 16 + ( 135 × 4 )

= 16 + 540

= 556 cm²
