Sesuai dengan hukum bernoulli:
jika kecepatan udaranya besar maka tekanan udaranya menjadi kecil.

Jadi jika anak berdiri dipinggir rel kereta api ketika kereta apinya sedang lewat, maka udara pada rel kereta api menjadi kencang dan tekanan udaranya menjadi kecil.
Tekanan udara dari luar rel lebih besar dari pada tekanan udara di dalam rel, sehingga ini membuat udara luar rel mendorong anak ke arah rel kereta api

29. Sebuah kereta api meluncur menjauhi stasiun kereta api dengan kecepatan 72 km/jam. Seorang anak berjalan searah kereta api dengan kecepatan tetap 5 m/s. Berapakah kecepatan anak terhadap stasiun kereta api ?


Penjelasan:

Sebuah kereta api meluncur menjauhi stasiun kereta api dengan kecepatan 72 km/jam. Seorang anak berjalan searah kereta api dengan kecepatan tetap 5 m/s. Berapakah kecepatan anak terhadap stasiun kereta api ?

Diketahui,

V1 (Kereta api) = 72 km/jam

V2 (seorang anak) = 5 m/s

Untuk mencari yang lebih cepat, kita samakan satuan kecepatannya.

72 km/jam

[tex] \frac{72000 \: m}{3600 \: s} = \frac{720 \: m}{36 \: s} = 20 \frac{m}{s} [/tex]

72 km/jam = 20 m/s

Dibanding dengan seorang anak itu, 5 m/s selisih 15 m/s dengan kecepatan kereta api.

Kecepatan anak tersebut, kurang 15 m/s dari kecepatan kereta api.
