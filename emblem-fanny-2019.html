4. Fanny mengeluarkan dua buah kabel dengan gaya masing masing 100 N, dan keduanya membentuk sudut 120°, berapa resultan gaya dari kabel fanny?​


Jawaban:

FR = 100 N

Langkah Pengerjaan:

FR = √(F₁²+F₂²+2.F₁.F₂.cosθ)

FR = √(100²+100²+2.100.100.cos120)

FR = √(10000+10000+20000.(-1/2))

FR = √(20000-10000)

FR = √10000

FR = 100 N
