maka yang dibutuhkan adalah 24 wallpaper (a)

22. Langkah langkah dalam mengatur tampilan walpaper di Windows 95 ?


1.Klik kanan di bagian desktopnya
2. cari dan klik "personalize"
3. klik "desktop background"
4. jika ingin mencari gambar yang di inginkan klik "browse"
5. lalu klik ok jika sudah menemukan gambar yang dipilih- Arahkan pointer/kursor pada bidang kosong di Desktop 
- Kemudian klik kanan, lalu klik menu Properties 
- Selanjutnya pilih tab Backgrounds, lalu pilih gambar wallpapernya, atau kalau ingin memakai gambar sendiri/gambar lain, klik tombol Browse, cari gambarnya, lalu klik Open 
- Kalau sudah, klik Apply dan Ok

28. jelaskan istilah desktop,walpaper,screen saver,icon/shortcut


desktop adalah interface pertama yang di sajikan oleh sebuah system operasi, untuk mempermudah melakukan kegiatan berikutnya (tasking)

wallpaper adalah gambar latar(background) yang dapat di aplikasikan atau di terapkan pada suatu interface, misal. desktop wallpaper, folder wallpaper,

screen saver, adalah sebuah layar tampilan yang muncul beberapa saat bila suatu system komputer atau alat komputasi, sedang tidak di gunakan, fungsinya adalah penghematan daya dari suatu alat komputasi, karna saat system komputer aktif akan memerlukan daya yang besar, sedangkan dengan adanya fitur ini bisa menekan daya hingga konsumsi daya mengecil,

shortcut adalah sebuah jalan pintas untuk menjalankan (excute) suatu progam dengan meng klik icon tersebut, cara ini lebih menghemat waktu dari pada mencari program tersebut dalam list program menu atau file program dalam direktori penyimpan program
