10. Sebutkan dua cara memindahkan file atau folder!


Cara memindahkan file atau folder

1. Gunakan menu cut pada klik kanan file atau folder yang ingin dipindahkan, lalu pilih menu paste di tempat yang dituju

2. Gunakan drag (seret) klik kiri file atau folder yang ingin dipindahkan langsung seret ke tempat yang dituju, dan lepaskan kliknya.

Semoga membantupertama di copy dan paste ke tempat yang ingin dipindahkan atau bisa diggunakan fungsi dari cut untuk memindahkan kan file ke tempat yang lain
