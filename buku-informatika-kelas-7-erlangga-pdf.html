12. jawaban tugas buku informatika kelas 7 halaman 92​


Jenis perangkat lunak :

Microsoft power point : Aplikasi, Close Source, pengolah data presentasiApacheOpenOffice : Aplikasi suite (kumpulan aplikasi office), Open Source, pengolah data angka kata presentasi dan gambarUbuntu : Sistem Operasi, Open Source, Software Induk untuk dapat mengoperasikan komputerAdobe Photoshop : Aplikasi, Close Source, Pengolah gambar bitmapIOS = Sistem Operasi, Close Source, Software Induk untuk dapat mengoperasikan smartphone (smartphone khusus produk Apple)Kaspersky Anti Vitus: Aplikasi, Close Source, untuk sistem / aplikasi pertahanan dari serangan malware seperti virus trojan spyware botnet dllVLC Player : Aplikasi, Open Source, Multimedia Player yg dapat memutar berbagai format video dan audioSafari : Aplikasi, Close Source (hanya component webkit safari yg open source), Browser atau aplikasi penjelajah internet yg biasa dan umumnya ada di perangkat produk dari AppleSnapchat : Aplikasi, Close Source, aplikasi Instant Message video chat photo sharePembahasan :

      Software atau Perangkat Lunak adalah istilah khusus untuk data yang  disimpan dalam bentuk digital, termasuk program komputer, dokumentasinya, dan  informasi lain yang dapat dibaca dan ditulis oleh komputer. Dengan kata lain, itu adalah bagian tidak berwujud dari sistem komputer. Istilah ini menekankan perbedaan dengan perangkat keras komputer.

      Pengembangan perangkat lunak itu sendiri membutuhkan "bahasa pemrograman" yang dibuat oleh seorang programmer. Ini dikompilasi menggunakan aplikasi kompiler ke dalam kode yang dimengerti oleh mesin perangkat keras. Perangkat lunak seperti Windows dan Linux adalah sumber kehidupan komputer, tetapi komputer tidak dapat berfungsi tanpa sistem operasi yang diinstal. Perangkat lunak sistem operasi biasanya disimpan di partisi C, jadi jika terjadi masalah dengan komputer Anda, partisi C akan terpengaruh. Perangkat lunak lain juga disimpan di partisi C. Jika suatu komponen tidak lagi bekerja secara optimal, maka dapat di-uninstall dan dipasang kembali. Disarankan untuk menyimpan data lain  di partisi D. Dengan cara ini, jika perangkat lunak terkena virus atau berhenti bekerja dengan benar, Anda dapat menginstal ulang dan menyimpan data penting Anda.

Di bawah ini adalah contoh berbagai jenis perangkat lunak:

Perangkat lunak aplikasi  seperti pengolah kata, spreadsheet, pemutar media, dan suite kantor seperti OpenOffice.org. Sistem operasi seperti Linux dan Windows. Alat pengembangan perangkat lunak  seperti compiler untuk bahasa pemrograman tingkat tinggi seperti Pascal dan bahasa pemrograman tingkat rendah yaitu bahasa assembly. Pengontrol perangkat keras (pengandar perangkat) adalah koneksi antara perangkat tambahan perangkat keras  dan komputer, perangkat lunak yang banyak digunakan di supermarket dan sekolah. Dengan kata lain, gunakan pemindai kode batang di aplikasi basis data lain. Software/firmware akuntansi diinstal pada jam digital dan remote control. Internet

1A. Sebagai media komunikasi

1B. Sebagai ... (Media pencari informasi)

2A. Media sosial & Email (Surel)

2B. Peramban & Mesin Pencari

3A. Kelebihan daripada surat konvensional :

- Waktu pengiriman lebih cepat

- Dapat mengirim foto, video, audio, dll

- Hemat biaya

3B. Etika berkomunikasi menggunakan surel :

- Mengetahui cara bicara & menulis orang

- Membalas semua pesan

- Gunakan tanda baca dengan benar

4A. Perbedaan Peramban & Mesin Pencari

1.) Browser : Dapat di install

   Search Engine : Hanya di Internet saja

2.) Browser : Membuka website dengan format "www"

   Search Engine : Hanya mengetik kata kata yang ingin dicari

3.) Browser : Bisa diakses tanpa Search Engine

   Search Engine : Tidak bisa diakses tanpa Browser

4B. Cara mencari informasi menggunakan mesin pencari

- Memikirkan topik yang ingin dicari

- Ketik topik yang diinginkan di search engine apa saja

- Selesai, kamu dapat informasi informasi disana

Fail (File) & Folder

A. Perbedaan Fail & Folder

File : Berisi data data, foto, gambar, installer, dll

Folder : Tempat untuk menyimpan file

B. Cara Pengelolaan Fail & Folder

Menyalin File/Folder

1. Cari file/folder yang ingin di salin

2. Klik kanan mouse lalu pilih "Copy"

3. Klik kanan lagi lalu pilih "Paste"

Mengelompokkan File/Folder

1. Buat Folder dengan Klik kanan mouse, lalu pilih "New"

2. Pilih beberapa file/folder yang ingin dipindahkan dengan "Select"

3. Kemudian tarik ke folder yang dibuat tadi

Mengubah nama File/Folder

1. Pilih file/folder yang ingin diganti nama

2. Klik kanan mouse lalu pilih "Rename"

3. Lalu ganti nama sesuai yang diinginkan dan Pencet "Enter"

Memilih beberapa File/Folder

1. Pikirkan file/folder apa yang mau dipilih

2. Cari beberapa file/folder yang diinginkan

3. Lalu tahan "Ctrl" dan klik file/folder yang diinginkan

Mencari File/Folder

1. Pikirkan file/folder yang ingin dicari

2. Klik "Search" lalu ketik nama file/folder yang ingin dicari

3. Lalu Pencet "Enter"
