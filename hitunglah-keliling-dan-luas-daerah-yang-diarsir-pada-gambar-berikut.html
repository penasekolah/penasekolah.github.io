6. Hitunglah keliling dan luas daerah yang diarsir pada gambar berikut !


Hitunglah keliling dan luas daerah yang diarsir pada gambar berikut !

Lingkaran adalah kumpulan semua titik-titik pada bidang datar yang berjarak sama terhadap suatu titik tertentu (titik pusat).

Rumus

Panjang busur = [tex]\frac{\angle~ x}{360}[/tex] × 2 π r

Luas juring = [tex]\frac{\angle~ x}{360}[/tex] × π r²

PembahasanNo a.

∠ x = 72°

r = 21 cm

Keliling daerah yg diarsir

Panjang busur = [tex]\frac{\angle~ x}{360}[/tex] × 2 π r

                        = [tex]\frac{72}{360}[/tex] × 2 × [tex]\frac{22}{7}[/tex] × 21 cm

                        = [tex]\frac{1}{5}[/tex] × 2 × 22 × 3 cm

                        = 26,4 cm

Keliling diarsir = 2 r + panjang busur

                        = 2 (21 cm) + 26,4 cm

                        = 42 cm + 26,4 cm

                        = 68,4 cm

Jadi keliling daerah yg diarsir adalah 68,4 cm

Luas daerah diarsir

Luas juring = [tex]\frac{\angle~ x}{360}[/tex] × π r²

                  = [tex]\frac{72}{360}[/tex] × [tex]\frac{22}{7}[/tex] × 21 × 21 cm²

                  = [tex]\frac{1}{5}[/tex] × 22 × 3 × 21 cm²

                  = 277,2 cm²

Jadi luas daerah yg diarsir adalah 277,2 cm²

No b.

R = 18 cm

r = 12 cm

Keliling daerah yg diarsir

Panjang busur besar = [tex]\frac{1}{4}[/tex] × 2 π R

                                   = [tex]\frac{1}{4}[/tex] × 2 × 3,14 × 18 cm

                                   = 3,14 × 9 cm

                                   = 28,26 cm

Panjang busur kecil = [tex]\frac{1}{4}[/tex] × 2 π r

                                 = [tex]\frac{1}{4}[/tex] × 2 × 3,14 × 12 cm

                                 = 3,14 × 6 cm

                                = 18,84 cm

K daerah diarsir = 28,26 cm + 18,84 cm + 2 (18 - 12) cm

                          = 47,1 cm + 2 (6) cm

                          = 59,1 cm

Jadi keliling daerah yg diarsir adalah 59,1 cm

Luas daerah yg diarsir

L juring besar = [tex]\frac{1}{4}[/tex] × π R²

                      = [tex]\frac{1}{4}[/tex] × 3,14 × 18 × 18 cm²

                      = 3,14 × 9 × 9 cm²

                     = 254,34 cm²

L juring kecil = [tex]\frac{1}{4}[/tex] × π r²

                     = [tex]\frac{1}{4}[/tex] × 3,14 × 12 × 12 cm²

                     = 3,14 × 3 × 12 cm²

                     = 113,04 cm²

Luas daerah diarsir = 254,34 cm² - 113,04 cm²

                                = 141,3 cm²

Jadi luas daerah yg diarsir adalah 141,3 cm²

13. hitunglah keliling dan luas daerah yg di arsir pada gambar berikut


Lingkaran adalah kumpulan titik - titik pada suatu bidang dalam jarak tertentu. Rumus lingkaran adalah sebagai berikut :

K lingkaran = [tex] \pi[/tex] x d

L lingkaran = [tex] \pi[/tex] x r², dimana :

[tex] \pi [/tex]= [tex] \frac {22} {7}[/tex] jika r kelipatan 7  

[tex] \pi[/tex] = 3,14 jika r bukan kelipatan 7

r = jari - jari lingkaran

Pembahasan

a. Luas lingkaran tersebut adalah :

= [tex] \pi[/tex] x r² x [tex] \frac {60} {360}[/tex]  

= [tex] \frac {60} {360}[/tex] x [tex] \frac {22} {7}[/tex] x 21 cm x 21 cm

= [tex] \frac {60 : 60} {360 : 60}[/tex] x [tex] \frac {22} {7 : 7}[/tex] x ( 21 : 7 ) cm x 21 cm

= [tex] \frac {1} {6}[/tex] x [tex] \frac {22} {1}[/tex] x 3 cm x 21 cm

= [tex] \frac {1} {6 : 2}[/tex] x [tex] \frac {22 : 2} {1}[/tex] x 3 cm x 21 cm

= [tex] \frac {1} {3}[/tex] x [tex] \frac {11} {1}[/tex] x 3 cm x 21 cm

= [tex] \frac {1} {3 : 3}[/tex] x [tex] \frac {11} {1}[/tex] x ( 3 : 3 ) cm x 21 cm

= [tex] \frac {1} {1}[/tex] x [tex] \frac {11} {1}[/tex] x 1 cm x 21 cm

= 11 x 21 cm²

= 231 cm²

Keliling lingkaran tersebut adalah :

= [tex] \pi[/tex] x 2r x [tex] \frac {60} {360}[/tex] + 2r

= [tex] \frac {22} {7}[/tex] x 2 x 21 cm x [tex] \frac {60} {360}[/tex] + 2 x 21 cm

= [tex] \frac {22 : 2} {7 : 7}[/tex] x ( 2 : 2 ) x ( 21 : 7 ) cm x [tex] \frac {60 : 60} {360 : 60}[/tex] + 42 cm

= [tex] \frac {11} {1}[/tex] x 1 x 3 cm x [tex] \frac {1} {6}[/tex] + 42 cm

= 11 x ( 3 : 3 ) cm x [tex] \frac {1} {6 : 3}[/tex] + 42 cm  

= 11 x 1 cm x 2 + 42 cm

= 22 cm + 42 cm

= 64 cm

Sehingga keliling dan luas lingkaran tersebut adalah 64 cm dan 231 cm²

b. Luas daerah yg diarsir adalah :

= luas [tex] \frac {1} {4}[/tex] lingkaran luar - luas [tex] \frac {1} {4}[/tex] lingkaran dalam  

= [tex] \frac {1} {4}[/tex] x [tex] \pi[/tex] x r² - [tex] \frac {1} {4}[/tex] x [tex] \pi[/tex] x r²  

= [tex] \frac {1} {4}[/tex] x 3,14 x 18 cm x 18 cm - [tex] \frac {1} {4}[/tex] x 3,14 x 12 cm x 12 cm

= 254,34 cm² - 113,04 cm²

= 141,3 cm²

Keliling daerah yg diarsir dalah :

= keliling [tex] \frac {1} {4}[/tex] lingkaran luar + keliling [tex] \frac {1} {4}[/tex] lingkaran dalam + 2 x ( jari-jari lingkaran luar - jari-jari lingkaran dalam )

=  ( [tex] \frac {1} {4}[/tex] x 3,14 x 2 x 18 cm ) + ( [tex] \frac {1} {4}[/tex] x 3,14 x 2 x 12 cm ) + ( 2 x 18 cm - 12 cm )

= 28,26 cm + 18,84 cm + 2 x 6 cm

= 28,26 cm + 18,84 cm + 12 cm

= 59,1 cm

c. Luas lingkaran luar adalah :

= [tex] \pi[/tex] x r² x [tex] \frac {72} {360}[/tex]  

= [tex] \frac {72} {360}[/tex] x [tex] \frac {22} {7}[/tex] x 42 cm x 42 cm

= [tex] \frac {72 : 72} {360 : 72}[/tex] x [tex] \frac {22} {7 : 7}[/tex] x ( 42 : 7 ) cm x 42 cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {1}[/tex] x 6 cm x 42 cm

= [tex] \frac {1 x 22 x 6 x 42} {5}[/tex] cm²  

= [tex] \frac {5.544} {5}[/tex] cm²  

= 1108,8 cm²  

Luas lingkaran dalam adalah :

= [tex] \pi[/tex] x r² x [tex] \frac {72} {360}[/tex]  

= [tex] \frac {72} {360}[/tex] x [tex] \frac {22} {7}[/tex] x 42 cm x 42 cm

= [tex] \frac {72 : 72} {360 : 72}[/tex] x [tex] \frac {22} {7 : 7}[/tex] x ( 28 : 7 ) cm x 28 cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {1}[/tex] x 4 cm x 28 cm

= [tex] \frac {1 x 22 x 4 x 28} {5}[/tex] cm²  

= [tex] \frac {2.464} {5}[/tex] cm²  

= 492,8 cm²  

Luas arsir adalah :

= luas lingkaran luar - luas lingkaran dalam

= 1108,8 cm² - 492,8 cm²  

= 616 cm²  

Keliling lingkaran dalam adalah :

= [tex] \pi[/tex] x 2 r x [tex] \frac {72} {360}[/tex]  

= [tex] \frac {72} {360}[/tex] x [tex] \frac {22} {7}[/tex] x 2 x 28 cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {7 : 7}[/tex] x 2 x ( 28 : 7 ) cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {1}[/tex] x 2 x 4 cm

= [tex] \frac {1 x 22 x 2 x 4} {5}[/tex] cm

= [tex] \frac {176} {5}[/tex] cm

= 35,2 cm

Keliling lingkaran luar adalah :

= [tex] \pi[/tex] x 2r x [tex] \frac {72} {360}[/tex]  

= [tex] \frac {72} {360}[/tex] x [tex] \frac {22} {7}[/tex] x 2 x 42 cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {7 : 7}[/tex] x 2 x ( 42 : 7 ) cm

= [tex] \frac {1} {5}[/tex] x [tex] \frac {22} {1}[/tex] x 2 x 6 cm

= [tex] \frac {1 x 22 x 2 x 6} {5}[/tex] cm

= [tex] \frac {264} {5}[/tex] cm

= 52,8 cm

Keliling bangun tersebut adalah :

= keliling lingkaran dalam + keliling lingkaran luar + 2 x ( jari-jari lingkaran luar - jari-jari lingkaran dalam)

= 35,2 cm + 52,8 cm + 2 x ( 42 cm - 28 cm )

= 35,2 cm + 52,8 cm + 2 x 14 cm

= 35,2 cm + 52,8 cm + 28 cm

= 116 cm

d. Maaf, untuk yg D tidak bisa diselesaikan. Karena jari - jari lingkaran harus sama, yaitu jari - jari yg ke arah atas maupun jari - jari yg ke arah samping. Tolong di foto lagi supaya jelas penyelesaiannya.

Nomor 2

Luas daerah yang diarsir adalah jumlah dari luas persegi, luas ½ lingkaran dengan diameter sama dengan panjang persegi, dan luas ½ lingkaran dengan diameter sama dengan lebar persegi.

L = pl + ½·¼πd₁² + ½·¼πd₂²
⇔ L = pl + (1/8)·π·(d₁² + d₂²)
(d₁ = p, d₂ = l)

Ukuran panjang dan lebar dijadikan mm terlebih dahulu, agar lebih mudah menghitungnya (pendapat pribadi saja).
(p = 6,3 cm = 63 mm, l = 3,5 cm = 35 mm)

⇔ L = 2.205 + ¼·(11/7)·(7²·9² + 7²·5²)

⇔ L = 4.245,5 mm²
⇔ L = (4.245,5 / 100) cm²
⇔ L = 42,455 cm²

∴ Jadi, luas daerah yang diarsir adalah 42,455 cm².
