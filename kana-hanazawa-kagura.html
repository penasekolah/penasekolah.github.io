27. Uang kana ⅓ uang kinanti ½ uang kana Rp.6000 berapakah jumlah uang kana dan kinanti?


Uang kana = 1/3 uang kinanti, berarti uang kinanti = 3 x uang kana
1/2 uang kana = 6000, berarti uang kana = 2 x 6000 = 12000
Sehingga, uang kinanti = 3 x 12000 = 36000

Jumlah uang kana dan kinanti = 12000 + 36000 = 48000
