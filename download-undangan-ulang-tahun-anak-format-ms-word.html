10. Apakah dokumen Ms word dapat diubah menjadi format lain? Perintah apa yang digunakan?​


Jawaban:

Cara agar bisa merubah file dokumen menjadi file ebook dengan format pdf adalah :

Buka file berekstensi dokumen yang akan Anda ubah menjadi file berekstensi PDF (dalam kasus ini penulis menggunakan aplikasi Ms. Word 2013)

Buka tab file, dan pilih ekspor

Pilih "Buat PDF/XPS"

Lalu beri nama file berekstensi PDF yang Anda inginkan

Dan klik "terbitkan"

Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF

PEMBAHASAN :

Sebelum menjawab soal tersebut, mari kita mengenal terlebih dahulu file yang berekstensi pdf ^_^

→ Portable Docu

ment Format atau biasa disingkat dengan PDF, merupakan suatu ekstensi yang dikembangkan oleh perusahaan Adobe Inc pada tahun 1993.

→ Ektensi PDF ini memungkinkan user untuk dapat merubah baik file berupa dokumen (doc atau docx) dan gambar menjadi satu file berekstensi PDF yang praktis dengan kompabilitas tinggi.

→ Untuk dapat membuka file PDF, diperlukan aplikasi reader khusus seperti Adobe reader, Foxit PDF, Nitro PDF, dan banyak aplikasi pihak ke 3 lainnya.

→ Saat ini tingkat penggunaan file berekstensi PDF sangat tinggi, disebabkan oleh beberapa kelebihan yang dimilikinya, seperti :

membutuhkan ruang penyimpanan (memori) yang relatif kecil dibanding ekstensi lainnya

file dokumen berupa tulisan atau grafik akan dianggap sebagai gambar, sehingga memudahkan untuk perpindahan halaman saat membaca ebook

tidak adanya kekhawatiran terhadap kemungkinan salah tekan atau perubahan konten ebook yang tidak disengaja, karena file pdf akan menganggap isinya sebagai sebuah gambar yang solid, dan hanya bisa di edit saat memiliki aplikasi pdf editor yang rata-rata berbayar

lebih tahan akan kemungkinan kerusakan dokumen, di banding format docx

Lalu beri nama file berekstensi PDF yang Anda inginkan

Dan klik "terbitkan"

Selamat file berekstensi dokumen Anda sudah menjadi file berformat PDF :)

*Jika Anda memiliki aplikasi pihak ke 3, seperti foxit phantom PDF, maka pengubahan file dokumen atau gambar akan menjadi lebih mudah, yakni hanya dengan klik kanan kursor pada file yang akan diubah ekstensinya, lalu pilih "convert to pdf"

14. sub menu dari menu format yang terdapat dalam MS word



MENU FILE
New: berfungsi untuk membuat file atau workbook baru

Open: berfungsi untuk membuka file atau workbook yang telah disimpan

Close: menutup file atau workbook yang sedang aktif

Save: menyimpan fille atau workbook yang saat ini sedang aktif

Save as: menyimpan sebuah file atau workbook dengan nama serta lokasi penyimpanan lain

Page setup: mengatur tata letak halaman sebelum dicetak

Print preview: menampilkan file yang dicetak pada tampilan monitor

Print: perintah untuk mencetak dokumen melalui printer

Exit: keluar dari program Microsoft Word 

MENU EDIT
Undo: berguna untuk membatalkan perintah yang telah kita lakukan sebelumnya

Redo: berguna untuk membatalkan perintah undo

Cut: berguna untuk memotong (sebagai proses memindahkan teks / obyek dengan cara cut-paste)

Copy: menyalin teks / obyek

Paste: meletakkan objek / teks yang telah dicopy atau dipaste

Paste special: meletakkan salinan / potongan dengan berbagai format

Select all: memilih (memblok) seluuruh dokumen

Find: mencari kata pada kata pada dokumen

Replace: mengganti

Go to: melompat

MENU VIEW
Normal: menampilkan lembar kerja word dalam tampilan normal

Web layout: membuat tampilan web

Prin layout: membuat tampilan dalam mode cetak

Task pane: menampilkan task pane atau jendela tugas

Toolbar: menampilkan / menyembunyikan toolbar

Ruler: Menampilkan / menyembunyikan bar rumus

Full screen: menampilkan lembar kerja dalam satu layar

Zoom: memilih besarnya tampilan lembar kerja 

MENU INSERT
Break: menyisipkan secsion

Page number: memberi nomor pada halaman

Auto text: menyisipkan teks otomatis

Symbol: menyisipkan simbol

Picture: menyisipkan gambar

Diagram: menyisipkan diagram

Text box: menyisipkan teks box

File: menyisipkan file

MENU FORMAT


Font: memformat font

Paragraph: memformat paragraph

Bullet and numbering: menyisipkan format bullet dan numbering

Border and shading: memformat garis dan bayangan

Columns: menerapkan format berkolom

Tab: memformat tab

Drop cap: mengatur format huruf yang lebih menonjol dari huruf lain

Change case: mengubah besar atau kecilnya huruf 

MENU TOOLS

Spelling and Grammar: Menjalankan pemeriksaan ejaan dan tata bahasa dari teks atau paragraf dalam sebuah dokumen

Language: menjalankan fasilitas penerjemah bahas

Autosummarize: Memberikan data statistik tentang jumlah huruf dan kata pada dokumen aktif

Protect Document: Melindungi dokumen agar aman dari bentuk pengubahan dokume

Letters and Mailings: Memuat surat melalui fasilitas wizar

Macro: Membuat dan menghapus macro sendir

Template and add-ins: Mengatur tampilan templat

AutoCorrect Options: Memperbaiki teks secara otomati

Customize: Menata toolbar, menambahkan perintah bari, membuat menu da toolbar                        sesuai keinginan

Options: mengatur berbagai hpilihan yang telah disediakan pada program


MENU TABLE


Draw table: menggambar tabel

Insert: menyisipkan tabel, baris, kolom dan sel

Delete: mengahpus tabel, baris kolom dan sel

Select: memilih tabel, baris, kolom dan sel

Marge cell: menggabungkan beberapa sel menjadi satu

Split cell: membelah sel mejadi beberapa sel

Sort: mengurutkan data dalam tabel 


MENU WINDOWS
New window: membuat jendela baru

Arrange all: membuat jendela - jendela yang aktif

Split: memecah jemdea kerja menjadi 2 bagian 

MENU HELP

Microsoft word help: membuka fasilitas bantuan (help) pada Microsoft Word

Show the fice of assistent: menampilkan asisten program

What's this?: menayangkan tentang sesuatu topik dengan mengklik topik tersebut

Office on the web: mencari bantuan secara online melalui internet
