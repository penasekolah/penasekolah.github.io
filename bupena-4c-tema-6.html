16. 3ab+4c=72ab, jika a=6-4c maka c


a=6-4c

3ab+4c=72ab

3b(6-4c) + 4c = 72b(6-4c)

18b - 12bc + 4c = 432b - 288bc

432b - 18b -288bc + 12bc - 4c = 0

414b - 276bc - 4c = 0

276bc + 4c = 414b

c(276b + 4) = 414b

c = 414b / (276b + 4)

25. 3ab+4c=72ab jika a=6-4c berapa c


Aljabar


→ 3ab + 4c = 72

→ 3(6 - 4c)b + 4c = 72

→ 18b - 12bc + c = 72

  18b + c(1 - 12b) = 72

  c = (72 - 18b) / (1 - 12b)
