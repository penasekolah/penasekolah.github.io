1. Tutorial membuat game tebak Kata dengan bahasa c++


Berikut programnya, untuk mengubah menjadi 1-100 kamu hanya perlu mengubah "rand()%100"
#include <iostream>
#include <cstdlib>
using namespace std;
int main()
{
    int arr[1];
    arr[0]=rand()%10;
    cout << "-----------Tebak Angka dari 1-10 (Cari angka X)--------\n";

    for (int i=2; i>=0; i--)
    {

        cout << "Kesempatan mu untuk menebak adalah sebanyak " << i+1 << "kali\n\nMasukkan Angka: ";
        int tebak;
        cin >> tebak;
        if (arr[i]==tebak)
        {
            cout << "Angka X adalah " << tebak << "\nTebakanmu benar!\n";
            return 0;
        }
        else if (i>0) cout << "\nAngka yang ditebak salah!\n" << endl;
        else cout << "\nKamu gagal menebak angka.\n";
    }
}

Berikut adalah beberapa kata baku yang terdiri dari 5 huruf dan diakhiri oleh dua huruf I dan S selain KAMIS yaitu

GADISBARISHABISMANISSADIS

Kata baku adalah kata yang sudah sesuai dengan ejaan kaidah bahasa Indonesia dan memiliki arti yang sudah jelas berdasarkan Kamus Besar Bahasa Indonesia (KBBI). Kata baku biasanya digunakan pada situasi yang resmi baik berupa tulisan seperti surat, laporan, buku dan sebagainya, maupun berupa pengungkapan atau perkataan seperti tes wawancara, diskusi secara formal dan sebagainya.

Pembahasan

Berikut adalah 20 kata baku yang terdiri dari 5 huruf dan diakhiri 2 huruf yaitu I dan S secara berurutan selain kata “KAMIS”.

AUTIS ARTIS BARIS BASIS BETIS BUGIS ETNIS GADISGARIS HABIS JENISKERIS KUDISLARISLAPISMANIS MIRISSADISTIPISTURIS

Berikut adalah beberapa pengertian dari kata-kata di atas.

AUTIS ⇒ suatu kondisi pada seseorang yang mengalami gangguan pada sistem syarafnya.ARTIS ⇒ seniman (penyanyi, pemain film) .BARIS ⇒ keadaan berjajar atau berderet dari depan ke belakang.BASIS ⇒ asas atau dasarBETIS ⇒ bagian dari kaki antara pergelangan kaki dan lutut.BUGIS ⇒ nama suku bangsa di Sulawesi Selatan.ETNIS ⇒ kelompok sosial.GADIS ⇒ panggilan perempuan yang belum menikah.GARIS ⇒ kumpulan dari titik-titik yang berkesinambungan.
