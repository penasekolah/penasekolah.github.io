5. tentukan sin,cos,tan pada segitiga siku siku di bawah ini​


Ingat rumus sin cos tan ( lihat gambar )

Dik : a ( depan ) = √3

b ( samping ) = 1

c ( miring ) = ?

c = √a^2 + b^2

= √(√3)^2 + 1^2

= √3 + 1

= √4

= 2

Dit : sina, cosa, tana ?

sin a = depan / miring

= a / c

= √3 / 2

cos a = samping / miring

= b / c

= 1 / 2

tan a = depan / samping

= a / b

= √3 / 1

= √3

Maka sin a = √3 / 2

cos a = 1 / 2

tan a = √3

9. perbandingan trigolometri segitiga siku siku (sin,cos,tan)


sin x = sisi depan / sisi miring

cos x = sisi samping / sisi miring

tan x = sisi depan / sisi samping

keterangan:

x adalah sudut pada segitiga siku-siku, namun x bukan sudut siku-siku.

sisi depan adalah sisi yang berada di depan sudut x.

sisi samping adalah sisi yang berada di samping sudut x, namun bukan sisi miring.

sisi miring adalah sisi yang berada di depan sudut siku-siku.

catatan:
rumus perbandingan trigonometri pada segitiga siku-siku di atas berlaku umum. sisi-sisi di atas tergantung pada posisi sudut x berada dan juga nama segitiga siku-siku itu sendiri beserta posisi sudut siku-sikunya.

*Saran saya, agar lebih jelas silahkan kamu gambar segitiga siku-siku.
