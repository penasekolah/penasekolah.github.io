Pengalaman Beraktivitas Selama Pandemi Covid 19 (2020-2022)

Di awal pandemi, saya mengalami mental shock karena perubahan aktivitas saya sehari-hari, yang membuat saya stres dan putus asa. Namun, setelah saya beradaptasi, saya mencoba mencari metode terbaik untuk mengatur ulang kebiasaan saya agar terhindar dari mental shock.

Hal pertama yang saya lakukan adalah membuat perencanaan. Saya tinggal di Depok dan kota saya saat ini memberlakukan lockdown, jadi saya harus tinggal di rumah terus-menerus. Untungnya, jadwal kelas saya tidak terlalu padat; oleh karena itu, saya memiliki banyak waktu luang di rumah. Jadi, saya memutuskan untuk membuat perencanaan waktu agar lebih efektif.
