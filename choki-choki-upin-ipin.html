4. Banyaknya uang tantri dan choki berjumlah Rp.300.000 . Jika uang tantri Rp.175.000,00, maka perbandingan uang tantri dan choki adalah .. a.5:12 c.12:7 b.12:5 d.7:12


Jawaban:

c. 12 : 7

Penjelasan dengan langkah-langkah:

perbandingan uang tantri dan choki =

300.000 : 175.000

300.000 : 25.000 = 12

175.000 : 25.000 = 7

jadi perbandingan nya = 12 : 7

Jawab:

misalkan uang tantri = T, dan uang choki = C

diketahui

T + C = 300.000 --> C = 300.000 - T

T = 175.000

ditanyakan

T : C = ?

penyelesaian

C = 300.000 - T

   = 300.000 - 175.000

   = 125.000

 T : C

= 175.000 : 125.000

= 7 : 5

Maka perbandingan uang tantri dan choki adalah 7 : 5

Banyak kelereng Eben : 50 butir

30. Kecepatan mobil choki 90km/jam. Jarak sukabumi-tasik 60km. Jika Choki berangkat dari sukabumi pukul 10.00. Pukul berapa tiba di tasik?


PEMBAHASAN

Diketahui:

K = 90 km/jam

J = 60 km

berangkat = 10.00

Ditanya:

tiba = ?

Jawaban:

Menentukan lama perjalanan

[tex]w = \frac{j}{k} [/tex]

[tex]w = \frac{60 \: km}{90 \:km/jam } [/tex]

[tex]w = \frac{60}{90} \: jam[/tex]

[tex]w = \frac{(60 \div 30)}{(90 \div 30)} \: jam[/tex]

[tex]w = \frac{2}{3} \: jam[/tex]

[tex]w = \frac{2}{3} \times 60 \: menit[/tex]

[tex]w = 2 \times 20 \: menit[/tex]

[tex]w = 40 \: menit[/tex]

Menentukan waktu tiba

tiba = berangkat + lama

tiba = 10.00 + 40 menit

tiba = 10.00 + 00.40

tiba = 10.40

Kesimpulan:

Jadi, Choki akan tiba di Tasik pada pukul 10.40.

بِسْـــمِ اللَّهِ الرَّحْمَــنِ الرَّحِيْمِ

Diketahui:

Kecepatan (K) = 90 km/jamJarak (J) = 60 kmBerangkat pada pukul 10.00

Ditanyakan:

Tiba pada pukul?

Penyelesaian:

Menentukan waktu perjalanan

W = J/K

= 60/90 (masing² bagi 30)

= 2/3 jam

Karena 2/3 jam kurang dari 1 jam, kita dapat mengubahnya menjadi satuan menit dengan mengalikannya dengan 60 (1 jam = 60 menit).

W = ⅔ × 60

= 2 × 20

= 40 menit

Menghitung pukul tibanya

Cara menghitungnya adalah dengan menjumlahkan pukul berangkat dengan waktu perjalanan. Sehingga:

Pukul tiba = 10.00 + 00.40

= 10.40

Kesimpulan:

Jadi, Choky akan tiba di Tasik pada pukul 10.40
