28. Quiss 22⁷ + 8⁸ Note: Gamabr anime couple Jan diambil ya heheh :-:


Maka Hasil dari 22⁷ + 8⁸ adalah 2.511.135.104

Pendahuluan

Apa yang dimaksud Perpangkatan ?

Perpangkatan adalah perkalian berulang atau berganda bilangan dengan faktor-faktor bilangan yang sama.

Rumus :

a¹ = a

a² = a x a

a³ = a x a x a

Pembahasan

diketahui :

22⁷ + 8⁸

ditanya :

Hasil ?

Jawab :

= 22⁷ + 8⁸

= (22 x 22 x 22 x 22 x 22 x 22 x 22) + (8 x 8 x 8 x 8 x 8 x 8 x 8 x 8)

= 2.494.357.888 + 16.777.216

= 2.511.135.104

Kesimpulan :

Maka Hasil dari 22⁷ + 8⁸ adalah 2.511.135.104
