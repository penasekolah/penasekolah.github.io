21. 1 .Apa yang dimaksud dengan Folder dan File? 2. Apa perbedaan dari Folder dan File?Tolong dijawab jam 9 malam dikumpulkan,, makasih :)"berbagi ilmu itu menyenangkan"​


Jawaban:

Ada suatu alat atau benda yang dipakai sebagai tempat menyimpan file berupa kertas yang agar file-file kertas tersebut dapat terlindung dari debu dan tidak mudah kusut. Di Amerika Serikat benda seperti itu dinamakan File Folder, sedangkan di Inggris dan Australia dinamakan Folder.

Jawaban:

Yang Dimaksud Folder dan File?

Folder: Folder adalah tempat untuk menyimpan atupun menampung file-file, baik itu file system maupun file data atau dokumen. Icon folder sendiri di computer umumnya berbentuk kecil seperti amplop berwarna cokelat. Folder dapat kita buat pada desktop computer maupun pada drive-drive dalam computer sesuai dengan kebutuhan.

File: File yaitu kumpulan data-data baik berupa angka, teks, gambar, video, slide, program, dan sebagainya, yang di beri nama secara digital.

Perbedaan Folder dan File?

Folder berfungsi sebagai wadah untuk file dan folder lain, tetapi file tidak dapat berisi file atau folder. File adalah unit penyimpanan untuk data, mereka dapat mengambil banyak bentuk, termasuk dokumen teks, trek audio, klip film, dan program komputer.

Anda dapat membuat folder atau sub-folder baru di dalam folder, tetapi Anda tidak dapat membuat folder baru di dalam file.
