18. Buatlah query untuk: ▪ Menampilkan paket yang belum dikirim ▪ Menampilkan biaya pengiriman paket dengan kode resi RS0000112 ▪ Mengupdate biaya pengiriman dari Jogja ke Semarang pliss donk bantu aku.... mas,mbk,kak


Jawaban:

SQL

SELECT * FROM tableName WHERE Dikirim = 0;

SELECT BiayaPengiriman FROM tableName WHERE KodeResi = 'RS0000112';

UPDATE tableName SET BiayaPengiriman = 25000 WHERE (Dari = 'Jogja' AND Tujuan = 'Semarang');

Penjelasan:

Silahkan sesuaikan penamaan table dan variabel-variabel lainnya sesuai dengan soal.

SELECT Statement: Select Statement digunakan untuk memilih sebuah data dari database.

UPDATE Statement: Update Statement digunakan untuk mengganti data yang sudah ada di dalam database.

SELECT * FROM tableName WHERE Dikirim = 0;

Contoh ini mengasumsi ada column "Dikirim" yang valuenya 0 bila belum dikirim dan 1 bila sudah dikirim.

Kurang lebih, kalau di translate, bacanya seperti, PILIH * DARI tableName DIMANA Dikirim = 0;

Jadi, column *(semua) akan dipilih dari 'tableName' yang isi column 'Dikirim' adalah 0(atau belum dikirim)

SELECT BiayaPengiriman FROM tableName WHERE KodeResi = 'RS0000112';

Contoh ini mengasumsi ada column "BiayaPengiriman" yang valuenya merupakan biaya pengiriman dan ada column "KodeResi" yang valuenya merupakan kode resi.

Kurang lebih, kalau di translate, bacanya seperti, PILIH BiayaPengiriman DARI tableName DIMANA KodeResi = 'RS0000112';

Jadi, column BiayaPengiriman akan dipilih dari 'tableName' yang isi column 'KodeResi' adalah RS0000112.

UPDATE tableName SET BiayaPengiriman = 25000 WHERE (Dari = 'Jogja' AND Tujuan = 'Semarang');

Contoh ini mengasumsi ada column "BiayaPengiriman" yang valuenya adalah biaya pengiriman, ada column "Dari" yang valuenya adalah asal pengiriman, dan ada column "Tujuan" yang isinya adalah tujuan pengiriman.

Kurang lebih, kalau di translate, bacanya seperti, GANTI tableName SET BiayaPengiriman = 25000 DIMANA (Dari = 'Jogja' DAN Tujuan = 'Semarang');

Jadi, isi dari column BiayaPengiriman akan diupdate menjadi 25000 dari 'tableName' yang isi column 'Dari' adalah Jogja dan isi column 'Tujuan' adalah 'Semarang'.

Jawaban:

Alasan mengapa kita perlu untuk menuliskan kode pos pada formulir di bagian alamat saat berkirim adalah untuk memudahkan petugas pengiriman (kurir) menemukan wilayah dari alamat yang kita tuju sebab kode pos sendiri sebenarnya adalah kode wilayah.

Penjelasan:

Kode Pos ini di luar negeri disebut dengan ZIP CODE. Kode Pos bukan asli dari Indonesia melainkan dari Jerman. Negara ini pertama kali menerapkan sistem Kode Pos di tahun 1941, tujuannya adalah untuk memudahkan menemukan wilayah dari alamat yang menjadi tujuan pengiriman.

Sebenarnya tanpa kode pos pun barang yang dikirim bisa sampai asal alamat yang dituliskan dilengkapi dengan nama nomor rumah, nama jalan, nama kompleks/perumahan, nama desa, nama kecamatan, nama kabupaten dan juga informasi tambahan lainnya seperti nomor telepon. Meski begitu, beberapa formulir (termasuk pengiriman) tetap mensyaratkan adanya informasi kode pos agar pencarian alamat lebih jelas dan lengkap lagi.

27. cara mengatasi game saat di pasang terjadi masalah saat mengurai paket di apkeditor​


Cara Memperbaiki "Kesalahaan Mengurai Paket" Pada Aplikasi Android :

Cara memperbaiki:

Buka file Apk menggunakan APK Editor Pro

Masuk kebagian folder Manifest

CARA YANG KE -1:

Cari "android:minSdkVersion", bila tertulis kode angka "21" silahkan rubah menjadi kode "14"

Cari "android:targetSdkVersion", bila tertulis kode "22" silahkan rubah menjadi kode "17"

Selesai & silahkan Rebuild

CARA YANG KE -2:

Masih sama di bagian folder Manifest.xml

Silahkan cari file "com.android.vending.BILLING" silahkan rubah menjadi "com.android.vending"

Selesai & Rebuild

CARA YANG KE -3:

Masuk ke bagian file

Cari folder "lib" bila tdk sesuai dengan basic perangkat

Contoh:

lib/armeabi

lib/armeabi-v7a

lib/arm64-v8a

lib/x86

Bila tidak terdapat salah satu folder lib yang tidak sesuai dengan perangkatmu, anda bisa mencari aplikasi yg sesuai dengan basic perangkat atau unduh via PlayStore langsung. Setelah anda unduh silahkan backup menjadi file Apk. Lalu extract file tersebut dan cari folder "lib" nya lalu copy paste kan ke Aplikasi yg tadi (yang tidak support). Setelah itu silahkan Rebuild

Kesimpulan nya adalah :

Sampai saat ini masalah yang menimbulkan munculnya kesalahan saat mengurai paket ketika instalasi aplikasi, adalah kedua hal diatas. Jadi kesimpulannya jika bukan karena persyaratan OS Android, kemungkinan besar akibat berkas APK yang tidak lengkap saat mengunduh.
