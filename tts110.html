Barisan aritmatika adalah barisan dimana setiap suku berikutnya diperoleh dengan menambahkan konstanta yang sama pada suku yang sebelumnya. 21. sederhanakan bentuk dibawah ini a. sin 70. cos20+cos 70. sin 20 b. sin 110. cos50-cos110.sin 50


TRIGONOMETRIRumus trigonometri Sudut Rangkap

Sin (A + B ) = sin A . cos B + cos A . sin B

Sin (A - B) = sin A . cos B - cos A . sin B

Cos (A + B) = cos A . cos B - sin A . sin B

Cos (A - B) = cos A . cos B + sin A . sin B

Tan (A + B) = (tan A + tan B) / (1 - tan A. tan B)

Tan (A - B) = (tan A - tan B) / (1 + tan A . tan B)

mari kita bahas soal

a. sin 70. cos20 + cos 70. sin 20

rumus →    sin A. cos B + cos A . sin B = sin (A + B)

                 sin 70. cos20 + cos 70. sin 20 = sin (70 + 20)

                                                                    = sin 90

                                                                    = 1

b. sin 110. cos50 - cos110.sin 50

rumus → sin A . cos B - cos A . sin B = sin (A - B)

              sin 110. cos50 - cos110.sin 50 = sin (110 - 50)

                                                                = sin 60

24. Sin200°+sin80°/cos170°-cos110°


Jawaban:

Sin 200° + Sin 80° / Cos 170° - Cos 110°

Gunakan prinsip Sin(t) + Sin(s) = 2 sin ( t + s / 2 ) Cos ( t - s )

= 2 Sin ( 200 + 80 / 2 ) Cos ( 200 - 80 / 2 )

= 2 Sin ( 280 / 2 ) Cos ( 120 / 2 )

= 2 Sin ( 140° ) Cos ( 60° )

Lalu untuk Cos,

Gunakan prinsip

Cos(t) - Cos(s) = -2 Sin ( t + s / 2 ) Sin ( t - s / 2 )

= -2 Sin ( 170 + 110 / 2 ) Sin ( 170 - 110 / 2 )

= -2 Sin ( 280/2 ) Sin ( 60/2)

= -2 Sin ( 140°) Sin (30°)

= 2 Sin ( 140°) Cos ( 60°) / -2 Sin ( 140°) Sin ( 30° )

Misal 60° = selisih

= 2 Sin ( 140°) Cos ( 90° - 30°) / -2 Sin ( 140°) Sin ( 30° )

Lalu, sederhanakan penyebut ( -2 )

= -Sin ( 140°) Cos ( 90° - 30°) / Sin ( 140°) Sin ( 30° )

Lalu sederhanakan kembali dengan faktor Sin ( 140°)

= -Cos ( 90° - 30° ) / Sin ( 30° )

Dengan menggunakan bentuk Cos ( 90° - t ) = Sin(t), maka

= -Sin (30°) / Sin ( 30° )

= -1
