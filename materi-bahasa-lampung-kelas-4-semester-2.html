21. 1. 10kata benda bahasa lampung 2. 10kata sifat bahasa lampung


Kata benda :

Abik : kain panjang untuk menggendong

Acingacing : benda kecil yang menyembul keluar, seperti jarum, duri, lidi.

Ajang : perlengkapan makan

Akkuk : pencabut bulu(pinset)

Akkos : kain untuk ikat pinggang

Akhi : tiang

Apay/sulan : tikar

Apung : pelampung jaring

Asahan : batu asahan

Awi/Kawokh : bambu

Kata sifat :

Pekhkhos : 1.rasa masam, 2.asam sebagai bumbu dapur

Haying : gelisah

Hasud : fitnah, Ngahasud : memfitnah

Hellaw : baik, bagus

Liyom : malu

Sikap/sikop : cantik

Abak : besar, deden

Pahik : pahit

Cadang : rusak

Hakal : akal, nalar, kecerdasan, berpikir dewasa
