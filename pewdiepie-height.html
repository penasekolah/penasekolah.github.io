9. The average average height of a good is 142 cm theheightis 138 cm. what is the height of the other girls?​


Jawab:

Penjelasan dengan langkah-langkah:

Diketahui :

Rata-rata tinggi perempuan = 142 cm

x₁ = tinggi perempuan pertama = 138 cm

Ditanya : x₂

Jawab :

Rata-rata

=> (x₁ + x₂ ) / 2 = 142

=> (138 + x₂ ) = 2 * 142 = 284

=> x₂ = 284 - 138 = 146 cm

Tinggi perempuan kedua = 146 cm

13. The height of two plants is in the ratio 7:9.the height of the taller plant is 26 cm more than the height of the shorter plant.what is their total height?​


s : t = 7 : 9 <=> 9s = 7t (s = shorter, t = taller)

t = s + 26

9s = 7t

9s = 7(s + 26)

9s = 7s + 182

2s = 182

s = 91

So

s + t

= s + s + 26

= 2s + 26

= 182 + 26

= 208 cm

15. 3/4 of Jevan’s height is equal to 2/5 of Nicslie’s height. The difference in their height is 77 cm . What is Nicslie’s height in cm?


Jawaban:

jevan height = x

Nicslie height = y

3/4x = 2/5y

x + 77 = y

use substitution method

¾x = ⅖ (x + 77)

¾x = ⅖x + ¹⁵⁴/5

¾x - ⅖x = ¹⁵⁴/5

¹⁵/20x - ⁸/20x = ¹⁵⁴/5

⁷/20x = ¹⁵⁴/5

x = ¹⁵⁴/5 ÷ ⁷/20

x = ¹⁵⁴/5 x 20/7

x = ⁴⁴⁰/5

x = 88cm

Jevan's height is 88cm

¾x = ⅖y

¾ • 88 = ⅖y

66 = ⅖y

y = 66 ÷ ⅖

y = 66 x ⁵/2

y = 165

thus Nicslie's height is 165cm
