15. jelaskan 3 perbedaan antara URL ghoper dengan URL file!


URL (Uniform Resource Locator)

-URL digunakan oleh sebuah browser untuk mengidentifikasi sumber daya yang ada di internet yang dituliskan dalam satu baris teks.
-URL merupakan bagian dari URI
-URL menunjuk ke suatu halaman web yang menggunakan “scheme” tanpa menggunakan code fragment.
Contoh :
http://binary-project.com/webdeveloper.html

URI (Uniform Resurce Identifier)

-URI digunakan browser untuk mengidentifikasi dokumen tunggal dan dituliskan dalam satu baris teks.
-URI digunakan untuk mengidentifikasi suatu resource dengan lokasi atau nama, ataupun keduanya.
-URI menunjuk ke suatu halaman web, termasuk lokasi code fragmentnya (apabila ada) dan “scheme”.

16. Apa itu URL dan fungsinya


URL(Unifrom Resource Locator) yaitu suatu rangkaian karakter menurut standar tertentu yang digunakan untuk menunjukan suatu alamat dari sumber, seperti file atau gambar yang ada di Internet.
fungsinya :
1.Untuk pengidentifikasi suatu dokumen yang terdapat pada Web.
2.Digunakan untuk memberikan nama dokumen yang terdapat pada web.
3.Untuk memberian alamat pada setiap Web yang berisikan dokumen.
4.Mempermudah dalam mengakses dokumen atau file melalui suatu web.
5.Mempermudah dalam mengingat alamat dokumen yang ada dalam suatu web.
