1. jika besar <AOE+<ACE+<ADE=225°,hitunglah <AOE ?


Kelas 8 Matematika
Bab Lingkaran

∠ABE = ∠ACE = ∠ADE = sudut keliling

∠ABE + ∠ACE + ∠ADE = 225°
3 . ∠ACE = 225°
∠ACE = 225°/3
∠ACE = 75°

∠AOE = 2 . ∠ACE
∠AOE = 2 . 75°
∠AOE = 150°
Jawab = 
∠ABE + ∠ACE + ∠ADE = 225°
jadi panjang salah satu sisinya adalah 225 : 3 = 75 
karena ∠ABE, ∠ACE,dan ∠ADE adalah sudutu keliling dan ∠AOE adalah sudut pusat
maka, 
sudut keliling = 1/2 sudut pusat
sudut pusat = 2x sudut keliling 
jadi, 
∠AOE = 75×2 = 150°

2. maka besar sudut AOE adalah


96:3=32
AOE= 32x2=64 derajat (c)

Mapel : Matematika
Kelas : 8/VIII
Bab : Lingkaran
Sub Bab : Sudut Pusat dan Sudut Keliling

Pembahasan Jawaban :

Diketahui :
<ABE + <ACE + <ADE = 96° : 3 = 32° (merupakan sudut keliling)

Ditanya :
Besar <AOE (merupakan sudut pusat)?

Jawab :
<AOE = 2 × sudut keliling
<AOE = 2 × 32°
<AOE = 64°

Opsi ➡ C

Informasi Tambahan : Besar sudut pusat dua kali sudut keliling jika menghadap busur yang sama.

10. Pada gambar diatas, besar sudut AOE adalah


Garis dan Sudut

<DOE = <AOB (bertolak belakang)
<DOE = 4p°

<EOF + <DOE + <COD = 180°
3p + 4p + 96 = 180
7p = 180 - 96
7p = 84
p = 84/7
p = 12°

<EOF = 3p
<EOF = 3 x 12
<EOF = 36°

<AOF = <COD (bertolak belakang)
<AOF = 96°

<AOE = <AOF + <EOF
<AOE = 96 + 36
<AOE = 132°

Jadi besar <AOE adalah 132° (C).
