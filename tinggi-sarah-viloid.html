r = 20 cm

Dijawab :

Tentukan panjang garis pelukis (s)

[tex]s = \sqrt{ {r}^{2} + {t}^{2} } [/tex]

[tex]s = \sqrt{ {21}^{2} + {20}^{2} } [/tex]

[tex]s = \sqrt{441 + 400} [/tex]

[tex]s = \sqrt{841} [/tex]

[tex]s = 29 \: cm[/tex]

Tentukan Luas permukaan (Lp)

Lp = π x r x (r + s)

Lp = 3,14 x 20 x (20 + 29)

Lp = 62,8 x 49

Lp = 3077,2 cm²
