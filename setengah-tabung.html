• tabung

V₁ = 120 cm³
t₂ = ½ t₁, pada jari-jari tetap
V₂ = __?

volume tabung
V = π r² t

perbandingan volume
V₂ / V₁ = (π r² t₂) / (π r² t₁)
V₂ / V₁ = t₂ / t₁
V₂ / V₁ = t₂ / t₁
V₂ / 120 = (½ t₁) / t₁
V₂ / 120 = ½
V₂ = ½ • 120
V₂ = 60 cm³ ✓
