Kuartil terdiri dari tiga macam yaitu :

> Kuartil tengah/median ( Q2 )  

> kuartil atas ( Q3 )  

> kuartil bawah ( Q1 )  

Dalam mencari nilai kuartil, dibedakan menjadi dua macam yaitu:  

1. kuartil data genap  

2. kuartil data ganjil.

Jangkauan Interkuartil ( H )

Jangkauan interkuartil atau Hamparan ( H ) adalah selisih antara kuartil ketiga ( Q3 ) dengan kuartil pertama ( Q1 ). Rumus jangkauan interkuartil = Q3 - Q1

Simpangan Kuartil ( Qd )

Simpangan Kuartil atau disebut Jangkauan Semi Antar Kuartil ( Qd ) yaitu setengah dari selisih kuartil ketiga dengan kuartil pertama. Maka Qd yaitu = [tex]\frac {1}{2}[/tex] x ( Q3 - Q1 )

Sedangkan Jangkauan ( Range ) adalah selisih data terbesar dengan data terkecil.

Pembahasan

1. Urutkan data dari terkecil sampai yg terbesar terlebih dahulu : 5, 6, 6, 7, 7, 7, 7, 8, 9

a. Menentukan kuartil bawah, median dan kuartil atas

Menentukan kuartil bawah

Tentukan kuartil bawah dengan membagi data menjadi dua bagian sama besar.

5, 6, 6, 7

      |                

kuartil bawah  

     Q1  

Q1 = [tex]\frac {6 + 6}{2}[/tex]    

Q1 = [tex]\frac {12}{2}[/tex]    

Q1 = 6

Sehingga diperoleh kuartil bawah adalah 5,5.

Menentukan kuartil atas

Tentukan kuartil atas dengan membagi data menjadi dua bagian sama besar.

7, 7, 8, 9

     |                

kuartil atas

    Q3

Q3 = [tex]\frac {7 + 8}{2}[/tex]    

Q3 = [tex]\frac {15}{2}[/tex]    

Q3 = 7,5

Sehingga diperoleh kuartil atas adalah 7,5.

b. Menentukan jangkauan interkuartil dan simpangan kuartil

Jangkauan interkuartil pada data tersebut yaitu :

= Q3 - Q1

= 7,5 - 6

= 1,5

Jangkauan interkuartil data tersebut adalah 1,5

Dan simpangan Kuartil ( Qd ), yaitu :

= [tex]\frac {1}{2}[/tex] x ( Q3 - Q1 )

= [tex]\frac {1}{2}[/tex] x 1,5

= 0,75

Sehingga diperoleh bahwa simpangan kuartil data tersebut yaitu 0,75
