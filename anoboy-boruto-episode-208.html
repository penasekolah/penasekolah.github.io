3. Berapakah (akar dari 208) √208 ?


Hasil dari [tex]\sqrt{208}[/tex] adalah [tex]4 \sqrt{13}[/tex]

Pembahasan

Jika hasilnya tidak bulat, maka bilangan tersebut masuk ke dalam bilangan irrasional. Bilangan irrasional adalah tidak dapat dibagi atau hasilnya cukup panjang. Seluruh bilangan irrasional didalamnya adalah hasil dari akar kuadrat dan hasilnya tidak habis.

Cara mencari tahu bilangan irrasional atau tidak, kita dapat memakai pohon faktor. Dalam soal ini adalah [tex]\sqrt{208}[/tex]. Berikut adalah cara memecahkannya

208

/   \

2  104

     / \

    2  52

         / \

        2 26

             / \

            2  13

Hasil => [tex]2^{4} \times 13[/tex]

Jika kita akar kuadratkan bilangan [tex]2^{4}[/tex] maka hasilnya adalah sebagai berikut :

[tex]= \sqrt{2^{4} } \\= \sqrt{16} \\= 4[/tex]

Maka hasilnya adalah [tex]4 \sqrt{13}[/tex]

21. 735 + 208 = 208 + n.. bantu dong kk jawabannya


735 + 208 = 208 + n

Pindahkan 208 ke ruas kiri agar bisa dikurangkan dengan bilangan lainnya.

735 + 208 - 208 = n

735 + 0 = n

735 = n

n = 735

Maka Nilai n = 735

Atau bisa dituliskan :

735 + 208 = 208 + 735 (Sifat Komutatif)
