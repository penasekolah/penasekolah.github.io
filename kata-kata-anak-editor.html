12. Tentukan banyaknya permutasi dari huruf-huruf penyusun kata berikut: a. EDITOR b. DASAR c. MALAM d. ASIMILASI


kalo menurut aku kaya gini :)a)EDITOR
n = 6
P = n!
= 6!
= 6×5×4×3×2×1
= 720
b) DASAR
n = 5
k = 2 ( kembar huruf A )
P = n! / k!
= 5! / 2!
= 5×4×3×2! / 2!
= 60
c) MALAM
n= 5
k = 2 (kembar huruf M)
l = 2 (kembar huruf A)
P= n! / k! l!
= 5! / 2! 2!
= 5×4×3×2! / 2!×2×1
= 60/2
= 30
d) ASIMILASI
n= 9
k = 2 (kembar huruf A)
l = 2 (kembar huruf S)
m = 3 (kembar huruf I)
P = n! / k! l! m!
= 9! / 2! 2! 3!
= 9×8×7×6×5×4×3! / 3!×2×1×2×1
= 60480 / 4
= 15120
