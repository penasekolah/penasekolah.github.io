12. sebutkan cara membuat Jaringan Local?​


Jawaban:

Siapkan alat dan bahan, yakni :

RJ45 = Untuk di pasang di kabel UTP dan ke LAN card maupun onboard

UTP = Kabel yang akan di pasang RJ45, sekaligus berguna untuk menyambungkan antar komputer.

HUB/Switch = Untuk menggabungkan dua buah Personal Compuer atau bahkan bisa lebih dalam sebuah jaringan.

Komputer = Untuk test berhasil ataukah gagal.

Pasang Kabel :

Ambil kabel yang sudah ada RJ45 dan pengkabelan harus straight. Jika sudah pasang ujung satu ke Lan Card maupun onboard, yang satunya ke HUB/switch, begitu pula komputer lain. Pastikan pada step ke 2 ini jangan sampai ada kesalahan karena akan mengakibatkan membuat jaringan LAN menjadi gagal total.

Setting IP :

Buka Control Panel.

Klik Network and Internet

Tutorial

Klik Network and Sharing Center.

Klik Change Adapter Setting.

Klik Kanan Local Area Connection.

Klik Properties.

Klik Internet Protocol Version 4 (TCP/IPv4), klik properties.

Setting IP seperti di bawah ini :

Jika sudah setting IP lalu cek dengan membuka cmd, dengan menggunakan perintah start->run->cmd lalu akan muncul kotak dialog berwarna hitam dan kalian menuliskan ping (IP tujuan). Jadi contoh misalkan milik kalian 192.168.1.1 maka kalian harus menuliskan ping 192.168.1.2 dan sebaliknya.

Untuk Wireless LAN bisa dengan cara seperti ini :

Buka kontrol panel.

Buka Network and Internet, lalu buka network and sharing center.

Klik set up a new connection or network.

Klik set up a wireless ad hoc network, lalu klik next.

Klik next lagi, kemudian ketikan nama jaringan dan bila perlu password juga.

Kemudian tinggal setting IP seperti pada cara membuat jaringan wired LAN diatas.

Bila sudah setting IP maka cek koneksi dengan menggunakan cmd.

Hasil dari [tex]\tt{} 50^2[/tex] adalah [tex]\tt{}2500.[/tex]

Pendahuluan

Pengertian Pangkat Bulat Positif

Jika a adalah rill dan n bilangan bulat positif maka [tex]\tt{}a^n [/tex] ( dibaca a pangkat n ) adalah hasil kali n dua faktor yang masing-masing faktornya adalah a. Jadi pangkat kulit positif secara umum dinyatakan dalam bentuk.

[tex] \begin{array}{cc}\huge\boxed{\tt{} {a}^{n} = a \times a\times a\times .... \times a \times a } \end{array} [/tex]

Sifat-Sifat Pangkat:

A) [tex] \tt{} {a}^{p} \times {a}^{q} = {a}^{p + q} [/tex]

B) [tex] \tt{} {a}^{p} \div {a}^{q} = {a}^{p -q} [/tex]

C) [tex]\tt{}(a^p)^q = a^pq [/tex]

D) [tex]\tt{}(a \times b)^n = a^n \times b^n [/tex]

E) [tex] \tt{}( \frac{a}{b})^n = \frac{a^n}{b^n} [/tex]

F) [tex] \tt{} {a}^{ - n} = \frac{1}{ {a}^{n} } \: atau \: \frac{1}{ {a}^{ - n} } [/tex]

G) [tex]a^0 = 1 [/tex]

Pembahasan

[tex]\tt{} {50}^{2} [/tex]

[tex]\tt{} = 50 \times 50[/tex]

[tex]\tt{} = 2500[/tex]

Kesimpulan: Maka hasilnya adalah [tex]\tt{} = 2500.[/tex]
