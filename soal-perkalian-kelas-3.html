Penjelasan dengan langkah-langkah:

Persamaan Kuadrat

Bentuk umum persamaan kuadrat ax² + bx + c = 0

x² - (k + 1)x + (k + 3) = 0

a = 1 , b = -(k + 1) dan c = (k + 3)

pemisalan :

a¹ = 2a²

a¹ + a² = -b/a

2a² + a² = -(-(k + 1)/ 1

2a² + a² = (k + 1)/1

3a² = (k + 1)

a² = (k + 1)/3

maka,

a¹ . a² = c/a

2a² . a² = (k + 3)/1

2(k + 1)/3 . (k + 1)/3 = (k + 3)

(2k + 2)/3 . (k + 1)/3 = (k + 3)

2k(k + 1) + 2(k + 1)/ 9 = (k + 3)

2k² + 2k + 2k + 2 = 9(k + 3)

2k² + 4k + 2 = 9k + 27

2k² + 4k - 9k + 2 - 27 = 0

2k² - 5k - 25 = 0 ---> pemfaktoran persamaan kuadrat

(2k + 5)(k - 5) = 0

k¹ = 5/2 atau k² = 5

HP = {( -5/2 , 5 )}

Jadi, nilai k yang memenuhi adalah -5/2 atau 5
