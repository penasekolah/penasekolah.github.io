Pakai Appsgeyser

Cara membuat game Android sendiri yang pertama kamu bisa melalui website Appsgeyser. Langkah-langkahnya sangat mudah untuk kamu ikuti.

1. Kamu buka website appsgeyser.com, lalu klik Create Now.

3. Kemudian pilih Matching Puzzle.

5. Ganti semua elemen sesuai kebutuhan kamu.

6. Jika sudah, kamu bisa melihat preview terlebih dahaulu. Jika sudah, klik Next.

7. Isi semua langkah sesuai keperluanmu.

8. Kemudian klik Create. Sebelumnya kamu harus memiliki akun Apssgeyser terlebih dahulu agar kamu bisa mengambil file Apk game yang kamu buat tadi.

9. Selesai.

Cara Membuat Game Android, Mudah dan Bisa Bikin Sendiri

Oleh Loudia Mahartika pada 26 Feb 2019, 10:10 WIB

Ilustrasi Perbesar

Ilustrasi main game di ponsel

Liputan6.com, Jakarta Fenomena game online kini banyak digandrungi oleh anak-anak muda di indoensia. Demam game online melanda banyak kalangan, dari anak-anak hingga orang dewasa. Para gamer di Indonesia jumlahnya terus bertambah. Perkembangan itu tak terlepas dari peran kecanggihan dari smartphone saat ini.

Terus berkembangnya smartphone memudahkan para pengguna smartphone untuk bermain game. Tren yang sempat ramai dimainkan oleh masyarakat Indonesia adalah Pokemon Go, Clash of Clan, Clash Royale, Mobile Legend dan masih banyak lagi.

Minat anak muda untuk bermain game yang semakin tinggi, tak sedikit orang yang mencoba membuat game sendiri. Walaupun banyak jenis games yang dapat dicari dan didownload di Google Play Store secara gratis.

Ternyata hanya berbekal dengan smartphone Android, kamu juga bisa membuat game sendiri. Enggak perlu keahlian khusus, bahkan beberapa cara membuat game Android tidak membutuhkan coding. Jadi kamu bisa game kamu sendiri dengan bebas berkreasi.

Dlansir Liputan6.com, Selasa (26/2/20190 dari berbagai sumber, berikut cara membuat game Android sendiri yang bisa kamu coba.

2 dari 4 halaman

Pakai Appsgeyser

Cara membuat game Android sendiri yang pertama kamu bisa melalui website Appsgeyser. Langkah-langkahnya sangat mudah untuk kamu ikuti.

1. Kamu buka website appsgeyser.com, lalu klik Create Now.

2. Lalu pilih Other App. Ini adalah halaman untuk kamu membuat game. Karena dalam websiter tersebut tak hanya menyediakan game saja.

3. Kemudian pilih Matching Puzzle.

4. Ini adalah contoh membuat game tentang puzzle. Saat kamu mencoba membeuat game ini, kamu harus sediakan setidaknya 6 gambar yang berbeda untuk kamu jadikan model puzzle.

5. Ganti semua elemen sesuai kebutuhan kamu.

6. Jika sudah, kamu bisa melihat preview terlebih dahaulu. Jika sudah, klik Next.

7. Isi semua langkah sesuai keperluanmu.

8. Kemudian klik Create. Sebelumnya kamu harus memiliki akun Apssgeyser terlebih dahulu agar kamu bisa mengambil file Apk game yang kamu buat tadi.

9. Selesai.

•Pakai Quickappninja•

Selanjutnya kamu bisa membuat game sendiri melalui Quickappninja. Cara membuat game Android ini pertama kamu harus mendaftar pada situs tersebut. Dalam situs ini tersedia 5 jenis game yang bisa kamu pilih.

5 jenis game yang tersedia itu antara lain menebak gambar, menebak gambar dengan kata, menebak gambar dengan puzzle gambar, merangkai kata-kata dan yang terakhir chat history. Cara membuat game Android ini akan mengambil contoh membuat games menebak gambar dengan kata.

1. Klik pilihan games. Disini akan memilih Guess the Picture atau menebak gambar dengan kata.

2. A kan muncul tampilan mengedit content dan warna game. Beberap ayang perlu diedit adalah icon game, tema game, isi quiz dalam game, suara game, menu game dan lain-lain. Kamu hanya tinggal pilih sesuai dengan keinginanmu.

3. Kemudian kamu akan diminta untuk mengatur source game. Kamu tinggal mengisi nama game, email, ketentuan dan amount of letters.

4. Setelah semuanya selesai, klik Generate Game yang berada di pojok kanan. kamu tinggal menunggu sebelum game bisa didownload dan diupload ke Google Play Store.
