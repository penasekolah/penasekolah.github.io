Name tag atau tanda nama merupakan bukti identitas seorang peserta ketika mengikuti suatu kegiatan. Ada beragam bentuk tanda nama yang bisa kita temukan dewasa ini. Ada tanda nama yang menggunakan teknologi chip, RFID, strip magnetik, bahan plastik, hingga yang paling sederhana seperti menggunakan bahan kertas karton. Tergantung dari bahannya, proses pembuatan masing-masing tanda nama berbeda-beda. Semakin canggih teknologi yang terlibat, semakin rumit proses pembuatannya.


Pada kesempatan ini, soal meminta kita untuk menjelaskan cara membuat name tag atau tanda nama dari kertas karton berwarna biru. Berikut cara-cara yang dapat kamu tempuh


Bahan:

Kertas karton berwarna biru

Gunting

Spidol atau alat tulis lain dengan tinta cukup tebal

Tempat name tag (bisa berupa plastik atau penjepit)

Pensil

Penggaris


Cara:

1. Tentukan ukuran name tag yang akan kamu buat. Jika name tag akan diletakkan di wadah tertentu seperti plastik name tag, pastikan ukurannya sesuai dengan tempat tersebut.

2. Gunakan penggaris dan pensil untuk membentuk name tag yang akan kamu buat, berdasarkan ukuran yang kamu peroleh dari langkah 1.

3. Gunakan gunting untuk memotong name tag.

4. Tulis nama kamu dan informasi lain dibutuhkan pada name tag sesuai kebutuhan kegiatan

5. Masukkan name tag ke tempatnya

15. Dengan menggunakan tag ini kita juga bisa membuat list tanpa simbol atau angka didepan keterangan list adalah a. Tag UL b. Tag OL c. Tag DL d.Tag Do​


Cara 1:
Dengan menggunakan tag <DL>, kita bisa membuat list tanpa simbol atau angka didepan keterangan list. Namun perlu diperhatikan bahwa tag <DL> digunakan untuk membuat description list.Cara 2:
Dengan menggunakan tag <UL> atau tag <OL> yang diberi styling “list-style: none;”, kita juga bisa membuat list tanpa simbol atau angka didepan keterangan list.

 

Pembahasan

Untuk membuat list pada dokumen HTML, kita dapat menggunakan tag <UL>, <OL>, atau <DL>.

UL adalah singkatan dari unordered list, artinya daftar yang tidak diurutkan berdasarkan angka. OL memiliki kepanjangan ordered list, artinya daftar yang terurut berdasarkan angka.DL adalah singkatan dari definition/description list, artinya daftar yang berisi definisi/deskripsi.

Berbeda dengan tag <UL> dan <OL> yang membutuhkan tag <LI> untuk setiap item dalam list, tag <DL> membutuhkan tag <DT> (term) dan tag <DD> (description).

Jadi, cara pertama untuk membuat list tanpa simbol atau angka didepan keterangan list adalah dengan tag <DL>.

Kita juga bisa membuat list tanpa simbol atau angka dengan tag <UL> dan <OL>, dengan menambahkan kode style “list-style: none;” pada kedua tag tersebut.

Contoh perbedaan dan tampilan kode-kode tag tersebut dapat dilihat pada gambar.

Jawaban:

A. <tag>objek text</tag>

Penjelasan:

Dalam Penulisan tag HTML, yang menggunakan pembuka dan penutup tag, maka format atau cara penulisan yang benar adalah seperti berikut:

<tag>objek text</tag>

Untuk pembuka tag kita akan menulis <tag>

Akan tetapi untuk menutup tag tersebut kita akan menulis </tag>. Perbedaan antara pembuka dan penutup adalah pada tanda slash atau "/"
