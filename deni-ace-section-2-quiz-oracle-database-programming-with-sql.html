8. penulisan statement sql yang tepat untuk memasukkan data baru ke dalam database


Jawaban:

INSERT INTO 'nama_tabel' VALUES('item1','item2','item3');

Penjelasan:

Jumlah isi dari values sendiri sesuai dengan kolom dari tabel yang dituju.

Terdapat cara lain untuk menambahkan record ke database :

INSERT INTO 'nama_tabel' ('kolom1','kolom2','kolom3') VALUES('item1','item2','item3');

Cara ini berguna, jika hanya ingin mengisi kolom tertentu saja di tabel yang dituju. (kolom yang kosong, otomatis bernilai null)

Disini juga bisa menambahkan beberapa data sekaligus, dengan menambahkan VALUES('item1','item2','item3') baru, setelah tanda koma.


9. Tugas :Cari pengertiana. Microsoft SQL Serverb. MySQLC. Oracle​


a. Microsoft SQL Server

Microsoft SQL Server adalah sebuah sistem manajemen basis data relasional (RDBMS) produk Microsoft. Bahasa kueri utamanya adalah Transact-SQL yang merupakan implementasi dari SQL standar ANSI/ISO yang digunakan oleh Microsoft dan Sybase.

b. MySQL

MySQL adalah salah satu jenis database yang bersifat open source. Website dan aplikasi berbasis mobile memerlukan database server untuk menampung informasi yang banyak.

c. Oracle

Basis data Oracle adalah basis data relasional yang terdiri dari kumpulan data dalam suatu sistem manajemen basis data RDBMS. Perusahaan perangkat lunak Oracle memasarkan jenis basis data ini untuk bermacam-macam aplikasi yang bisa berjalan pada banyak jenis dan merk perangkat keras komputer (platform).

22. Yang bukan database sistem dari SQL Server adalah ... .A. master B. modelC. tempdbD. Northwind​


Jawaban:

D. Northwind

Penjelasan:

SQL Server adalah sebuah sistem manajemen basis data yang dikembangkan oleh Microsoft. SQL Server menyediakan beberapa database sistem yang digunakan untuk mengelola dan mengatur kegiatan basis data, di antaranya adalah master, model, tempdb, dan msdb.

Dari pilihan jawaban yang diberikan, hanya jawaban D. Northwind yang bukan merupakan database sistem dari SQL Server. Northwind adalah sebuah database contoh yang dibuat oleh Microsoft untuk menunjukkan bagaimana sebuah database dapat dibuat dan digunakan dengan SQL Server. Northwind biasanya digunakan sebagai bahan latihan atau contoh dalam tutorial atau dokumentasi SQL Server.

Jadi, jawabannya adalah D. Northwind.
