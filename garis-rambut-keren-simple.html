15. Simple present tense Ayah saya memotong rambut adik saya di teras


Jawaban:

Rumus simple present tense =

(+) Subject + V1(+e or es) + Object

(-) Subject + do/does + not + V1 + object

(?) Do/Does + Subject + V1 + Object

Penjelasan:

Ayah saya memotong rambut adik saya di teras.

Ayah saya = My Father (Subject)

Memotong = Cut (V1)

Rambut adik saya = My brother's hair (Object)

Di teras = In the terrace (Compliment)

Jadi kalimat nya adalah

My Father Cut(s) my brother's hair in the terrace.
