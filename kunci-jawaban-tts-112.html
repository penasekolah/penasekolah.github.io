Soal nomor 3Diketahui : rusuk = 10 cmLuas permukaan 1.570 cm² π = 3,14

Ditanya :tinggi tabung?

Jawab :

1.570 = 2 x 3,14 x 10 (10 + t)

1.570 = 62,8 (10 + t)

1.570/ 62,8 = 10 + t

25 = 10 + t

t = 25 - 10

t = 15 cm

Jadi, tinggi tabung adalah 15cm

Soal nomor 4 Diketahui :diameter 8 cmtinggi 9cm π = 3,14Ditanya :Luas permukaan kain ? Jawab :

= (3,14 x 4 x 4) + (2 x 3,14 x 4 x 9 )

= 50,24 + 2.260,8

= 2.311,04 cm²

Jadi, luas permukaan kain adalah 2.311,04cm²

Soal nomor 5Diketahui :diameter dalam 7 cmdiameter total 11 cmtinggi 3 cmπ = 3,14rusuk dalam (A) 3,5 cmrusuk luar (B) 5,5 cm

πrB² - πrA²

= (3,14 x 5,5 x 5,5 ) - (3,14 x 3,5 x 3,5 )

= (3,14 x 30,25) - (3,14 x 0,5 x 3,5)

Pendahuluan

Rumus-rumus skala, jarak peta, dan jarak sesungguhnya.

[tex]\boxed{Jp = S ~ x ~ Js}}[/tex]

[tex]\boxed{S = \frac{Jp}{Js}}}[/tex]

[tex]\boxed{Js = \frac{Jp}{S}}}[/tex]

Dimana:

Jp = jarak pada peta (cm)

S = skala pada peta (1 mewakili x cm)

Js = jarak sebenarnya/sesungguhnya (km/m)

Pembahasan

1. Diketahui

S = 1 mewakili 20 m = 1 : (20 x 100) = 1 : 2.000

Jp = 6 cm x 4,25 cm

Ditanya

Panjang dan lebar lapangan sepak bola (Js)?

Dijawab

Panjang sesungguhnya

Js = [tex]\frac{Jp}{S}[/tex]

= [tex]\frac{6}{\frac{1}{2.000} }[/tex]

= 6 x 2.000

= 12.000 cm

= 120 m

Lebar sesungguhnya

Js = [tex]\frac{Jp}{S}[/tex]

= [tex]\frac{4,25}{\frac{1}{2.000} }[/tex]

= 4,25 x 2.000

= 8.500 cm

= 85 m

Jadi, panjang dan lebar sesungguhnya sepak bola tersebut adalah 120 m x 85 m.

2. (a) Diketahui

S = 1 mewakili 15 km = 1 : (15 x [tex]10^{5}[/tex])

Jp = 23,4 cm

Ditanya

Jarak sesungguhnya (km)?

Dijawab

Js = [tex]\frac{Jp}{S}[/tex]

= [tex]\frac{23,4}{\frac{1}{15 x 10^{5} } }[/tex]

= 23,4 x 15 x [tex]10^{5}[/tex]

= 35.100.000 cm

= 351 km

(b) Diketahui

Js = 231 km

Ditanya

Jp?

Dijawab

[tex]Jp = S ~ x ~ Js[/tex]

= [tex]\frac{1}{15 x 10^{5} }[/tex] x 231

= 231 : 1.500.000

= 0,000154 km

= 154/1.000.000 x 100.000

= 15,4 cm

(c) Diketahui

Pembahasan

Aksara Jawa, juga dikenal sebagai Hanakalaka, Karakan, atau Dentauyanjana, adalah salah satu aksara tradisional Indonesia yang berkembang di pulau Jawa. Aksara tersebut terutama digunakan untuk menulis bahasa Jawa, namun dalam perkembangannya, beberapa bahasa daerah lainnya seperti bahasa Sunda, Madura, Sasak, Melayu, dan  bahasa sejarah seperti Sansekerta dan Kawi. Aksara Jawa berasal dari aksara Brahmi India. melalui  aksara Kawi dan berkaitan erat dengan aksara Bali. Aksara Jawa aktif digunakan dalam sastra Jawa dan tulisan sehari-hari  sejak pertengahan abad ke-15 hingga pertengahan abad ke-20, setelah itu fungsinya secara bertahap digantikan oleh aksara Latin. Aksara ini masih diajarkan sebagai bagian dari muatan lokal di sebagian kecil DI Yogyakarta, Jawa Tengah, Jawa Timur dan  Jawa Barat, namun penerapannya dalam kehidupan sehari-hari masih terbatas.
