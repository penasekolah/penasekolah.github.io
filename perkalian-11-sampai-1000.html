V = p × l × t

L = 2((p × l) + (p × t) + (l × t))

bd1 = db1 × t

Keterangan:

p = panjang balok

Sebuah satelit mengorbit bumi pd ketinggian 1000 km dari permukaan bumi.diketahui massa bumi 5,98×10²⁴ kg, jari jari bumi 6,38×10⁶ m dan konstanta gravitasi universal 6,67×10⁻¹¹ Nm²/kg² . Maka :

laju satelit agar bergerak dalam lintasan lingkaran adalah 7,3 x 10⁴ m/s.satelit mengorbit bumi  rata - rat 14 kali/hari

[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:7.\:HUKUM\:NEWTON\:(GAYA\:GRAVITASI)}}[/tex]

Gaya Gravitasi  merupakan gaya tarik menarik antara dua benda bermassa.

Persamaan Gaya Gravitasi adalah :

[tex]\boxed{F=G\frac{mM}{R^{2}}}[/tex]

Percepatan gravitasi merupakan percepatan suatu benda akibat ditarik benda bermassa lainnya.

Persamaan percepatan gravitasi adalah :

[tex]\boxed{g=G\frac{M}{R^{2}}}[/tex]

Periode orbit suatu satelit terhadap planet :

[tex]\boxed{T=2\pi \sqrt{\frac{r^{3} }{GM}}}[/tex]

Kecepatan Orbit satelit :

[tex]\boxed{v=\sqrt{\frac{GM}{r} }}[/tex]

Keterangan :

F : Gaya gravitasi (N)

g : Percepatan gravitasi benda (m/s²)

m : Massa benda (kg)

M : massa benda (kg)

R : Jari-jari benda (m)

r : Jari-jari orbit (m)

T : Periode orbit (s)

G : konstanta universal gravitasi (6,674 x 10⁻¹¹ Nm²/kg²)

v : kecepatan orbit

[tex]\boxed{Diketahui}[/tex]

Ketinggan satelit dari permukaan bumi (h) = 1.000 km

Massa bumi (M) = 5,98 x 10²⁴ kg

Jari - jari bumi (R) = 6,38 x 10⁶ m

Tetapan gravitasi universal (G) = 6,67 x 10⁻¹¹ Nm²/kg²

[tex]\boxed{Ditanya}[/tex]

Laju satelit dan berapa kali satelit mengorbit bumi dalam sehari.

[tex]\boxed{Penyelesaian}[/tex]

Besar laju satelit tersebut bisa dicari dengan menggunakan persamaan :

[tex]v=\sqrt{\frac{GM}{r} }[/tex]

dimana r = R + h

r = 6,38 x 10⁶ m + 1.000 km

r = 6,38 x 10⁶ m + 1 x 10⁶ m

r = 7,38 x 10⁶ m

Sehingga :

[tex]v=\sqrt{\frac{GM}{r} }[/tex]

[tex]v=\sqrt{\frac{6,67x10^{-11}Nm^{2}/kg^{2}\:5,98x10^{24}kg }{7,38x10^{6}m } }[/tex]

[tex]v=\sqrt{\frac{39,8866x10^{13}Nm^{2}/kg}{7,38x10^{6}m } }[/tex]

[tex]v=\sqrt{5,4x10^{7} }\:m/s[/tex]

[tex]v=\sqrt{54x10^{8} }\:m/s[/tex]

v = 7,3 x 10⁴ m/s

Jadi laju satelit agar bergerak dalam lintasa adalah 7,3 x 10⁴ m/s.

Untuk mengetahui Banyaknya satelit mengorbit bumi maka terlebih dahulu mencari periode satelit dengan menggunakan persamaan:

[tex]T=2\pi \sqrt{\frac{r^{3} }{GM}}[/tex]

[tex]T=2(3,14)\sqrt{\frac{(7,38x10^{6}m) ^{3} }{6,67x10^{-11}Nm^{2}/kg^{2}\:5,98x10^{24}kg}}[/tex]

[tex]T=6,28\sqrt{\frac{401,95x10^{18} }{39,9x10^{13} } }\:s[/tex]

[tex]T=6,28\sqrt{10x10^{5} } \:s[/tex]

T = 6,28 x 10³ s

T = 6280 s

Dalam sehari ada 86400 detik.

n = t / T

n = 86400 detik/6280 detik

n = 13,75 kali

Jadi satelit tersebut mengorbit rat-rata 14 kali/hari.
