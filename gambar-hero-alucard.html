Materi : SPLDV

Pemisalan : Alucard = x , Tigreal = y

Persamaan 3 thn lalu

Konsep = ( x - 3 ) dan ( y - 3 )

→ ( x - 3 ) = 4( y - 3 )

Persamaan sekarang

Konsep = ( x ) dan ( y )

→ [ y = 11 thn ]

\_______________________/

Substitusi ( x → y )

x - 3 = 4( 11 - 3 )

x - 3 = 4( 8 )

x = 32 + 3

[ x = 35 thn ]

( x + 5 ) = 40 thn
