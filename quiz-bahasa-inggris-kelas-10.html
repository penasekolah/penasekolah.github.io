3. Quiz- by Hana Diketahui: A = { x | 4 ≤ x ≤ 8, x ⋲ bilangan asli }. B = { x | 6 ≤ x ≤ 10, x ⋲ bilangan cacah }. Maka tentukanlah anggota dari A ∪ B ? Di ketahui : P = { x | 5 < x < 25, x ⋲ bilangan prima }. Q = { x | 4 < x < 14, x ⋲ bilangan ganjil }. Maka tentukanlah anggota dari A ∩ B ? Suatu kelas terdiri dari 40 orang siswa, dan diantaranya ada 15 orang siswa yang menyukai pelajaran matematika, lalu ada 13 orang siswa yang menyukai pelajaran bahasa inggris dan yang 7 orang siswa yang menyukai keduanya. Berapa banyak siswa yang tidak menyukai pelajaran matematika maupun bahasa inggris ?


[tex]\huge{\purple{\mathfrak{Jawaban : }}}[/tex]_______________

Diketahui:

A = { x | 4 ≤ x ≤ 8, x ⋲ bilangan asli }.

B = { x | 6 ≤ x ≤ 10, x ⋲ bilangan cacah }.

Maka tentukanlah anggota dari A ∪ B ?

[tex]\pink{Pembahasan }[/tex]

A = { 4, 5, 6, 7, 8}

B = {6, 7, 8, 9, 10}

A ∪ B merupakan himpunan yang anggotanya adalah gabungan semua anggota A dan semua anggota B, maka:

A ∪ B = { 4, 5, 6, 7, 8, 9, 10}

Jadi, anggota dari himpunan A ∪ B adalah { 4, 5, 6, 7, 8, 9, 10}.

-------------------------------

Di ketahui :

P = { x | 5 < x < 25, x ⋲ bilangan prima }.

Q = { x | 4 < x < 14, x ⋲ bilangan ganjil }.

Maka tentukanlah anggota dari A ∩ B ?

[tex]\pink{Pembahasan }[/tex]

P = {7, 11, 13, 17, 19, 23}

Q = {5, 7, 9, 11, 13}

A ∩ B merupakan himpunan yang anggotanya merupakan anggota P sekaligus merupakan anggota Q, maka:.

A ∩ B = {7, 11, 13}

Jadi, anggota dari himpunan A ∩ B adalah { 3, 5, 7 }.

_________________

Suatu kelas terdiri dari 40 orang siswa, dan diantaranya ada 15 orang siswa yang menyukai pelajaran matematika, lalu ada 13 orang siswa yang menyukai pelajaran bahasa inggris dan yang 7 orang siswa yang menyukai keduanya.

Berapa banyak siswa yang tidak menyukai pelajaran matematika maupun bahasa inggris ?

[tex]\pink{Pembahasan }[/tex]

Misal

x = banyak siswa yang tidak menyukai kedua pelajaran.

Maka:

Banyak siswa yang hanya menyukai matematika adalah 15 – 7 = 8 orang siswa.

Banyak siswa yang hanya menyukai bahasa inggris adalah 13 – 7 = 6 orang siswa.

Himpunan tersebut bisa digambarkan dengan diagram venn - Gambar-

Banyak anak yang tidak menyukai kedua pelajaran ialah :

40 = 8 + 7 + 6 + x

40 = 21 + x

x = 40 – 21

x = 19

Jadi, banyak siswa yang tidak menyukai pelajaran matematika maupun bahasa inggris adalah 19 orang.

        Himpunan adalah kumpulan objek-objek yang dapat didefinisikan dengan jelas dan terukur sehingga dapat diketahui termasuk atau tidaknya di dalam himpunan tertentu.
