7. tuliskan reaksi elektrolisis lelehan Al2O3 dengan elektrode grafit !


Kelas           : XII
Pelajaran     : Kimia
Kategori       : Redoks & Elektrokimia
Kata Kunci   : sel elektrolisis, lelehan, aluminium oksida

Pembahasan

Pelajari aturan mendasar sel elektrolisis disini https://brainly.co.id/tugas/4028489, terkait apa yang akan terjadi di katoda dan anoda dilihat dari wujud senyawa elektrolit serta jenis kation dan anion.

Diminta untuk menuliskan reaksi elektrolisis lelehan Al₂O₃ dengan elektroda grafit.

Karena wujudnya lelehan (atau leburan), kation akan langsung mengalami reduksi di katoda dan begitu pula anion akan langsung mengalami oksidasi di anoda. Perhatikan, oksigen tidak termasuk sisa asam oksi sehingga ion oksigen akan langsung teroksidasi, bukan pelarut air. Digunakan elektroda jenis inert yang tak berpengaruh di anoda.

Reaksi elektrolisis

Lelehan Al₂O₃ (aluminium oksida)
Al₂O₃(l) → 2Al³⁺(aq) + 3O²⁻(aq)

Katoda : Al³⁺(aq) + 3e → Al(s)......(kalikan 4)
Anoda :  2O²⁻(aq) → O₂(g) + 4e....(kalikan 3)
-----------------------------------------
Redoks ⇒ 4Al³⁺(aq) + 6O²⁻(aq) → 4Al(s) + 3O₂(g)

Kesimpulan:
Hasil di katoda adalah endapan aluminium
Hasil di anoda adalah gas oksigen

10. Tulislah reaksi elektrolisis Larutan Na2SO4(elektrode grafit)


Pembahasan

Tulislah reaksi elektrolisis larutan Na₂SO₄ (elektroda grafit).

Ini merupakan persoalan elektrokimia, khususnya mengenai elektrolisis.

Aturan elektrolisis yang digunakan pada soal kali ini adalah sebagai berikut:

Reaksi pada Katoda

Kation akan tereduksi, KECUALI kation yang berasal dari logam aktif golongan IA, IIA, Al, dan Mn saat berwujud larutan maka pelarut air yang akan tereduksi dan dihasilkan gas hidrogen di katoda.

[tex]\boxed{2H+2O_{(l)} + 2e \rightarrow H_2_{(g)} + 2OH^-_{(aq)}}[/tex]

Reaksi pada Anoda

Pada penggunaan elektroda (anoda) jenis inert (karbon C dan platina Pt), semua jenis anion teroksidasi KECUALI sisa asam oksi seperti [tex]SO_4^{2-}, NO_3^-, CO_3^{2-}[/tex] tidak mengalami oksidasi, melainkan pelarut air yang teroksidasi dan dihasilkan gas oksigen di anoda.

[tex]\boxed{2H_2O_{(l)} \rightarrow O_2_{(g)} + 4H^+_{(aq)} + 4e}[/tex]

Dilakukan elektrolisis larutan Na₂SO₄ dengan menggunakan elektroda grafit (atau karbon, C) yang termasuk elektroda inert.

Kation natrium termasuk logam golongan IA. Anion sulfat termasuk sisa asam oksi

Sesuai penjelasan di atas, berikut hasil reaksi elektrolisis larutan Na₂SO₄ menggunakan elektroda grafit. Diawali oleh reaksi ionisasi dari natrium sulfat.

[tex]\boxed{Na_2SO_4_{(aq)} \rightarrow 2Na^+_{(aq)} + SO_4^{2-}}[/tex]

[tex]\boxed{Katoda:~2H+2O_{(l)} + 2e \rightarrow H_2_{(g)} + 2OH^-_{(aq)}}[/tex]

[tex]\boxed{Anoda:~2H_2O_{(l)} \rightarrow O_2_{(g)} + 4H^+_{(aq)} + 4e}[/tex]

Hasil di katoda adalah gas hidrogen

Hasil di anoda adalah gas oksigen

27. Tuliskan reaksi elektrolisis larutan Na2SO4 dengan elektrode grafit​


Elektrolisis larutan [tex] Na_2SO_4 [/tex] dengan elektrode grafit

Katoda: [tex] 2H_2O \ + \ 2e^- \ \rightarrow \ 2OH^- \ + \ H_2 [/tex] | x2

Anoda: [tex] 2H_2O \ \rightarrow \ 4H^+ \ + \ O_2 \ + \ 4e^- [/tex] | x1

________________________________

Total: [tex] 6H_2O \ \rightarrow \ 4OH^- \ + \ 2H_2 \ + \ 4H^+ \ + \ O_2 [/tex]

»PEMBAHASAN«

[tex] Na_2SO_4 \ \rightarrow \ 2Na^{+} \ + \ SO_4^{2-} [/tex]

Elektrolisis Larutan menghasilkan kation [tex] Na^+ [/tex] yang merupakan logam golongan IA, maka reaksi di katoda adalah reduksi air, sedangkan reaksi lainnya menghasilkan anion [tex] SO_4^{2-} [/tex] dan menggunakan elektrode inert C (grafit), jadi karena anionnya merupakan anion sisa asam oksi (memiliki oksigen didalamnya) maka reaksi di anoda adalah oksidasi air.

Jadi untuk reaksi di katoda dan anoda dan reaksi keseluruhannya adalah seperti berikut:

Katoda: [tex] 2H_2O \ + \ 2e^- \ \rightarrow \ 2OH^- \ + \ H_2 [/tex] | x2

Anoda: [tex] 2H_2O \ \rightarrow \ 4H^+ \ + \ O_2 \ + \ 4e^- [/tex] | x1

___________________________________

Total: [tex] 6H_2O \ \rightarrow \ 4OH^- \ + \ 2H_2 \ + \ 4H^+ \ + \ O_2 [/tex]

30. Tuliskan contoh persamaan linear beserta tabel dan grafitnya


Tentukan daerah  penyelesaian dari sistem pertidaksamaan linear berikut!
3x + 5y  15
x  0
y  0Penyelesaian:
Gambar garis 3x + 5y =15, x = 0, dan y =0
Untuk 3x + 5y  15
Pilih titik (0,0), kemudian substitusikan ke pertidaksamaan sehingga diperoleh:
3 × 0 + 5× 0  15
0  15 (benar), artinya dipenuhi
Sehingga daerah penyelesaiannya adalah daerah yang memuat titik (0,0)
Untuk x  0, pilih titik (1,1) kemudian disubstitusikan ke pertidaksamaan sehingga diperoleh:
1  0 (benar), artinya dipenuhi.
Sehingga daerah penyelesaiannya adalah daerah yang memuat titik (1,1)
Untuk y  0, pilih titik (1,1) kemudian substitusikan ke pertidaksamaan sehingga diperoleh:
1  0 (benar), artinya dipenuhi.
Sehingga himpunan penyelesaiannya adalah daerah yang memuat titik (1,1).Selanjutnya arsir daerah yang memenuhi persamaan, seperti gambar dibawah ini.Daerah  penyelesaian sistem pertidaksamaan merupakan irisan dari ketiga himpunan penyelesaian pertidaksamaan di atas, yaitu seperti terlihat pada gambar berikut ini (daerah yang diarsir).Pertidaksamaan Linear juga dapat digunakan untuk memecahkan masalah dalam kehidupan sehari-hari. Hal ini dapat dilakukan dengan memodelkan masalah menjadi model matematika. Jadi, Model matematikamerupakan suatu cara sederhana untuk menerjemahkan suatu masalah ke dalam bahasa matematika dengan menggunakan persamaan, pertidaksamaan, atau fungsi.Perhatikan contoh berikut :Pak Adi merupakan seorang pedagang roti. Beliau menjual roti menggunakan gerobak yang dapat memuat 600 bungkus roti. Roti yang dijualnya yaitu roti manis dan roti tawar dengan harga masing-masing  Rp 5.500,00 untuk roti manis dan Rp 4.500,00 untuk roti tawar per bungkusnya. Dari penjualan roti tersebut, beliau memperoleh keuntungan Rp 500,00 dari sebungkus roti manis dan Rp 600,00 dari sebungkus roti tawar. Apabila modal yang dimiliki oleh Pak Budi adalah Rp 600.000, buatlah model matematika agar beliau dapat memperoleh keuntungan sebesar-besarnya!Penyelesaian :Permasalahan Pak Adi diatas  dapat dimodelkan dalam bentuk matematika dengan menggunakan sistem pertidaksamaan linear dua variabel. Dengan memisalkan banyaknya roti manis sebgai x dan roti tawar sebagai y sehingga diperoleh tabel sebagai berikut.Berdasarkan tabel diatas jika kita tuliskan dalam bentuk pertidaksamaan linear menjadix + y ≤ 600,
5.500x + 4.500y ≤ 600.000,
