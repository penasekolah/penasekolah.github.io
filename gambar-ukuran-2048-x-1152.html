5. Suatu deret ukur X mempunyai nilai a=2048 dan p=2. Sedangkan deret ukur Y memiliki nilai a=4 dan p=16. Para suku keberapakah kedua deret ukur tersebut mempunyai nilai yang sama dan berapa nilainya?


Cara:

Deret ukur X :

Suku pertama = S1 = a = 512

Karena p = 0,5, maka S2 = S1 x p = 512 x 0,5 = 256

S3 = S2 x p = 256 x 0,5 = 128

S4 = S3 x p = 128 x 0,5 = 64

Deret ukur Y :

Karena p = 4, maka S4 = S3 x p = 16 x 4 = 64

Jadi, nilai suku dari kedua deret tersebut sama adalah pada suku ke4

9. Berapa (21^2048 modulo 30) x (49^105 modulo 50)


[tex]\begin{aligned}&\left(21^{2048}\ {\rm mod\ }30\right)\times\left(49^{105}\ {\rm mod\ }50\right)\\&=\ \boxed{\,\large\text{$\bf1029$}\,}\end{aligned}[/tex]
 

Penjelasan dengan langkah-langkah:

[tex]\left(21^{2048}\ {\rm mod\ }30\right)\times\left(49^{105}\ {\rm mod\ }50\right)[/tex] akan diselesaikan dengan notasi kongruensi modular, di mana:
a mod b = c ekuivalen dengan a ≡ c (mod b).

Untuk [tex]21^{2048}\ {\rm mod\ }30[/tex]:

[tex]\begin{aligned}&21^{2048}\\&\equiv(30-9)^{2048}&({\rm mod\ }30)\\&\equiv(-9)^{2048}&({\rm mod\ }30)\\&\equiv81^{1024}&({\rm mod\ }30)\\&\equiv(-9)^{1024}&({\rm mod\ }30)\\&\equiv81^{512}&({\rm mod\ }30)\\&\equiv(-9)^{256}&({\rm mod\ }30)\\&\equiv81^{128}&({\rm mod\ }30)\\&\qquad\vdots\\&\equiv(-9)^{2}&({\rm mod\ }30)\\&\equiv81&({\rm mod\ }30)\\&\equiv21&({\rm mod\ }30)\\\end{aligned}[/tex]

[tex]\begin{aligned}&21^{2048}\equiv21\ ({\rm mod\ }30)\\&\implies 21^{2048}\ {\rm mod\ }30=\bf21\end{aligned}[/tex]

Untuk [tex]49^{105}\ {\rm mod\ }50[/tex]:

[tex]\begin{aligned}&49^{105}\\&\equiv(50-1)^{105}&({\rm mod\ }50)\\&\equiv(-1)^{105}&({\rm mod\ }50)\\&\equiv(-1)&({\rm mod\ }50)\\&\equiv49&({\rm mod\ }50)\\\end{aligned}[/tex]

[tex]\begin{aligned}&49^{105}\equiv49\ ({\rm mod\ }50)\\&\implies 49^{105}\ {\rm mod\ }50=\bf49\end{aligned}[/tex]

Oleh karena itu:

[tex]\begin{aligned}&\left(21^{2048}\ {\rm mod\ }30\right)\times\left(49^{105}\ {\rm mod\ }50\right)\\&=21\times49=(20+1)\times49\\&=980+49\\&=\boxed{\,\bf1029\,}\quad\blacksquare\end{aligned}[/tex]

27. Barisan angka terdiri dari: 3, 6, 24, 144, 1152, x. berpakah nilai x pada barisan angka tersebut?


Terdapat barisan angka yang terdiri dari 3, 6, 24, 144, 1152, x. Pola bilangan pada barisan yaitu dikali 2, dikali 4, dikali 6 dan seterusnya. Nilai x yang memenuhi pada barisan angka tersebut yaitu 11520.

Penjelasan dengan Langkah-Langkah

Diketahui terdapat barisan angka. Ditanya untuk nilai x yang memenuhi pada barisan angka tersebut. Kita perhatikan bahwa barisan angka tersebut yaitu barisan geometri. Barisan geometri yaitu barisan dimana setiap sukunya yang berurutan memiliki perbandingan yang sama. Sehingga pada soal ini dapat kita selesaikan dnegan perhitungan berikut.

Diketahui :

Barisan angka 3, 6, 24, 144, 1152, x

a = 3

Ditanya : nilai x

Jawab :

Sebelum mencari nilai x maka kita tentukan dahulu rasio pada setiap sukunya

r1 = [tex]\frac{U(1+1)}{u1}[/tex]

r1 = [tex]\frac{6}{3}[/tex]

r1 = 2

r2 = [tex]\frac{U(2+1)}{U2}[/tex]

r2 = [tex]\frac{24}{6}[/tex]

r2 = 4

r3 = [tex]\frac{U(3+1)}{U3}[/tex]

r3 = [tex]\frac{144}{24}[/tex]

r3 = 6

Setelah mengetahui r1, r2 dan r3, dapat kita lihat polanya bahwa setiap suku selanjutnya yaitu rasio sebelumnya ditambah 2. Maka untuk memenuhi nilai x, kita perlu mengetahui r5 dulu.

r5 = r4 + 2

r5 = 8+2

r5 = 10

Sehingga nilai x yang memenuhi yaitu

U6 = U5 x 10

U6 = 1152 x 10

U6 = 11520

28. Tentukan nilai x jika x^x=2^2048


Nilai x yang memenuhi [tex]x^x=2^{2048}[/tex] adalah 256.

PEMBAHASAN

Pangkat atau eksponen adalah perkalian berulang dari suatu bilangan yang sama. Bentuk berarti kita mengalikan bilangan a dengan bilangan a sebanyak n kali atau

[tex]\underbrace{a^n=a\times a\times a\times...\times a}_{sebanyak~n~kali}[/tex]

dengan :

a = bilangan pokok/basis.

n = bilangan pangkat.

Operasi pada bilangan pangkat adalah sebagai berikut.

[tex](i)~a^b\times a^c=a^{b+c}[/tex]

[tex](ii)~\frac{a^b}{a^c}=a^{b-c}[/tex]

[tex](iii)~(a^b)^c=a^{b\times c}[/tex]

[tex](iv)~a^{-b}=\frac{1}{a^b}[/tex]

[tex](v)~\sqrt[n]{a}=a^{\frac{1}{n}}[/tex]

.

DIKETAHUI

[tex]x^x=2^{2048}[/tex]

.

DITANYA

Tentukan nilai x.

.

PENYELESAIAN

[tex]x^x=2^{2048}[/tex]

[tex]x^x=2^{8\times256}[/tex]

[tex]x^x=(2^8)^{256}[/tex]

[tex]x^x=(256)^{256}[/tex]

[tex]x=256[/tex]

.

KESIMPULAN

Nilai x yang memenuhi [tex]x^x=2^{2048}[/tex] adalah 256.

.
