2. hasil dari integral (x+1)(x2+2x-3)4 dx=


Jawaban:

Rumus integral -

[tex] \tt \int ax {}^{m} dx = \frac{a}{m + 1} x {}^{m + 1} + C \\ [/tex]

Rumus integral (cepat untuk perkalian pangkat tinggi)

[tex] \tt \int f(x).h(x)^{n} dx = \frac{f(x) . h(x)^{n + 1}}{(n + 1) . h'(x)} + C \\ [/tex]

[tex] \\ [/tex]

Ada dua cara dalam penyelesaian ini

Pertama, saya pakai rumus yang diatas

[tex] \sf \int(x + 1)(x {}^{2} + 2x - 3) {}^{4} dx[/tex]

[tex]= \sf \frac{(x + 1)(x {}^{2} + 2x - 3) {}^{4 + 1} }{(4 + 1).(2x + 2)} + C[/tex]

[tex]= \sf \frac{(x + 1)(x {}^{2} + 2x - 3) { }^{5} }{5(2x + 2)} + C[/tex]

[tex] = \sf\frac{(x {}^{2} + 2x - 3) {}^{5} }{5.2} + C[/tex]

[tex]= \sf \frac{(x {}^{2} + 2x - 3)}{10} + C[/tex]

[tex] \\ [/tex]

kedua, saya pakai integral subtitusi

misal:

k = (x² + 2x - 3)dk/dx = 2x + 2dx = 1/(2x + 2) dk

[tex] \sf \int(x + 1) \: . \: k {}^{4} \: . \: \frac{1}{2x + 2} dk \: [/tex]

[tex] \sf \int k {}^{4} \: . \: \frac{x + 1}{2x + 2}dk[/tex]

[tex] = \sf \frac{k {}^{4 + 1} }{4 + 1} . \frac{1}{2} + C [/tex]

[tex] = \sf \frac{k ^{5} }{5} . \frac{1}{2} + C [/tex]

[tex] = \sf\frac{k {}^{5} }{10} + C [/tex]

ganti k dengan (x² + 2x - 3)

[tex] = \sf \frac{(x {}^{2} + 2x3) {}^{5} }{10} + C [/tex]

[tex] \\ [/tex]

KesimpulanJadi, hasil dari [tex] \sf \int(x + 1)(x {}^{2} + 2x - 3) {}^{4} dx [/tex] adalah [tex] \sf \frac{(x {}^{2} + 2x - 3)}{10} + C [/tex]

26. hasil dari integral 2x^2 (x^3-1)^3 dx


Hasil dari integral 2x^2 (x^3 – 1)^3 dx adalah (1/6) (x^3 – 1)^4 + C. Hasil tersebut diperoleh dengan menggunakan integral substitusi. Integral adalah lawan dari turunan (anti turunan). Bentuk umumnya: ∫ f’(x) dx = f(x) + C

Rumus dasar:

∫ kxⁿ dx = [tex]\frac{k}{n + 1} x^{n + 1}[/tex] + C, dengan n ≠ –1  

Pembahasan

∫ 2x² (x³ – 1)³ dx = ….

Misal

u = x³ – 1

[tex]\frac{du}{dx}[/tex] = 3x²

[tex]\frac{du}{3x^{2}}[/tex] = dx

Substitusikan pemisalan tersebut ke dalam soal

∫ 2x² (x³ – 1)³ dx

= ∫ 2x² . u³ [tex]\frac{du}{3x^{2}}[/tex]

= ∫ [tex]\frac{2}{3}[/tex]  u³ du

= [tex]\frac{2}{3}[/tex] ∫ u³ du

= [tex]\frac{2}{3} \: . \: \frac{1}{3 + 1} u^{3 + 1}[/tex] + C

= [tex]\frac{2}{3} \: . \: \frac{1}{4}[/tex] u⁴ + C

= [tex]\frac{1}{3} \: . \: \frac{1}{2}[/tex] u⁴ + C

= [tex]\frac{1}{6}[/tex] (x³ – 1)⁴ + C

Cara lain

[tex]\int {f(x) \: . \: g^{n} (x)} dx = \frac{f(x) \: . \: g^{n + 1}(x)}{(n + 1) \: . \: g'(x)} [/tex]  + C

Dengan syarat:

[tex]\frac{f(x)}{g'(x)}[/tex] = konstanta

Jadi

∫ 2x² (x³ – 1)³ dx

= [tex]\frac{2x^{2} \: . \: (x^{3} - 1)^{3 + 1}}{(3 + 1) \: . \: 3x^{2}}[/tex] + C

= [tex]\frac{2 \: . \: (x^{3} - 1)^{4}}{4 \: . \: 3}[/tex] + C

= [tex]\frac{2(x^{3} - 1)^{4}}{12}[/tex] + C

= [tex]\frac{1}{6}[/tex] (x³ – 1)⁴ + C
