Perhatikan: Anda mungkin melihat sembulan notifikasi di atas pratinjau. Ini memberi tahu Anda mengenai telah tersedianya versi yang telah diperbarui dan siapa yang membuat perubahan. Mengetuk Segarkan akan memperbarui pratinjau untuk menampilkan versi file yang terakhir.

Untuk jenis file lain, aplikasi akan otomatis akan mencari perangkat Anda untuk menemukan aplikasi yang dapat membuka jenis file yang Anda sedang coba lihat atau edit. Secara default, sistem operasi Android dapat membuka format file berikut ini:

Audio (.aac, .flac, .m4a, .mp3, .oga, .wav, .wma)

Perangkat Anda mungkin dapat langsung membuka lebih banyak format file secara bawaan, bergantung pada perangkat keras dan versi Sistem Operasi Anda. Sebaliknya, jika Dropbox tidak dapat menemukan aplikasi yang dapat membuka jenis file, Anda harus memasang aplikasi yang dapat membuka file.

Ada lebih banyak informasi di situs web Android tentang format media yang didukung Android. Untuk mengetahui lebih banyak tentang media pemutar melalui aplikasi mobile Dropbox, kunjungi artikel pusat bantuan kami tentang mengonversi media untuk perangkat Anda.
