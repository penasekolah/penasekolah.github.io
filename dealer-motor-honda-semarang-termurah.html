11. Sepeda motor dibeli pak dadang dari dealer resmi honda dengan harga Rp 21.000.000 kemudian pak dadang hanya membayar Rp 20.500.000. Berapa % diskon yang dimiliki pak dadang?​


Jawaban:

Untuk mengetahui berapa persentase diskon yang didapat oleh Pak Dadang, kita bisa menggunakan rumus:

Diskon = (Harga asli - Harga bayar) / Harga asli x 100%

Dengan memasukkan nilai harga asli sepeda motor sebesar Rp 21.000.000 dan harga yang dibayarkan sebesar Rp 20.500.000, maka:

Diskon = (21.000.000 - 20.500.000) / 21.000.000 x 100%

Diskon = 500.000 / 21.000.000 x 100%

Diskon = 0,0238 x 100%

Diskon = 2,38%

Jadi, Pak Dadang mendapatkan diskon sebesar 2,38%.

Dealer B harus membayar Rp. 108.000.000,00 untuk membeli 4 sepeda motor jenis I dan 3 sepeda motor jenis II. Jika dealer C membeli 2 sepeda motor jenis I dan 5 sepeda motor jenis II, dealer C harus membayar sebesar …


Diketahui :

• Dealer A membayar 92.000.000 untuk 4 sepeda motor jenis 1 dan 2 sepeda motor jenis 2

• Dealer B membayar 108.000.000 untuk 4 sepeda motor jenis 1 dan 3 sepeda motor jenis 2

Ditanya :

• Berapa dealer C harus membayar jika akan membeli 2 sepeda motor jenis 1 dan 5 sepeda motor jenis 2

Jawab :

Sepeda motor jenis 1 = x

Sepeda motor jenis 2 = y

( eliminasi )

4x + 2y = 92.000.000

4x + 3y = 108.000.000

___________________-

-y = -16.000.000

y = 16.000.000

( substitusi ke persamaan 1 )

4x + 2y = 92.000.000

4x + 2 (16.000.000) = 92.000.000

4x + 32.000.000 = 92.000.000

4x. = 92.000.000 - 32.000.000

4x. = 60.000.000

x. = 60.000.000 ÷ 4

x. = 15.000.000

Dealer C membeli 2 sepeda motor jenis 1 dan 5 sepeda motor jenis 2 :

2x + 5y

2 ( 15.000.000 ) + 5 ( 16.000.000)

30.000.000 + 80.000.00

110.000.000

jadi, dealer C harus membayar sebesar :

Rp. 110.000.000,-.
