Arwin : "OK. Combien pour une chambre double ?" ["Bon. Combien coûte la chambre double ?"]

Réceptionniste : "Quarante dollars par nuit, taxes comprises." ["Quarante dollars la nuit, taxes incluses."]

Arwin : "Les repas sont-ils inclus dans ce prix ?" [“Est-ce que la nourriture est comprise dans le prix?”]
