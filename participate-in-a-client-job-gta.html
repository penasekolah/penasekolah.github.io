1. "Elisa and Rachel participate in English Speech Competition". The negative form of the sentence isA. Elisa and Rachel doesn't participate in English Speech Competition.B. Elisa and Rachel aren't participate in English Speech Competition.C. Elisa and Rachel don't participate in English Speech Competition.D. Elisa and Rachel isn't participate in English Speech Competition​


-simple present tense-

Answer:c. Elisa and Rachel don't participate in English Speech Competition

—

Discussion:

Kalimat pada di atas lebih mengacu ke dalam bentuk simple present tense yang digunakan saat suatu pekerjaan terjadi di masa kini secara berkala atau rutinitas.

Kalimat di atas berpola:

S + V1(-s/es) + O

dimana sufiksasi -s/es akan berlaku untuk subyek berupa:

he, she, it, name, object

Pada kalimat di atas, subyeknya adalah Elisa dan Rachel berupa names atau kata gantinya they maka participate tidak berlaku sufiksasi.

Untuk merubahnya ke dalam kalimat negatif dapat seperti:

S + do/does + not + V1 + O

dimana do/does dipilih salah satu berdasarkan subyeknya.

do: I, you, they, we, names, objectsdoes: he, she, it, name, objects

Agar dapat berubah ke dalam bentuk negatif, cukup tambahkan do/does + not setelah subyek. Elisa dan Rachel merupakan names jadi menggunakan don't.

Elisa and Rachel don't participate in English Speech Competition

—> jawabannya: c.

Jawaban:

Subject Pronouns:

I will participate in a carnival. my parents allow me to join it.

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran Subject Pronouns (kata ganti subjek) dalam Bahasa Inggris.

Petunjuknya adalah my parents allow me, artinya kita membutuhkan kata ganti subjek yaitu "I" dalam kalimat tersebut.

Subject Pronouns adalah kata ganti subjek yang terdiri dari: I (aku/saya), You (anda/kamu/kalian), We (kami/kita), They (mereka), He (dia pria), She (dia wanita), it (itu untuk benda / hewan)

29. what is the function main of HUB A. To close the network from one server to a computer client in one network B. To open the network from one server to a computer client in one network C. To divide the network from one server to a computer client in one network D. To the remove from one server to a computer client in one network E. To divide the client from one server to a computer in one network ​


Jawaban:

The main function of a hub is to connect multiple devices in a network. A hub is a networking device that receives incoming data from a network device and then broadcasts the data to all other connected devices. This allows all devices in the network to communicate with each other and access the same information. In this way, a hub acts as a central point of communication and helps to establish a network connection between devices. Therefore, the correct answer is B. To open the network from one server to a computer client in one network.
