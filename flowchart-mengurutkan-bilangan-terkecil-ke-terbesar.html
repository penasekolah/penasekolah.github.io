5. Buatlah flowchart untuk memilih satu bilangan terbesar dari n buah bilangan


flowchart untuk memilih satu bilangan terbesar dari n buah bilangan adalah seperti di lampiran.

Pembahasan

Untuk mencari nilai maksimal dari n buah bilangan, kita bisa gunakan fungsi percabangan dan perulangan untuk mendapatkan bilangan terbesar.

Kita definisikan dari awal, sebuah variabel max = 0. Kenapa 0, karena kita ingin mencari bilangan terbesar, sehingga nilai awal harus sekecil mungkin.

Misal daftar angka adalah 3 4 2 5 19 9, angka terbesar dari bilangan bilangan tersebut adalah 19.

Pertama tama kita cek angka pertama (3), bandingkan dengan max. Jika angka ke 1 lebih besar dari max, maka max kita beri nilai angka ke 1. 3 > 0 maka max = 3 sekarang.Kemudian kita cek angka kedua, bandingkan dengan max. Jika angka ke 2 lebih besar dari max, maka max kita beri nilai angka ke 2. 4 > 3, maka nilai max sekarang adalah 4Kemudian kita cek angka berikutnya, bandingkan dengan max. Jika angka ke 3 lebih besar dari max, maka max kita beri nilai angka ke 3. 2 tidak lebih besar dari 4, maka nilai max tidak berubah, tetap = 4.Kemudian kita cek angka berikutnya, bandingkan dengan max. Jika angka ke 4 lebih besar dari max, maka max kita beri nilai angka ke 4. 5 lebih besar dari 4, maka nilai max sekarang = 5.Kemudian kita cek angka berikutnya, bandingkan dengan max. Jika angka ke 5 lebih besar dari max, maka max kita beri nilai angka ke 5. 19 lebih besar dari 5, maka nilai max sekarang = 19.Kemudian kita cek angka berikutnya, bandingkan dengan max. Jika angka ke 6 lebih besar dari max, maka max kita beri nilai angka ke 6. 9 tidak lebih besar dari 19, maka nilai max tetap = 19.Angka sudah habis, dan nilai max yang kita dapatkan adalah 19.

Untuk membuat flowchartnya berikut ini adalah penjelasan dari simbol-simbol flowchart

Simbol-simbol dalam flowchart

-Terminator/Simbol Titik Terminal menunjukkan permulaan (start) atau akhir (stop) dari suatu proses.

-Flow Line / garis alir adalah simbol yang digunakan untuk menghubungkan antara simbol yang satu dengan simbol yang lain (connecting line). Simbol ini juga berfungsi untuk menunjukkan arah aliran program.

-Processing Symbol / Simbol Proses digunakan untuk menunjukkan kegiatan yang dilakukan oleh komputer. Pada bidang industri (proses produksi barang), simbol ini menggambarkan kegiatan inspeksi atau yang biasa dikenal dengan simbol inspeksi

-Decision Symbol / Simbol Keputusan merupakan simbol yang digunakan untuk memilih proses atau keputusan berdasarkan kondisi yang ada. Simbol ini biasanya ditemui pada flowchart program.

-Input-Output / Simbol Keluar-Masuk menunjukkan proses input-output yang terjadi tanpa bergantung dari jenis peralatannya.

-Connector (On-page)  

Simbol ini fungsinya adalah untuk menyederhanakan hubungan antar simbol yang letaknya berjauhan atau rumit bila dihubungkan dengan garis dalam satu halaman

-Connector (Off-page)  

Sama seperti on-page connector, hanya saya simbol ini digunakan untuk menghubungkan simbol dalam halaman berbeda. label dari simbol ini dapat menggunakan huruf atau angka

-Preparation Symbol / Simbol Persiapan merupakan simbol yang digunakan untuk mempersiapkan penyimpanan di dalam storage.

Pengertian Algoritma  

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan.  

Dalam dunia pemrograman, algoritma digunakan untuk membuat software dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

 

Pembahasan

Mencari nilai terbesar dari N bilangan dapat dilakukan dengan algoritma membandingkan angka dan mengambil nilai terbesarnya. Berikut langkah-langkahnya dalam pseudocode:  

ALGORITMA mencari nilai terbesar dari N bilangan

Deklarasi:  

    max, angka[4], N : integer  

Deskripsi:  

    Read(N)  

    max = 0  

    for i = 0 to N  

         read(angka[i])  

         if angka[i] > max then max = angka[i]  

    end for  

    write( max )  

Flowchart dapat dibuat dengan mengikuti simbol-simbol dengan deskripsi berikut ini, gambar simbol ada di lampiran.

-Decision Symbol / Simbol Keputusan merupakan simbol yang digunakan untuk memilih proses atau keputusan berdasarkan kondisi yang ada. Simbol ini biasanya ditemui pada flowchart program.  

-Input-Output / Simbol Keluar-Masuk menunjukkan proses input-output yang terjadi tanpa bergantung dari jenis peralatannya.  

-Connector (On-page)    

Simbol ini fungsinya adalah untuk menyederhanakan hubungan antar simbol yang letaknya berjauhan atau rumit bila dihubungkan dengan garis dalam satu halaman

Program C++  

#include<stdio.h>  

int main()  

{  

    int N, max, angka[4];  

    printf("masukan jumlah N: ");  

    scanf("%i", &N);  

    max = 0;  

    for(int i = 0 ; i < N ; i++)  

    {  

         printf("masukan angka ke %i : ", i+1);  

         scanf("%i", &angka[i]);  

         if(angka[i] > max) max = angka[i];  

    }  

    printf("Angka terbesar adalah : %i", max);  

    return 0;  

}

Pembahasan

Mencari nilai terbesar dari N bilangan dapat dilakukan dengan algoritma membandingkan angka dan mengambil nilai terbesarnya. Berikut langkah-langkahnya dalam pseudocode:  

ALGORITMA mencari nilai terbesar dari N bilangan

Deklarasi:  

   max, angka[4], N : integer  

Deskripsi:  

   Read(N)  

   max = 0  

   for i = 0 to N  

        read(angka[i])  

        if angka[i] > max then max = angka[i]  

   end for  

   write( max )  

Flowchart dapat dibuat dengan mengikuti simbol-simbol dengan deskripsi berikut ini, gambar simbol ada di lampiran.  

Simbol-simbol dalam flowchart  

-Terminator/Simbol Titik Terminal menunjukkan permulaan (start) atau akhir (stop) dari suatu proses.  

-Flow Line / garis alir adalah simbol yang digunakan untuk menghubungkan antara simbol yang satu dengan simbol yang lain (connecting line). Simbol ini juga berfungsi untuk menunjukkan arah aliran program.  

-Processing Symbol / Simbol Proses digunakan untuk menunjukkan kegiatan yang dilakukan oleh komputer. Pada bidang industri (proses produksi barang), simbol ini menggambarkan kegiatan inspeksi atau yang biasa dikenal dengan simbol inspeksi  

-Decision Symbol / Simbol Keputusan merupakan simbol yang digunakan untuk memilih proses atau keputusan berdasarkan kondisi yang ada. Simbol ini biasanya ditemui pada flowchart program.  

-Input-Output / Simbol Keluar-Masuk menunjukkan proses input-output yang terjadi tanpa bergantung dari jenis peralatannya.  

-Connector (On-page)    

Simbol ini fungsinya adalah untuk menyederhanakan hubungan antar simbol yang letaknya berjauhan atau rumit bila dihubungkan dengan garis dalam satu halaman  

-Connector (Off-page)    

Sama seperti on-page connector, hanya saya simbol ini digunakan untuk menghubungkan simbol dalam halaman berbeda. label dari simbol ini dapat menggunakan huruf atau angka  

-Preparation Symbol / Simbol Persiapan merupakan simbol yang digunakan untuk mempersiapkan penyimpanan di dalam storage.  

Program C++  

#include<stdio.h>  

int main()  

{  

   int N, max, angka[4];  

   printf("masukan jumlah N: ");  

   scanf("%i", &N);  

   max = 0;  

   for(int i = 0 ; i < N ; i++)  

   {  

        printf("masukan angka ke %i : ", i+1);  

        scanf("%i", &angka[i]);  

        if(angka[i] > max) max = angka[i];  

   }  

   printf("Angka terbesar adalah : %i", max);  

   return 0;  

}
