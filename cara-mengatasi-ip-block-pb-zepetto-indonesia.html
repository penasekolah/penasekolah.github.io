5. Sebuah IP 192.168.80.∅/ 2+1A. Jumlah subnet B. Jumlah hasil persubnetC. Block subnetD. Range IP ​


A. Jumlah subnet:

IP 192.168.80.0/2+1 memiliki 2^(2+1) = 2^3 = 8 subnet.

B. Jumlah hasil per subnet:

Setiap subnet memiliki 2^(32-2-1) = 2^29 = 536,870,912 alamat IP.

C. Block subnet:

Block subnet dalam IP 192.168.80.0/2+1 adalah 192.168.80.0/2+1, dimana "/2+1" menunjukkan mask subnet /2+1, yang berarti 2 bit dari 32-bit alamat IP digunakan untuk subnetting.

D. Range IP:

Range IP setiap subnet akan bervariasi tergantung pada subnet mask yang digunakan. Misalnya, jika subnet mask adalah /25, maka setiap subnet akan memiliki 128 alamat IP (dari 64 hingga 191) dan range IP akan menjadi 192.168.80.0 hingga 192.168.80.127. Namun, jika subnet mask adalah /26, maka setiap subnet akan memiliki 64 alamat IP (dari 0 hingga 63 atau dari 64 hingga 127) dan range IP akan menjadi 192.168.80.0 hingga 192.168.80.63 atau 192.168.80.64 hingga 192.168.80.127.

14. Sebuah IP 192.168.6.∅/28 carilah:A. Jumlah subnet B. Jumlah hasil persubnetC. Block subnetD. Range IP ​


A. Jumlah subnet dapat ditemukan dengan menggunakan rumus 2^n, dimana n adalah jumlah bit yang digunakan untuk subnetting. Dalam hal ini, /28 menunjukkan bahwa 28 bit digunakan untuk alamat IP dan 4 bit digunakan untuk subnetting. Jadi, jumlah subnet adalah 2^4 = 16.

B. Jumlah hasil per subnet dapat ditemukan dengan menggunakan rumus 2^(32-n) - 2, dimana n adalah jumlah bit yang digunakan untuk subnetting. Dalam hal ini, jumlah hasil per subnet adalah 2^(32-28) - 2 = 2^4 - 2 = 14.

C. Block subnet adalah jumlah alamat IP yang tersedia dalam setiap subnet. Dalam hal ini, block subnet adalah 14 alamat IP.

D. Range IP adalah jangkauan alamat IP yang tersedia dalam setiap subnet. Dalam hal ini, range IP dapat ditemukan dengan menggunakan block subnet dan alamat IP awal dari setiap subnet. Misalnya, range IP dari subnet pertama adalah 192.168.6.0 - 192.168.6.14.

A. Jumlah subnet: Terdapat 16 subnet dalam jaringan dengan prefix length /28.

B. Jumlah hasil persubnet: Terdapat 14 host yang dapat di-assign ke setiap subnet.

C. Block subnet: Block subnet untuk jaringan ini adalah 192.168.6.0/28.

D. Range IP: Rentang IP untuk setiap subnet dalam jaringan ini adalah dari 192.168.6.1 hingga 192.168.6.14.

30. bagaimana cara mengatasi printer canon ip 2700 series error 0×00000709 ?​


1. Mengubah User Account Default Dengan Yang Baru

Cara pertama yang dapat kita lakukan untuk menghilangkan pesan error 0x00000709 printer dan bisa kembali membuat set default printer adalah dengan mengubah User Account Komputer Anda. User Account yang selama ini Anda gunakan harus diubah dengan User Account yang baru. Alasannya karena User Account tersebut sudah mengidap penyakit yang menyebabkan error 0x00000709 printer.

Untuk mengubah sekaligus membuat User Account baru caranya mudah, silakan Anda pergi ke Control Panel. Mengaktifkan Control Panel bisa melalui Start kemudian Control Panel atau dengan jalan pintas Run dengan cara menekan Windows+R pada keyboard kemudian ketik Control Panel > Enter.

Jika layar kerja Control Panel sudah aktif, silakan Anda pilih Add or remove user accounts. Fitur ini selain bisa membuat User Account baru, kita juga bisa menghapus User Account.

Perlu diperhatikan sebelum Anda membuat User Account baru, silakan Anda lihat apakah User Account mode Administrator pada komputer Anda terdiri lebih dari satu? Jika terdapat dua atau lebih User Account mode Administrator maka disini Anda tidak perlu membuat User Account baru lagi. Anda cukup menggantikan posisi User Account yang aktif.

Saya asumsikan komputer Anda tidak mempunyai User Account mode Administrator lagi selain User Account yang sedang digunakan sekarang. Maka tugas Anda sekarang adalah membuat lagi User Account. Caranya silakan Anda klik pilih Create a new account.

Silakan Anda ketik nama User Account baru yang akan dibuat pada kolom New account name kemudian Anda pilih mode Administrator. Terakhir jika kedua langkah tadi sudah selesai, klik pilih Create Account.

Sekarang restart (hidupkan ulang) komputer dan ketika komputer Start Up silakan Anda pilih User Account baru yang tadi telah Anda buat.
