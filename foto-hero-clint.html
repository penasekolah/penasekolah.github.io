22. jelaskan sistem kerja antara workstation,clint,dan server


Dalam sebuah topologi jaringan, terutama Client-Server, ada yang namanya Client dan Server.
Server adalah penyedia layanan dalam suatu topologi jaringan. Didalam server terdapat layanan-layanan yang dibutuhkan oleh Client. Client adalah yang meminta layanan kepada Server. Jika client melakukan permintaan ke Server, maka server akan berusaha memberi layanan sesuai permintaan Client. Jika Server tidak dapat memberikan layanan sesuai permintaan Client, maka Server akan memberikan pesan bahwa layanan tidak tersedia.

Workstation itu serupa client, cuma dia berupa PC. Client dapat berupa PC, Laptop ataupun Tablet dll.
