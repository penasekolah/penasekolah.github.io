4. POIN MAX apa itu localhost? apa bedanya dengan 0.0.0.0 atau 127.0.0.1?


Jawaban:

Localhost adalah perangkat milik pengguna baik handphone, desktop ataupun laptop. Address 127.0.0.1 bisa juga disebut localhost secara umum.

Perbedaan:

Secara umum banyak orang menyebut 127.0.0.1 adalah localhost, tetapi 127.0.0.1 berbeda dengan 0.0.0.0 dimana 127.0.0.1 adalah special addresses atau alamat khusus yang digunakan sebagai loopback yang mana data yang dikirim ke alamat tersebut tidak akan mungkin muncul di bagian internet manapun.

Untuk 0.0.0.0 sendiri adalah non-routable meta-address yang mana dapat disimpulkan sebagai alamat yang tidak tentu atau alamat yang salah (invalid address).

Artinya data yang dikirim ke 0.0.0.0 tidak dikirim ke manapun sedangkan data yang dikirim ke 127.0.0.1 akan kembali ke si pengirim data.

Jawaban:

localhost itu alamat lokal atau sering disebut 127.0.0.1, kalau 0.0.0.0 bukan alamat apapun

7. Who is localhost on my wifi?


Localhost juga dikenal sebagai antarmuka loopback. Nama ‘localhost’ biasanya memutuskan untuk alamat IP 127.0.0.1, data yang ditujukan tidak dikirim ke adaptor jaringan fisik apa pun, semuanya tetap berada di dalam mesin lokal Anda (dengan demikian nama ‘localhost’.)

Sebenarnya seluruh rentang alamat dicadangkan untuk loopback, alamat IP dari 127.0.0.1 hingga 127.255.255.254 (atau dalam notasi CIDR 127.0.0.1/8) hanya dapat digunakan oleh loopback, adaptor fisik tidak dapat dikonfigurasi untuk menggunakannya. Tetapi loopback secara implisit dikonfigurasi untuk menggunakan semua 16.777.214 dari mereka.

Adapun bagaimana mungkin berguna untuk memiliki koneksi jaringan yang hanya dapat bertukar data antara proses yang berjalan pada sistem yang sama, pertimbangkan bahwa API jaringan cepat dan efisien dan sebagian besar asynchronous, protokol yang didasarkan pada TCP / IP menjamin bahwa kerangka data adalah dikirimkan ke penerima sesuai urutan pengirimannya, bingkai dibenci jika terjadi kegagalan, dan koneksi yang terputus ditangani dengan baik. Dengan kata lain API jaringan dapat digunakan dengan mudah dan efektif untuk komunikasi antar-proses, jauh lebih kuat daripada pipa bernama.

Juga nyaman untuk menjalankan modul server yang dijamin tidak dapat dijangkau dari mesin lain, seperti menguji prototipe situs web dalam fase desain awal tanpa risiko memaparkan apa pun secara tidak sengaja. Dan konfigurasi implisit dari seluruh rentang juga sangat nyaman.

Jadi lebih mudah untuk dipahami jika Anda seorang programmer, tetapi Anda dapat dengan aman berasumsi bahwa lalu lintas localhost tidak akan membuatnya ke kawat jaringan apa pun (atau tidak diinginkan) dan dalam hal apa pun Anda tidak akan ditagih untuk itu.

21. Cara mengetahui nama localhost mysql server


Jawaban:

Jika sudah masuk ke klien baris perintah cobalah ini:

mysql> select user();

Ini akan menampilkan sesuatu yang mirip dengan ini:

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.41 sec)

contoh diatas login sebagai rootdari localhost.

Jika ingin mengetahui nomor port host lokal di mana Mysql berjalan,gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'port';

mysql> SHOW VARIABLES WHERE Variable_name = 'port';

+---------------+-------+

| Variable_name | Value |

+---------------+-------+

| port | 3306 |

+---------------+-------+

1 row in set (0.00 sec)

kueri diatas akan memberi nomor port tempat MySQL dijalankan.

Jika ingin mengetahui nama host Mysql Anda, gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'hostname';

mysql> SHOW VARIABLES WHERE Variable_name = 'hostname';

+-------------------+-------+

| Variable_name | Value |

+-------------------+-------+

| hostname | Dell |

+-------------------+-------+

1 row in set (0.00 sec)

Kueri diatasakan memberi Anda nama host untuk mysql.

Jika ingin mengetahui nama pengguna Mysql, gunakan kueri ini pada klien baris perintah:

MySQL -

select user();

mysql> select user();

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.00 sec)

Kueri diatas akan memberi Anda nama pengguna untuk mysql.
