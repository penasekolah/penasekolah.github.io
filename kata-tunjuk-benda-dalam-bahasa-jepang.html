1. kata tunjuk benda dalam bahasa Jepang


Jawaban :

Kata tunjuk benda dalam Bahasa Jepang antara lain yaitu Kore, Sore, Are, Dore, Kono, Sono, Ano, dan Dono.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kata Tunjuk dalam Bahasa Jepang

Dalam Bahasa Jepang, kata tunjuk disebut dengan 指示詞 (Shiji-shi). Kata tunjuk ini bisa digunakan untuk menunjuk benda, orang, tempat, dan lain sebagainya. Untuk menghafal kata tunjuk Bahasa Jepang, kalian bisa mengingatnya dengan "Ko-So-A-Do". Kata tunjuk Bahasa Jepang, kata-katanya kebanyakan dimulai dari huruf-huruf tersebut.

Kata tunjuk yang diawali 'Ko' : Digunakan untuk menunjuk sesuatu yang dekat dengan pembicara. Contoh kata nya Kore, Kono, Koko, Kochira, dll. Artinya seperti "Ini" atau "Sini".Kata tunjuk yang diawali 'So' : Digunakan untuk menunjuk sesuatu yang jauh dari pembicara, tapi dekat dengan lawan bicara. Contoh kata nya Sore, Sono, Soko, Sochira, dll. Artinya seperti "Itu" atau "Situ".Kata tunjuk yang diawali 'A' : Digunakan untuk menunjuk sesuatu yang jauh dari pembicara juga lawan bicara. Contoh kata nya Are, Ano, Asoko, Achira, dll. Artinya seperti "Itu*jauh" atau "Sana".Kata tunjuk yang diawali 'Do' : Digunakan untuk menanyakan sesuatu yang ditunjuk lawan bicara. Contoh kata nya Dore, Dono, Doko, Dochira, dll. Artinya seperti "Yang mana" atau "Mana".

Untuk kata tunjuk benda yang lebih jelasnya, simak pembahasan di bawah ini.

- - -

これ・それ・あれ・どれ

(Kore / Sore / Are / Dore)

(Ini / Itu / Itu*jauh / Yang mana)

Kata tunjuk ini digunakan untuk menunjukkan benda. Kata-kata ini dapat berdiri sendiri, tanpa harus menerangkan kata benda. Contoh-contoh kalimatnya sebagai berikut.

これが好き (Kore ga suki) = Suka iniそれが好き (Sore ga suki) = Suka ituあれが好き (Are ga suki) = Suka itu*jauhどれが好き (Dore ga suki) = Suka yang manaこれはペン (Kore wa pen) = Ini pulpenそれはペン (Sore wa pen) = Itu pulpenあれはペン (Are wa pen) = Itu*jauh pulpenどれがペン (Dore ga pen) = Yang mana pulpenこれです (Kore desu) = Yang iniそれです (Sore desu) = Yang ituあれです (Are desu) = Yang itu*jauhどれですか (Dore desuka) = Yang mana?

- - -

この・その・あの・どの

(Kono / Sono / Ano / Dono)

(Ini / Itu / Itu*jauh / Yang mana)

Kata-kata tunjuk ini juga digunakan untuk menunjukkan benda. Perbedaannya dengan yang sebelumnya yaitu untuk kata-kata tunjuk yang ini tidak dapat berdiri sendiri, jadi harus menerangkan kata benda. Contoh-contoh kalimatnya sebagai berikut.

このパン美味い (Kono pan umai) = Roti ini enakそのパン美味い (Sono pan umai) = Roti itu enakあのパン美味い (Ano pan umai) = Roti itu*jauh enakどのパンが美味い (Dono pan ga umai) = Roti mana yang enakこの本です (Kono hon desu) = Buku iniその本です (Sono hon desu) = Buku ituあの本です (Ano hon desu) = Buku itu*jauhどの本ですか (Dono hon desuka) = Buku yang mana

Dari contoh-contoh kalimat di atas dapat diketahui bahwa kata tunjuk yang ini perlu menerangkan kata benda. Setelah kata tunjuk nya selalu terdapat kata benda yang diterangkan.

apa itu kore ? ( ini )

kore adalah kata tunjuk benda yang di gunakan ketika ada benda yg ditunjuk ( sedang berada ) di dekat pembicara ,, dan jauh dari Lawan bicara .

apa itu sore? ( itu )

sore adalah kata tunjuk benda yg digunakan jika benda yg ditunjuk berada jauh dari pembicara . namun , berada di dekat lawan bicara .

apa itu are ? ( itu )

are adalah kata untuk menunjuk benda yang di gunakan jika benda yg ditunjuk jauh dari pembicara & lawan bicara
