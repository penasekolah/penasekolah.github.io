22. ketas gambar yang biasanya digunaka untuk menggamba komi adalah​


Kertas gambar yang biasanya digunakan untuk menggambar Komik adalah HVS, Kuoarto, dan Karton.

Pembahasan

Komik merupakan sarana yang biasanya digunakan untuk menyampaikan pesan melalui rangkaian gambar. Di dalam Komik, biasanya terdapar gambar dan dialog yang memiliki kesan kesatuan yang utuh antara bahasa dan gambar dengan bahasa kata. Komik merupakan sebuah karya seni yang memuat komposisi antara huruf dan gambar. Komik biasanya disebut dengan cerita bergambar. Menggambar Komik memerlukan ketelitian dan ketekunan dalam menggambar karakter atau tokoh dalam cerita. Tokoh dalam Komik dapat berupa manusia atau hewan.

Komik mempunyai fungsi, yaitu menyampaikan pesan secara singkat dengan menggunakan kata-kata dan rangkaian gambar. Pada Komik penggunaan kata-kata yang ditulis biasanya singkat tetapi memiliki kesan yang kuat dan jelas.

Syarat menggambar Komik

Untuk menggambar sebuah Komik dibutuhkan beberapa syarat antara kemampuan dalam menggambar dan penyusunan kata-kata. Syarat-syarat nya adalah sebagai berikut :

Menentukan topik dan tujuanMembuat kalimat singkat dan mudah dipahamiMenggunakan unsur gambarMenggunakan media yang tepat

Bahan dan Alat

Untuk membuat Komik, biasanya digunakan dengan manual tanpa adanya alat bantuan seperti komputer (digital). Bahan dan Alat untuk membuat komik adalah sebagai berikut :

Kertas (HVS/Kuarto/Karton)Pensil (H/B)Pensil warnaPenggaris

[tex]a + b = \sqrt{ {a}^{2} + 2ab + {b}^{2} } = \sqrt{ {a}^{2} + b }2a + b = \sqrt{4 {a}^{2} + 4ab + {b}^{2} } = \sqrt{ {a}^{2} + (a + b)3a + b = \sqrt{ 9{a}^{2} + 6ab + {b}^{2} } = \sqrt{ {a}^{2} + (2a + b).....}}[/tex]

nah abistu baru bandingin sama :

[tex]B =\sqrt{4 + 0+ \sqrt{4 + 2 + 0 + \sqrt{4 + (2 \times 2) + 0 + ...} } }[/tex]
