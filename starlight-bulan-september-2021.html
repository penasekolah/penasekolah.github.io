15. Jumlah penduduk di suatu kecamatan pada bulan Juni 2021 tercatat 60.000 jiwa. Pada bulan Juli 2021 penambahan jumlah penduduk tercatat 50 jiwa, pada bulan Agustus 2021 penambahan jumlah penduduk tercatat 60 jiwa, pada bulan September 2021 penambahan jumlah penduduk tercata 70 jiwa. Jika penambahan jumlah penduduk per bulannya membentuk barisan aritmatika, berapa total jumlah penduduk di kecamatan tersebut pada bulan Maret 2022?​


total jumlah penduduk di kecamatan tersebut pada bulan Maret 2022 yaitu 60.810 jiwa.

Penjelasan dengan langkah-langkah:

Diketahui : umlah penduduk di suatu kecamatan pada bulan Juni 2021 tercatat 60.000 jiwa. Pada bulan Juli 2021 penambahan jumlah penduduk tercatat 50 jiwa, pada bulan Agustus 2021 penambahan jumlah penduduk tercatat 60 jiwa, pada bulan September 2021 penambahan jumlah penduduk tercata 70 jiwa.Ditanya : total jumlah penduduk di kecamatan tersebut pada bulan Maret 2022?​Jawab :

Untuk menjawab pertanyaan ini siswa perlu memahami materi barisan aritmatika dengan baik dan proses yang digunakan dalam penyelesaiannya.

Penduduk bulan juni (U1) = 60.000

Penduduk bulan juli (U2) = 60.000 + 50 = 60.050

Penduduk bulan Agustus (U3) = 60.050 + 60 = 60.110

Penduduk bulan September (U4) = 60.110 + 70 = 60.180

Untuk mencari suku ke n pada barisan aritmatika bertingkat 2, kita gunakan rumus :

Un = an² + bn + c

U1 = 60.000

a.1² + b.1 + c = 60.000

a+b+c = 60.000.....persamaan 1

U2 = 60.050

a.2² + b. 2 + c = 60.050

4a + 2b + c = 60.050 ....persamaan 2

U3 = 60.110

a.3² + b. 3 + c = 60.110

9a + 3b + c = 60.110 ...persamaan 3

U4 = 60.180

a.4² + b. 4 + c = 60.180

16a + 4b + c = 60.180 ...persamaan 4

Eliminasi c pada persamaan 3 dan 2 :

9a + 3b + c = 60.110

4a + 2b + c = 60.050

______________ _

5a + b = 60.....persamaan 5

Eliminasi c pada persamaan 2 dan 1 :

4a + 2b + c = 60.050

a+b+c = 60.000

_____________ _

3a + b = 50....persamaan 6

Eliminasi b pada persamaan 5 dan 6:

5a + b = 60

3a + b = 50

________ _

2a = 10

a = 10/2

a = 5

Substitusi a = 5 ke persamaan 6 :

3a + b = 50

3·5 + b = 50

15 + b = 50

b = 50 - 15

b = 35

Substitusi a = 5 b = 35 ke persamaan 1 :

a+b+c = 60.000

5 + 35 + c = 60.000

40 + c = 60.000

c = 60.000 - 40

c = 59.960

Sehingga diperoleh Un = 5n² + 35n + 59.960

Pada bulan Maret 2022 : Dari bulan Juni ke bulan Maret ada 10 bulan

n = 10

U10 = 5·10² + 35·10 + 59.960

= 500 + 350 + 59.960

= 60.810

Sehingga total jumlah penduduk di kecamatan tersebut pada bulan Maret 2022 adalah 60.810 jiwa.

Rumus Tingkat Laju Inflasi:

(LI) = IHKt-IHKt0 / IHKt0 × 100%
