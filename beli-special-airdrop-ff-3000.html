Misal:
banyaknya penonton karcis 2000 = x
banyaknya penonton karcis 3000 = y
2000x +3000y = 510.000
x+y =200

disubtitusikan:
2000x +3000y = 510.000 (dikalikan 1)
x+y =200 (dikalikan 2000)
menjadi :
2000x +3000y = 510.000
2000x +2000y = 400.000 -
1000y = 110.000
y = 110.000/1000
y = 110

mencari x :
x+y = 200
x + 110 = 200
x = 200-110
x = 90

jadi banyaknya penonton yang membeli karcis dengan harga 2000 adalah 90 penonton
