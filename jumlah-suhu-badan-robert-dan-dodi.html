4. jumlah suhu badan robert dan dodi 133,2°F. saat itu robert dalam keadaan flu sehingga suhu badannya 39°C. berapa derajat celcius suhu badan dodi?

Penjelasan : 

Untuk rumus konversi suhu Celcius, Reamur, Farenheit dan Kelvin bisa dilihat pada lampiran.

Soal :

Jumlah suhu badan Robert dan Dodi 133,2°F. Saat itu Robert dalam keadaan flu sehingga suhu badannya 39°C. Berapa derajat celcius suhu badan Dodi?

Pembahasan : 

Diketahui : 
Suhu Robert dan Dodi = 133,2° F
suhu Robert = 39° C

Ditanya : 
Suhu badan Dodi ?

Jawab : 

Kita ubah dulu farenheit ke celcius.

Robert + Dodi = 133,2° F
C = 5/9 × (°F - 32)
    = 5/9 × (133,2° - 32)
    = 5/9 × 101,2
    = 56,22° C

Robert + Dodi = 56,22° C
39° C + Dodi = 56,22° C
             Dodi = 56,22° C - 39° C
                      = 17,22° C

Jadi suhu badan Dodi adalah 17,22° C

29. jumlah suhu badan robert dan dodi 133,2 F. saat itu robert dalam keadaan flu sehingga suhu badannya 39 C berapa derajat Celcius suhu badan dodi?


Jumlah suhu badan Robert dan Dodi 133,2 °F. Saat itu Robert dalam keadaan flu sehingga suhu badannya 39°C maka suhu badan Dodi 17,22°C. Untuk caranya, bisa dilihat dipembahasan. Satuan suhu terdiri dari celcius (C), Fahrenheit (F) dan Reamur (R).

Termometer skala Celcius ⇒ Titik didih air = 100° C dan Titik beku = 0° C   Termometer skala Reamur ⇒ Titik didih air = 80° R dan Titik beku = 0° R   Termometer skala Fahrenheit ⇒ Titik didih air = 212° F dan Titik beku = 32° F  

Perbandingan Celcius dengan Reamur

(C - 0°) : (R - 0°) = (100° - 0°) : (80° - 0°)  

C : R = 100° : 80°  

C : R = 5 : 4  

maka  

C = [tex]\frac{5}{4}[/tex] × R  R = [tex]\frac{4}{5}[/tex] × C  

Perbandingan Celcius dengan Fahrenheit

(C - 0°) : (F - 32°) = (100° - 0°) : (212° - 32°)

C : (F - 32°) = 100° : 180°  

C : (F - 32°) = 5 : 9  

maka  

C = [tex]\frac{5}{9}[/tex] × (F - 32°)  (F - 32)° = [tex]\frac{9}{5}[/tex] × C ⇒ F = [tex]\frac{9}{5}[/tex] × C + 32°  

Perbandingan Reamur dengan Fahrenheit  

(R - 0°) : (F - 32°) = (80° - 0°) : (212° - 32°)

R : (F - 32°) = 80° : 180°  

R : (F - 32°) = 4 : 9  

maka  

R = [tex]\frac{4}{9}[/tex] × (F - 32°)  (F - 32°) = [tex]\frac{9}{4}[/tex] × R ⇒ F = [tex]\frac{9}{4}[/tex] × R + 32°
Pembahasan

Diketahui

Jumlah suhu badan Robert dan Dodi = 133,2 °F

Suhu badan Robert = 39°C


Ditanyakan

Suhu badan Dodi dalam derajat Celcius


Jawab

Sebelumnya kita ubah dulu 133,2°F ke derajat celcius


133,2°F = .... °C

C = [tex]\frac{5}{9}[/tex] × (F - 32°)

C = [tex]\frac{5}{9}[/tex] × (133,2 - 32°)

C = [tex]\frac{5}{9}[/tex] × 101,2°

C = [tex]\frac{506^{o}}{9}[/tex]

C = 56,22ᵒ

Jadi 133,2°F = 56,22 °C


(Suhu badan Robert) + (suhu badan Dodi) = 56,22 °C

39 °C + (suhu badan Dodi) = 56,22 °C

Suhu badan Dodi = 56,22 °C – 39 °C

Suhu badan Dodi = 17,22 °C

Jumlah suhu badan Robert dan Dodi 133,2 °F. Saat itu Robert dalam keadaan flu sehingga suhu badannya 39°C maka suhu badan Dodi adalah 17,22°C. Untuk caranya, bisa dilihat dipembahasan. Satuan suhu terdiri dari celcius (C), Fahrenheit (F) dan Reamur (R).

Termometer skala Celcius ⇒ Titik didih air = 100° C dan Titik beku = 0° C   Termometer skala Reamur ⇒ Titik didih air = 80° R dan Titik beku = 0° R   Termometer skala Fahrenheit ⇒ Titik didih air = 212° F dan Titik beku = 32° F  

Perbandingan Celcius dengan Reamur

(C - 0°) : (R - 0°) = (100° - 0°) : (80° - 0°)  

C : R = 100° : 80°  

C : R = 5 : 4  

maka  

C = [tex]\frac{5}{4}[/tex] × R  R = [tex]\frac{4}{5}[/tex] × C  

Perbandingan Celcius dengan Fahrenheit  

(C - 0°) : (F - 32°) = (100° - 0°) : (212° - 32°)

C : (F - 32°) = 100° : 180°  

C : (F - 32°) = 5 : 9  

maka  

C = [tex]\frac{5}{9}[/tex] × (F - 32°)  (F - 32)° = [tex]\frac{9}{5}[/tex] × C ⇒ F = [tex]\frac{9}{5}[/tex] × C + 32°  

Perbandingan Reamur dengan Fahrenheit  

(R - 0°) : (F - 32°) = (80° - 0°) : (212° - 32°)

R : (F - 32°) = 80° : 180°  

R : (F - 32°) = 4 : 9  

maka  

R = [tex]\frac{4}{9}[/tex] × (F - 32°)  (F - 32°) = [tex]\frac{9}{4}[/tex] × R ⇒ F = [tex]\frac{9}{4}[/tex] × R + 32°
Pembahasan

Diketahui

Jumlah suhu badan Robert dan Dodi = 133,2 °F

Suhu badan Robert = 39°C


Ditanyakan  

Suhu badan Dodi dalam derajat Celcius


Jawab

Sebelumnya kita ubah dulu 133,2°F ke derajat celcius


133,2°F = .... °C

C = [tex]\frac{5}{9}[/tex] × (F - 32°)

C = [tex]\frac{5}{9}[/tex] × (133,2 - 32°)

C = [tex]\frac{5}{9}[/tex] × 101,2°

C = [tex]\frac{506^{o}}{9}[/tex]

C = 56,22ᵒ

Jadi 133,2°F = 56,22 °C


Suhu badan Robert + suhu badan Dodi = 56,22 °C

39 °C + suhu badan Dodi = 56,22 °C

Suhu badan Dodi = 56,22 °C – 39 °C

Suhu badan Dodi = 17,22 °C
