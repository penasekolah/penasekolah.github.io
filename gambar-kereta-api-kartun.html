Jawaban:

Penjelasan:

(panjang rel kereta api pada gambar kita namakan saja g dan panjang rel kereta api sebenarnya kita namakan saja s)

g = 15 cm

skala = 1 : 450.000

s = g ÷ skala

s = 15 ÷ 1/450.000

s = 15 × 450.000

s = 6.750.000 cm

s = 6.750.000 ÷ 100.000 = 6,75 km
