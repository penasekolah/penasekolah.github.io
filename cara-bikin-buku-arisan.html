y= penggaris

maka :

5x + 3y= 21.000 |dikali 4|

4x + 2y= 16.000 |dikali 5|

mencari nilai y :

20x + 12y= 84.000

20x + 10y= 80.000 _

2y= 4.000

y= 4.000 / 2

y= 2.000

Jadi, harga sebuah penggaris adalah Rp 2.000,00

mencari nilai x :

5x + 3y= 21.000

5x + 3(2.000)= 21.000

5x + 6.000= 21.000

5x= 21.000 - 6.000

5x= 15.000

x= 15.000 / 5

x= 3.000

Jadi, harga sebuah buku adalah Rp 3.000,00.

Harga 10 buku dan 3 penggaris adalah :

10x + 3y

30.000 + 6.000

36.000

Jawab:

Rp. 36.000;

Penjelasan dengan langkah-langkah:

Mislkan

x = buku

y = penggaris

5x + 3y = 21.000    (1)

4x + 2y = 16.000    (2)

Langkah 1 : Eliminasi variabel y

5x + 3y = 21.000   (dikali 2) ↔ 10x + 6y = 42.000

4x + 2y = 16.000   (dikali 3) ↔ 12x + 6y = 48.000 _

                                                  -2x = -6.000

                                                   x = -6.000 : (-2) = 3.000

Jadi, Harga 1 buku = Rp. 3.000;

Langkah 2 : Substitusikan  x = 3.000 ke persamaan (2)

4x + 2y = 16.000

4(3.000) + 2y = 16.000

12.000 + 2y = 16.000

2y = 16.000 - 12.000

2y = 4.000

y = 4.000 : 2 = 2.000

Jadi, harga 1 penggaris adalah Rp. 2.000;

Langkah 3 : Mencai harga 10 buku dan 3 penggaris

10x + 3y = 10(3.000) + 3(2.000)

              = 30.000 + 6.000
