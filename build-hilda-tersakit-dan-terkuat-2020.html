29. Perbandingan tabungan ikwan dan tabungan hilda 3:7 jumlah tabungan ikwan dan hilda adalah 70,000,00.besar tabungan hilda adalah


Jawaban:

Besar tabungan Aris Rp 210.000 dan besar tabungan Riris Rp 490.000.

Pembahasan

Misal perbandingan A dan B adalah m : n

m berarti rasio untuk A, dan n adalah rasio untuk B

Jumlah perbandingan (jumlah rasio) = m + n

Selisih perbandingan (selisih rasio) = m - n dengan m > n

Jumlah A dan B adalah K

Selisih A dan B adalah L

Maka berlaku rumus ↓

A = x K

B = x K

A = x L

B = x L

A = x B

B = x A

K = x A

K = x B

L = x A

L = x B

Materi tentang Perbandingan dapat disimak juga di brainly.co.id/tugas/20889798

Penyelesaian Soal

Diketahui:

Perbandingan tabungan Aris dan Riris = 3 : 7

Jumlah tabungan mereka = Rp 700.000

Ditanya:

Besar tabungan mereka masing masing

Jawab:

Tabungan Aris = x Rp 700.000

= x Rp 700.000

= 3 x Rp 70.000

= Rp 210.000

Tabungan Riris = x Rp 700.000

= x Rp 700.000

= 7 x Rp 70.000

= Rp 490.000

Besar tabungan Aris Rp 210.000 dan besar tabungan Riris Rp 490.000.

30. Tiga tahun lalu,Bu Hilda menabung di sebuah bank. Tabungan Bu Hilda sekarang Rp 3.808.000.00. Jika bank tersebut memberikan bunga tunggal sebesar 4% per tahun.tabungan awal bu hilda sebesar...


Tabungan awal bu Hilda sebesar Rp 3.400.000,00.

Pembahasan

Aritmatika sosial adalah materi matematika yang berhubungan dengan kehidupan sehari", seperti menghitung harga jual, harga beli, untung, rugi, diskon, rabat, bruto, netto, tara, tangungan awal, tabungan ahir, bunga dll.

Beberapa rumus dalam bab Aritmatika sosial

Bunga dalam tahunan = prosentase bunga tahunan x modal awal

Bunga dalam bulanan = prosentase bunga tahunan x modal awal x bulan/12

Tabungan awal = 100% / (100% + bunga%) x tabungan ahir

Lama menabung = [tex]\frac{Bunga~yang~didapat}{Bunga~tahunan}[/tex] x 12

Prosentase Suku Bunga/tahun = [tex]\frac{Bunga/Tahun}{Tabungan Awal}[/tex] x 100%

Penyelesaian Soal

Diketahui:

Tabungan setelah 3 tahun = Rp 3.808.000,00

Bunga tunggal 4% pertahun

Ditanya:

Tabungan awal

Jawab:

Tabungan awal = 100%

Tabungan ahir = Tabungan awal + bunga

                        = 100% + (4% x 3)

                        = 100% + 12%

                        = 112%

Tabungan awal = [tex]\frac{100}{112}[/tex] x Rp 3.808.000,00

                          = Rp 3.400.000,00
