22. perbedaan antara hard copy dan soft copy


Istilah hard copy dan soft copy memiliki perbedaan, yaitu pada bentuk akhir hasil yang akan dicetak. Pada hard copy, hasil yang dicetak berupa print out di atas kertas. Sedangkan pada soft copy, hasil yang dicetak berupa file yang disimpan di dalam media penyimpanan lainnya, seperti flashdisk, CD, atau bahkan file yang dikirimkan melalui email.

Pembahasan

Kata copy berarti salinan. Jadi, arti hard copy adalah salinan yang dapat dilihat dengan jelas atau dapat dipegang. Sedangkan soft copy adalah salinan yang hanya dilihat melalui media lainnya, seperti layar monitor.

Hard copy dapat juga berarti salinan yang berupa cetakan. Contoh pada kalimat berikut. "Berkas yang dikumpulkan berupa 2 hard copy dan soft copy yang dikirimkan melalui email."

Dari kalimat tersebut, berkas yang dikumpulkan adalah 2 salinan yang dicetak dan file yang dikirimkan melalui email. Kata copy juga memiliki makna yang sama pada kata foto copy. Foto copy berarti menyalin sama persis isinya dengan berkas aslinya.

30. apa beda save as a copy dengan save as a text dlm komputer


Apa perbedaan save as copy dengan save as text. Pada lampiran gambar dapat kita lihat terdapat dua macam jenis save as, yang pertama adalah save as copy dan yang berikutnya adalah save as text. Kedua menu tersebut ada pada aplikasi pembaca dokumen portabel yang berformat .pdf

Perbedaan mendasar pada jenis save as tersebut adalah hasil setelah melakukan save as, apabila kita memilih save as copy, maka hasil save as copy akan tetap berupa file .pdf dan menjadi dua file pdf yaitu file yang pertama dan yang kedua adalah file hasil save as.

Untuk jenis save as text, maka hasil penyimpanan akan menjadi teks. Format teks tentu akan mudah untuk dibaca menggunakan berbagai macam aplikasi. Namun, gambar dan beberapa format penulisan akan berbeda ketika file tersebut masih berupa pdf.

Demikian penjelasan mengenai perbedaan antara save as copy dengan save as text, untuk perbedaan akan dapat dilihat pada hasil setelah melakukan save as. Untuk membaca materi lainnya mengenai meu save as atau menu save sebagai menu untuk menyimpan dokumen dapat dibaca pada link berikut
