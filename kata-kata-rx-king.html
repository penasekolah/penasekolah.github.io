21. Jika f(x) = px+q / rx+s, maka f^-1(x) =.....a. ax+q/rx+pb. sx-q/rx-p c. -sx+q/rx-p d. sx-q/rx+p e. sx-q/p-rx​


Jawaban:

C

Penjelasan dengan langkah-langkah:

f(x)=y

y = px+q / rx+s

y(rx+s) = px+q

yrx + ys = px + q

yrx-px = q-ys

x(ry-p) = q-ys

x= q-ys / ry-p

kembalikan y=>x

f^-1(x) = q-xs / rx-p
