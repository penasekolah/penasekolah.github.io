7. rumus luas permukaan bola setengah tertutup​


Jawaban:

Rumus-rumus tentang bangun ruang bola

1) Volume bola = ⁴/₃ x π x r³

2) Luas permukaan bola = 4 x π x r²

3) Luas belahan bola pepat (padat) = 3 x π x r²  

Pembahasan

Luas setengah bola = ¹/₂ x Luas permukaan bola

                                = ¹/₂ x 4 x π x r²

                                = 2 x π x r²

Luas setengah bola tertutup = ¹/₂ x Luas permukaan bola + Luas lingkaran (tutup)

                                 = ¹/₂ x 4 x π x r² + π x r²

                                 = 2 x π x r² +  π x r²

                                 = 3 x   π x r²

14. rumus luas permukaan bangun setengah bola tertutup ​


Jawaban:

Rumus rumus tentang bangun ruang bola

Volum Bola : ⁴/³ × π × r³Luas Permukaan Pada Bola : 4 × π × r²Volum luas Belahan Pepat ( padat ) : 3 × π × r²

Pembahasan

Luas Setengah Bola = ½ × Luas permukaan Bola

= ½ × 4 × π × r²

= 2 × π × r²

Luas Bola Tertutup = ½ × Luas

Permukaan Bola + Luas lingkaran tutup

= ½ × 4 × π × r² + π × r²

= 2 × π × r² + π × r²

= 3 × π × r²

Tentukan rumus menghitung luas permukaan setengah bola tertutup.

Bola adalah bangun ruang 3 dimensi yang terdiri dari beberapa lingkaran yang tak hingga jumlahnya dengan jari-jari yang sama.

Rumus-rumus tentang bangun ruang bola

1) Volume bola = ⁴/₃ x π x r³

2) Luas permukaan bola = 4 x π x r²

3) Luas belahan bola pepat (padat) = 3 x π x r²  

Pembahasan

Luas setengah bola = ¹/₂ x Luas permukaan bola

                                 = ¹/₂ x 4 x π x r²

                                 = 2 x π x r²

Luas setengah bola tertutup = ¹/₂ x Luas permukaan bola + Luas lingkaran (tutup)

                                  = ¹/₂ x 4 x π x r² + π x r²

                                  = 2 x π x r² +  π x r²

                                  = 3 x  π x r²
